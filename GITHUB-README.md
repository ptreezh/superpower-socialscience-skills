## 🚀 快速开始

### 1. 下载技能包

**GitHub**:
```bash
git clone https://github.com/socienceai/agentskills.git
```

**Gitee (国内镜像)**:
```bash
git clone https://gitee.com/socienceai/agentskills.git
```

### 2. 复制技能到技能目录

**OpenClaw**:
```bash
cp -r grounded-theory-expert ~/.openclaw/skills/
```

**WorkBuddy**:
```bash
cp -r grounded-theory-expert ~/.workbuddy/skills/
```

**Qwen**:
```bash
cp -r grounded-theory-expert ~/.qwen/skills/
```

### 3. 使用技能

```bash
openclaw --skill grounded-theory-expert "分析以下访谈数据..."
```

## 📋 技能包结构

每个技能包包含：

```
skill-name/
├── SKILL.md              # 技能定义
├── skill.yaml            # 技能配置
├── soul.md               # 角色定义
├── README.md             # 使用说明
├── tools/                # 工具模块
│   ├── analyze.py
│   └── ...
└── templates/            # 模板文件
    └── task_plan.md.template
```
