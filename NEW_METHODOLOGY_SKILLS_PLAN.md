### 技能转化关键要素

```
├── 理论基础 → references/detailed-guide.md

├── 禁止原则 → SKILL.md（约束规则）
└── CRCT思维链 → prompts/crct-reflection.md
```

## 五、技能结构设计模板

### 标准技能文件结构（15文件）
```
{skill-name}/
├── SKILL.md                    # 技能主文档
├── skill.yaml                  # 技能配置
├── prompts/
│   ├── system-prompt.md        # 系统提示词
│   ├── {analysis}-prompt.md    # 分析提示词
│   └── crct-reflection.md      # CRCT反思
├── tools/
│   ├── {analyzer}.py           # 分析工具（可选）
│   └── requirements.txt        # Python依赖
├── cases/
│   ├── positive/               # 正面案例
│   └── negative/               # 反面案例
├── references/
│   ├── detailed-guide.md       # 详细指南
│   └── classic-literature.md   # 经典文献
├── test_data/                  # 测试数据
└── subagents.yaml              # 子智能体配置（可选）
```

### 核心设计要素
1. **方法论基础** - 明确理论来源和经典文献
2. **禁止原则** - 6条绝对禁止规则
3. **CRCT思维链** - 持续比较/记录/链条/追踪
4. **分析流程** - 标准化步骤分解
5. **输出模板** - 结构化报告格式
