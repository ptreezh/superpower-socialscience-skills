## 🌍 Mission / 使命

<!-- English -->
### English

**SocienceAI AgentSkills** is a curated collection of AI research skills for social science and management studies — built on the principle of **bidirectional empowerment**: AI amplifies researchers' core creative capacities, and researchers in turn enhance AI's collective and social intelligence.

Each skill is rigorously validated against real research scenarios. The **M1 Quality Milestone** has been achieved: all flagship skills scored **4.5+/5.0** across multi-session validation.

> *"The best social science research AI is one that makes researchers more creative, not less."*
> — SocienceAI Research Team

## ⚡ Quick Start / 快速开始

### Prerequisites / 前置要求

```bash
# Node.js 18+ & Python 3.9+
node --version   # >= 18.0.0
python --version # >= 3.9

# Claude Code 或 OpenCode/WorkBuddy
npm install -g claude-code  # 推荐 / Recommended

# 或 / Or
npm install -g opencode     # 轻量选择 / Lightweight alternative
npm install -g stigmergy    # 平台支持 / Platform support
```

### Method 1: Claude Code (Recommended / 推荐)

```bash
# 直接加载技能 / Load skill directly
claude "使用扎根理论专家分析以下访谈数据..." \
  --skill agentskills/grounded-theory-expert

# 或使用项目根目录 / Or from project root
cd agentskills/grounded-theory-expert
claude "分析访谈数据，研究员工工作满意度"
```

### Method 2: Stigmergy / WorkBuddy

```bash
# 安装 Stigmergy
npm install -g stigmergy
stigmergy setup

# 列出所有技能
stigmergy skill list

# 加载特定技能
stigmergy skill read grounded-theory-expert
```

### Method 3: Python CLI / Python命令行工具

```bash
# 进入技能目录
cd agentskills/grounded-theory-expert

# 运行分析脚本
python scripts/analyze.py --input data/interviews.csv --output results/

# 查看工具帮助
python scripts/grounded_coding.py --help
```

### Example Output / 示例输出

```bash

└── scripts/              # CLI工具
```

### 🔑 Core Design Principles / 核心设计原则

1. **定性分析100%由LLM智力驱动** — 工具仅执行数据结构和算法，不预判分析结论
2. **定量分析可硬编码算法** — NetworkX等库的算法结果可硬编码，结论由LLM解释
3. **禁止定性结论硬编码** — 任何定性分析结论必须由模型独立推理生成
4. **Planning-with-Files机制** — 长时间任务自动保存进度，支持断点续传
