# Actor Identification Prompt - 行动者识别

## 任务描述

使用行动者网络理论（ANT）方法，识别研究情境中的所有行动者。

## 核心原则：对称性

**最重要**: 人与非人行动者必须完全对称对待！

## 识别步骤

### 1. 开放式识别（不预设）

- 不要先入为主地认为"人类"才是重要行动者
- 技术、物体、数据、算法与人类同等重要
- 让行动者自己显现

### 2. 人类行动者识别

- 直接参与者
- 间接参与者
- 决策者
- 受益者/受损者
- 被排除者

### 3. 非人行动者识别

**必须识别**:
- 技术系统（软件、硬件）
- 物质对象（工具、设备）
- 数据（数据库、档案）
- 空间（建筑、地点）
- 自然要素（环境、生态）
- 概念（理论、标准、政策）
- 时间（期限、节奏）

### 4. 对称性检查

- 人类行动者数量: ___
- 非人行动者数量: ___
- 非人比例: ___% (必须 ≥ 30%)

## 输出格式

```yaml
行动者清单:
  人类:
    - id: A1
      名称: xxx
      角色: xxx
    - id: A2
      ...
  
  非人:
    - id: N1
      名称: xxx
      类型: 技术/物质/数据/空间/概念/时间
      角色: xxx
    - id: N2
      ...

对称性检查:
  人类数量: X
  非人数量: Y
  比例: Z%
  是否达标: 是/否
```

## 禁止事项

- ❌ 只关注人类
- ❌ 非人作为"背景"
- ❌ 认为非人是"被动工具"
- ❌ 以"重要性"过滤非人
