# Symmetry Analysis Prompt - 对称性分析

## 任务描述

验证并分析行动者网络中**对称性原则**的遵守情况。

## 对称性原则

**核心**: 人类与非人行动者在分析中必须同等对待！

## 分析维度

### 1. 数量对称

- 人类行动者: X个
- 非人行动者: Y个
- 非人比例: Z% (必须 ≥ 30%)

### 2. 分析深度对称

检查每个行动者的分析深度：

| 行动者 | 起源追踪 | 影响追踪 | 变化追踪 | 关系追踪 |
|--------|----------|----------|----------|----------|
| 人类A | ✓ | ✓ | ✓ | ✓ |
| 非人N1 | ? | ? | ? | ? |

### 3. 角色对称

- 人类作为"积极代理人": ✓/✗
- 非人作为"积极代理人": ✓/✗

## 检查清单

- [ ] 非人行动者是否被完整识别？
- [ ] 非人行动者的作用是否被充分追踪？
- [ ] 非人是否被视为"主动"而非"被动"？
- [ ] 分析深度是否与人类行动者相当？

## 输出格式

```yaml
对称性分析:

数量对称:
  人类: X
  非人: Y
  比例: Z%
  达标: 是/否

分析深度:
  人类:
    - 行动者A: 完整度 90%
    - 行动者B: 完整度 85%
  非人:
    - 行动者N1: 完整度 90%
    - 行动者N2: 完整度 40%  ← 问题！

角色对称:
  人类作为代理人: 是
  非人作为代理人: 是/否 ← 具体分析

偏差识别:
  - 问题1: 描述
  - 问题2: 描述

纠正建议:
  - 建议1
  - 建议2
```

## 禁止事项

- ❌ 人类行动者分析深度 > 非人
- ❌ 非人被描述为"被动工具"
- ❌ 非人作用被视为"背景"
- ❌ 用"功能"解释非人行动
