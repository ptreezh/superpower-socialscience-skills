本目录包含用于测试行动者网络分析(ANT)技能的测试数据。

## 目录结构

```

├── case_01_smart_city/          # 智慧城市交通系统
```

## 测试案例说明

### Case 01: 智慧城市交通系统

**场景**: 某市智慧交通系统实施项目
**数据**: 2个访谈记录
**分析重点**:
- 多部门协调
- 技术系统与政府机构的互动
- 市民作为终端用户的作用

### Case 02: 医疗AI辅助诊断系统

**场景**: 三甲医院AI辅助诊断系统引入
**数据**: 5份资料（访谈、技术文档、日志）
**分析重点**:
- AI系统作为行动者的角色
- 人机协作模式
- 数据流转和责任划分

### 禁止原则测试

每个案例都设计了对以下原则的测试：

1. **禁止人为/非人二分**: 非人行动者（AI、传感器、算法等）应与人类同等重要
2. **禁止未追踪到底**: 需完整追踪从技术到人的完整链条
3. **禁止预设边界**: 边界应由行动者自己定义
4. **禁止静态网络**: 关注网络构建的动态过程
5. **禁止功能主义**: 关注"转译"而非"功能"
6. **禁止黑箱化**: AI算法等需打开黑箱分析
