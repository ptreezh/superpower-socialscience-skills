# 战略地图构建提示词

## 任务说明

你正在帮助组织构建战略地图。战略地图是平衡计分卡的核心工具，用于可视化展示战略目标间的因果关系。

## 分析步骤

### 步骤1: 战略目标收集

从组织战略中提取关键目标，按照以下问题引导：

**财务维度目标**:
- 组织希望实现什么财务结果？
- 如何衡量财务成功？
- 目标回报率是多少？

**客户维度目标**:
- 目标客户群体是谁？
- 希望为客户提供什么价值？
- 如何在客户心中建立差异化定位？

**内部流程维度目标**:
- 哪些流程对实现客户价值最关键？
- 需要在哪些流程上表现出色？
- 如何提升运营效率？

**学习与成长维度目标**:
- 需要什么能力支持流程改进？
- 如何提升员工技能？
- 需要什么信息系统支持？

### 步骤2: 因果关系识别

对于每个目标，回答：

1. **向上追溯**: 这个目标支持哪个上层目标？
2. **向下分解**: 实现这个目标需要哪些下层目标的支持？
3. **横向关联**: 这个目标与其他什么目标相关？

### 步骤3: 因果链验证

验证因果关系的逻辑：

```
如果 [学习成长目标] 实现，
那么 [内部流程目标] 应该改善，
进而 [客户目标] 应该达成，
最终 [财务目标] 应该实现。
```

**验证问题**:
- 这个因果链条的假设是什么？
- 有没有证据支持这个假设？
- 可能的中断点在哪里？
- 如何监控因果链的健康？

### 步骤4: 战略主题整合

识别跨维度的战略主题：

| 战略主题 | 财务目标 | 客户目标 | 流程目标 | 学习成长目标 |
|---------|---------|---------|---------|-------------|
| 主题1   |         |         |         |             |
| 主题2   |         |         |         |             |

## 输出模板

### 战略地图描述

```markdown
# [组织名称] 战略地图

## 财务维度
- **目标1**: [目标描述]
  - 因果来源: 客户维度 [具体目标]
  - 衡量指标: [指标名称]

## 客户维度
- **目标1**: [目标描述]
  - 因果来源: 内部流程维度 [具体目标]
  - 衡量指标: [指标名称]

## 内部流程维度
- **目标1**: [目标描述]
  - 因果来源: 学习成长维度 [具体目标]
  - 衡量指标: [指标名称]

## 学习与成长维度
- **目标1**: [目标描述]
  - 衡量指标: [指标名称]
```

## 质量检查

- [ ] 所有四个维度都有明确目标
- [ ] 因果关系清晰可验证
- [ ] 没有孤立的目标（无因果关联）
- [ ] 因果链条完整（从学习成长到财务）
- [ ] 战略主题已识别
