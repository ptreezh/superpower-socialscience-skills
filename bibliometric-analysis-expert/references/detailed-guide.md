## 第一章：文献计量学概论

### 1.1 学科定义

文献计量学(Bibliometrics)是应用数学和统计学方法，对书面交流过程进行定量分析的学科。科学计量学(Scientometrics)是其子领域，专注于科学研究的定量分析。

| 年份 | 里程碑 | 贡献者 |

| 1926 | 洛特卡定律 | Lotka |

| 1948 | 齐普夫定律 | Zipf |
| 1963 | Price的科学计量学 | Price |
| 1963 | Science Citation Index | Garfield |

## 第三章：描述性分析

### 3.1 发文量分析

**时间序列**:
- 年度发文量
- 累积发文量
- 增长模型拟合

**增长模型**:
```
Price模型: N(t) = N₀ × e^(at)
Logistic模型: N(t) = K / (1 + e^(-a(t-t₀)))
```

### 3.2 作者分布

**洛特卡定律**:
```
f(x) = C / x^α

x: 发文篇数
f(x): 发表x篇的作者比例
α: 通常接近2
```

**验证方法**:
- K-S检验
- 卡方检验

### 3.3 期刊分布

**布拉德福定律**:
```
核心区: 相关区: 边缘区 = 1:n:n²

n: 布拉德福系数
```

**核心期刊识别**:
- 载文量排序
- 累积百分比
- 分区方法

### 3.4 普赖斯定律

```
核心作者发文量阈值: m = 0.749 × √n_max

n_max: 最高产作者的发文数
发表m篇以上的作者为核心作者
```

## 第四章：引文分析

### 4.1 引用频次

**基本指标**:
| 指标 | 计算 | 解释 |
|------|------|------|
| 总被引次数 | Σ citations | 总影响力 |
| 篇均被引 | 总被引/发文量 | 平均影响力 |
| 被引率 | 被引文献/总文献 | 影响广度 |

### 4.2 h指数

**定义**: 有h篇论文被引用至少h次

**变体**:
```
g指数: 考虑高被引论文的贡献
e指数: 补偿h指数的不足
hI指数: 标准化作者数量
```

### 4.3 期刊指标

| 指标 | 全称 | 特点 |
|------|------|------|
| IF | Impact Factor | 两年期引用 |
| 5YIF | 5-Year IF | 五年期引用 |
| IPP | Impact per Publication | Elsevier |
| SNIP | Source Normalized IF | 考虑领域差异 |
| SJR | SCImago Journal Rank | 考虑来源声望 |

### 4.4 共引分析

**共引定义**: 两篇文献同时被第三篇文献引用

**共引强度**:
```
Cocitation(i,j) = count(papers citing both i and j)
```

**相似度计算**:
```
Cosine(i,j) = C_ij / √(C_i × C_j)
Jaccard(i,j) = C_ij / (C_i + C_j - C_ij)
```

### 7.3 Bibliometrix (R)

```r
library(bibliometrix)

# 导入数据
M <- convert2df("wos.txt", dbsource = "wos")

# 描述性分析
results <- biblioAnalysis(M)
summary(results)

# 共现分析
NetMatrix <- biblioNetwork(M, analysis="co-occurrences",
                           network="keywords")

# 可视化
networkPlot(NetMatrix, cluster="walktrap")
```
