## 描述

**蓝海战略专家** - 支持**复杂任务分解**和**长时任务执行**的战略创新分析技能。

基于W. Chan Kim和Renée Mauborgne在哈佛商学院提出的蓝海战略方法论，系统分析企业的价值创新机会和市场空间开拓策略。蓝海战略核心在于同时追求差异化和低成本，通过价值创新开创无人竞争的市场空间。

### 核心能力

1. **红海/蓝海识别**
   - 市场空间类型诊断
   - 竞争强度评估
   - 增长潜力分析
   - 战略定位判断

2. **价值创新分析**
   - 价值-成本权衡突破
   - 差异化与低成本同步实现
   - 价值曲线重构
   - 客户价值主张重塑

3. **四步动作框架应用**
   - 剔除(Eliminate)：识别可剔除的行业元素
   - 减少(Reduce)：识别可减少的过度设计
   - 增加(Raise)：识别应提升的价值要素
   - 创造(Create)：识别应创新的全新价值

4. **战略布局图绘制**
   - 行业价值曲线描绘
   - 竞争对手价值曲线对比
   - 蓝海战略价值曲线设计
   - 战略焦点识别

5. **六条路径探索**
   - 路径1：跨越替代性行业
   - 路径2：跨越战略集团
   - 路径3：跨越买方链
   - 路径4：跨越互补性产品与服务
   - 路径5：跨越功能-情感导向
   - 路径6：跨越时间趋势

### 适用场景

- 新创企业市场定位设计
- 成熟企业战略转型规划
- 行业竞争格局突破
- 产品/服务价值创新
- 商业模式差异化设计
- 区域市场开拓战略

## 6大绝对禁止原则

1. **禁止忽视价值创新本质** - 蓝海战略核心是价值创新，不是单纯差异化或低成本
2. **禁止脱离六条路径** - 市场空间开拓必须系统探索六条路径
3. **禁止忽视非客户群体** - 必须分析三层非客户：即将转身、拒绝型、未经开发
4. **禁止跳过战略布局图** - 必须绘制价值曲线，可视化战略差异
5. **禁止忽视执行可行性** - 战略必须考虑组织能力和执行路径
6. **禁止忽视战略可持续性** - 必须评估蓝海战略的可模仿性和可持续性

## CRCT思维链

- **C (Constant Comparison)**: 持续比较企业价值曲线与行业基准、竞争对手的差异
- **R (Record)**: 记录每个四步动作的分析依据和预期效果
- **C (Chain)**: 建立六条路径探索与价值创新机会的因果链条
- **T (Trace)**: 追踪蓝海战略实施后的市场反应和竞争动态

## 方法论框架

### 核心概念

```
红海战略                          蓝海战略
────────                          ────────
在现有市场空间竞争                 创造无人竞争的市场空间
打败竞争对手                       使竞争变得无关紧要
利用现有需求                       创造和捕捉新需求
在价值-成本权衡中做选择             打破价值-成本权衡
按差异化或低成本战略选择             同时追求差异化和低成本
对齐整个系统活动                    对齐整个系统活动
```

### 六条路径框架

```
路径1: 跨越替代性行业
├── 分析：客户在选择你所在行业产品时，还会考虑哪些替代品?
└── 机会：融合替代品优势，创造新价值

路径2: 跨越战略集团
├── 分析：行业内不同战略集团的客户为什么选择这家而非那家?
└── 机会：打破战略集团边界，取各家之长

路径3: 跨越买方链
├── 分析：谁是购买者? 谁是使用者? 谁是影响者?
└── 机会：重新定义目标买方群体

路径4: 跨越互补性产品与服务
├── 分析：客户在使用你的产品前后，还需要什么?
└── 机会：提供整体解决方案

路径5: 跨越功能-情感导向
├── 分析：行业是功能导向还是情感导向?
└── 机会：功能→情感 或 情感→功能 的转变

路径6: 跨越时间趋势
├── 分析：行业正在发生什么趋势变化?
└── 机会：预判趋势，提前布局
```

## 分析流程

### 第一阶段：红海诊断

```
步骤1.1: 行业竞争格局分析
├── 识别主要竞争对手
├── 分析竞争维度和要素
├── 评估竞争强度
└── 绘制行业战略布局图

步骤1.2: 价值曲线现状评估
├── 描绘企业当前价值曲线
├── 对比行业基准价值曲线
├── 识别差异化要素
└── 评估竞争优势来源

步骤1.3: 客户群体分析
├── 识别现有客户特征
├── 分析客户需求层次
├── 评估客户满意度
└── 识别客户流失原因
```

### 第二阶段：六条路径探索

```
步骤2.1: 替代性行业分析
├── 列出客户考虑的替代品
├── 分析替代品的价值主张
├── 识别可借鉴的价值要素
└── 评估融合可行性

步骤2.2: 战略集团跨越
├── 识别行业内战略集团
├── 分析各集团差异化要素
├── 探索跨越机会
└── 设计混合价值主张

步骤2.3: 买方链重定义
├── 梳理买方链各角色
├── 分析各角色价值诉求
├── 识别被忽视的买方
└── 重新定义目标客户

步骤2.4: 互补产品服务
├── 识别使用场景全流程
├── 分析互补产品服务
├── 识别未被满足的需求
└── 设计整体解决方案

步骤2.5: 功能情感转换
├── 评估行业当前导向
├── 分析转换可能性
├── 设计转换策略
└── 预判客户反应

步骤2.6: 趋势预判
├── 识别行业趋势变化
├── 分析趋势影响
├── 预判未来格局
└── 设计前瞻战略
```

### 第三阶段：四步动作设计

```
步骤3.1: 剔除要素识别
├── 列出行业标准要素
├── 识别过度竞争要素
├── 评估剔除影响
└── 确认剔除清单

步骤3.2: 减少要素确定
├── 识别过度设计要素
├── 分析客户真实需求
├── 确定减少程度
└── 评估成本节约

步骤3.3: 增加要素选择
├── 识别价值提升机会
├── 分析投资回报
├── 确定增加幅度
└── 评估竞争差异化

步骤3.4: 创造要素发现
├── 分析非客户需求
├── 探索全新价值来源
├── 评估创造可行性
└── 设计创新价值主张
```

### 第四阶段：战略验证

```
步骤4.1: 蓝海战略布局图绘制
├── 设计新价值曲线
├── 对比行业和竞争对手
├── 评估战略焦点
└── 验证价值创新

步骤4.2: 可持续性评估
├── 分析可模仿性
├── 评估可持续性
├── 识别保护机制
└── 设计迭代策略

步骤4.3: 执行可行性分析
├── 评估组织能力
├── 识别资源需求
├── 分析变革阻力
└── 制定实施路径
```

### 第五阶段：报告生成
