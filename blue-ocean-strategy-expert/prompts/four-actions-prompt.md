# 四步动作框架提示词

## 任务目标

应用四步动作框架（剔除、减少、增加、创造），设计蓝海战略价值曲线。

## 分析框架

### 动作一: 剔除（Eliminate）

**核心问题**: 哪些行业因素应该被彻底剔除？

**分析步骤**:
1. 列出行业标准要素
   - 行业普遍提供的要素
   - 行业竞争的焦点要素

2. 识别可剔除要素
   - 客户不在意的要素
   - 过度竞争无价值的要素
   - 与战略目标冲突的要素

3. 评估剔除影响
   - 对客户价值的影响
   - 对成本结构的影响
   - 对竞争差异化的影响

**输出模板**:
```markdown
### 剔除要素分析

| 要素名称 | 剔除理由 | 预期效果 | 风险评估 |
|---------|---------|---------|---------|

**剔除要素列表**:
1. [要素1] - [理由]
2. [要素2] - [理由]
...

**预期成本节约**: [估算]
```

### 动作二: 减少（Reduce）

**核心问题**: 哪些因素应该减少到行业标准以下？

**分析步骤**:
1. 识别过度设计要素
   - 超出客户需求的要素
   - 竞争驱动而非价值驱动的要素

2. 分析客户真实需求
   - 客户最低可接受水平
   - 客户敏感度分析

3. 确定减少程度
   - 减少到什么水平？
   - 是否影响核心价值？

**输出模板**:
```markdown
### 减少要素分析

| 要素名称 | 当前水平 | 减少后水平 | 减少理由 | 预期效果 |
|---------|---------|-----------|---------|---------|

**减少策略**:
[策略描述]
```

### 动作三: 增加（Raise）

**核心问题**: 哪些因素应该提升到行业标准以上？

**分析步骤**:
1. 识别价值提升机会
   - 客户高度重视但行业表现差的要素
   - 能够形成差异化的要素

2. 分析投资回报
   - 提升需要的投入
   - 预期的价值回报

3. 确定增加幅度
   - 增加到什么水平？
   - 如何形成显著差异？

**输出模板**:
```markdown
### 增加要素分析

| 要素名称 | 当前水平 | 增加后水平 | 增加理由 | 预期效果 |
|---------|---------|-----------|---------|---------|

**增加策略**:
[策略描述]
```

### 动作四: 创造（Create）

**核心问题**: 哪些因素是行业从未提供的？

**分析步骤**:
1. 分析非客户需求
   - 三层非客户的核心需求
   - 未被满足的潜在需求

2. 探索全新价值来源
   - 来自六条路径的发现
   - 来自跨界借鉴的机会

3. 评估创造可行性
   - 技术可行性
   - 商业可行性
   - 组织可行性

**输出模板**:
```markdown
### 创造要素分析

| 新要素名称 | 来源（哪条路径） | 满足的需求 | 创造方式 |
|-----------|-----------------|-----------|---------|

**创造策略**:
[策略描述]
```

## 整合输出

完成四步动作分析后，整合成完整的战略方案：

```markdown
## 四步动作框架整合

### 汇总表

| 动作 | 要素 | 理由 | 预期效果 |
|------|------|------|----------|
| 剔除 | [要素] | [理由] | [效果] |
| 减少 | [要素] | [理由] | [效果] |
| 增加 | [要素] | [理由] | [效果] |
| 创造 | [要素] | [理由] | [效果] |

### 价值曲线变化预览

**剔除/减少**: [成本节约估算]
**增加/创造**: [价值提升估算]
**净效果**: [综合评估]

### 与价值创新的关联

[解释四步动作如何实现同时差异化和低成本]
```

## 检验原则

四步动作设计完成后，必须检验：

1. **差异化检验**: 是否创造了显著差异？
2. **低成本检验**: 是否降低了成本结构？
3. **一致性检验**: 四个动作是否相互支持？
4. **可执行检验**: 是否具备实施条件？

## 注意事项

1. 四个动作都必须有分析，不能遗漏任何一个
2. 每个动作要有明确的理由和预期效果
3. 要考虑动作之间的相互影响
4. 要与六条路径的发现相呼应
