## ⛔ 绝对禁止原则（方法论红线）

> **核心原则**：关系性、反本质主义、场域自主性永远高于简化和本质化

### 一、禁止本质主义解释

**❌ 绝对禁止**：
- 用个体特质解释行为（"他聪明"、"她勤奋"）
- 用群体本质解释差异（"中国人..."、"女性..."）
- 忽略场域位置的约束作用
- 将习性视为"天赋"或"性格"
- 用"文化"、"传统"作为解释终点

**✅ 正确做法**：
- 所有行为必须放在场域关系中理解
- 个体和群体都是场域位置的产物
- 习性是场域经验的内化，不是天赋
- 行为是位置+资本+习性的结果
- "文化"、"传统"本身需要被场域化

**📊 量化标准**：
| 项目 | 最低标准 | 检查方法 |
|------|---------|---------|
| 场域位置分析 | 100%覆盖 | 位置清单检查 |
| 资本类型分析 | ≥3种资本 | 资本类型检查 |
| 习性分析 | 必须有 | 习性模式检查 |
| 关系性解释 | 必须 | 解释方式检查 |
| 本质化语言 | 无 | 语言检查 |

---

### 二、禁止单一资本分析

**❌ 绝对禁止**：
- 只关注经济资本
- 忽略文化资本、社会资本、符号资本
- 认为资本可以互相转换而无损耗
- 忽略资本分布的不对称性
- 将资本视为"资源"而非"权力关系"

**✅ 正确做法**：
- 必须分析至少3种资本
- 每种资本有独立的分布逻辑
- 资本转换有损耗和条件
- 关注资本的符号效应
- 资本是权力关系，不只是资源

**📋 资本分析清单**：
```
1. 经济资本：财富、收入、资产
2. 文化资本：教育、资质、文化能力
3. 社会资本：关系网络、社会联系
4. 符号资本：声誉、认可、权威
5. [可选] 其他资本：信息资本、政治资本...

每种资本必须：
  - 定义如何测量
  - 分析分布特征
  - 识别转换条件
  - 评估符号效应
```

---

### 三、禁止忽略场域自主性

**❌ 绝对禁止**：
- 认为场域是"封闭"的
- 忽略场域间的权力关系
- 忽略外部资本的影响
- 将场域视为独立的"王国"
- 不分析场域的依赖关系

**✅ 正确做法**：
- 场域有相对自主性，但不独立
- 分析场域间的资本流动
- 分析权力场域的影响
- 识别场域的依赖程度
- 分析场域的保卫策略

**📊 场域自主性评估**：
- [ ] 场域是否有自己的规则？
- [ ] 规则是否自主制定？
- [ ] 外部资本如何进入？
- [ ] 场域如何保卫自主性？
- [ ] 依赖程度如何？

---

### 四、禁止静态位置观

**❌ 绝对禁止**：
- 将场域视为固定的结构
- 忽略位置的争夺和变化
- 不分析场域的演变历史
- 认为位置是"天赋"的
- 忽略轨迹和策略

**✅ 正确做法**：
- 场域是争夺的场所
- 位置是持续被定义和重新定义的
- 分析场域的历史演变
- 位置是资本+习性+策略的结果
- 关注轨迹（上升、下降、维持）

**📋 动力学分析清单**：
- [ ] 位置如何被争夺？
- [ ] 哪些位置在变化？
- [ ] 谁在上升？谁在下降？
- [ ] 场域边界是否在变化？
- [ ] 新行动者是否在进入？

---

### 五、禁止忽略习性

**❌ 绝对禁止**：
- 只分析客观位置，忽略主观倾向
- 认为行为是理性选择
- 忽略"实践感"（惯习）
- 不分析行为的无意识维度
- 将策略视为有意识的选择

**✅ 正确做法**：
- 习性是场域位置的内化
- 行为是习性对场域情境的反应
- "实践感"是无意识的、即时的
- 分析习性的双重特性（结构化/结构化结构）
- 策略是习性的有意识表达

**📊 习性分析要求**：
| 项目 | 要求 | 检查方法 |
|------|------|---------|
| 习性识别 | 必须有 | 习性模式检查 |
| 场域经验 | 必须追溯 | 经历分析 |
| 实践感 | 必须分析 | 行为分析 |
| 双重特性 | 必须理解 | 理论检查 |
| 与位置关系 | 必须关联 | 关系检查 |

---

### 六、禁止预测性研究

**❌ 绝对禁止**：
- 声称"预测"行为
- 建立"预测模型"
- 将场域分析简化为变量测量
- 认为场域分析是"应用"研究
- 追求"实用性"而牺牲理论严谨性

**✅ 正确做法**：
- 场域分析是"理解"而非"预测"
- 提供概念工具，非预测模型
- 关注关系性，非变量
- 场域分析是"理论"研究
- 严谨性第一，实用性第二

**📋 研究定位检查**：
- [ ] 研究目的是"理解"还是"预测"？
- [ ] 是否提供概念工具？
- [ ] 是否关注关系性？
- [ ] 是否保持理论严谨性？
- [ ] 是否牺牲理论而追求应用？
