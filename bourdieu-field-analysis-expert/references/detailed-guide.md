## 🔧 工作流程（嵌入绝对禁止检查）

```
用户提供场域研究问题（如"分析学术场域结构"）
    ↓
【前置检查】任务分解质量验证
  - 粒度是否足够在一次会话中完成？
  - 是否有明确的量化标准？
  - 是否有完成度检查清单？
  - 是否要求子 agent 输出数据清单？
    ↓
1. 场域边界识别
   - 场域的相对自主性
   - 场域的核心行动者
   - 场域的核心资本
   - 场域的规则（illusio）
   - 【检查点】场域是否被正确定义？边界是否清晰？
   - 【禁止】本质化场域？忽略场域间关系？
    ↓
2. 行动者位置映射
   - 在资本空间中的位置
   - 在场域结构中的位置
   - 位置的争夺和变化
   - 【检查点】所有行动者都被定位？位置是关系的？
   - 【禁止】静态位置？忽略位置争夺？
    ↓
3. 资本分布分析
   - 经济资本分布
   - 文化资本分布
   - 社会资本分布
   - 符号资本分布
   - 资本转换分析
   - 【检查点】≥3种资本？分布特征清晰？
   - 【禁止】单一资本？忽略转换损耗？
    ↓
4. 习性模式识别
   - 场域经验的内化
   - 实践感的识别
   - 习性的双重特性
   - 习性-位置的对应关系
   - 【检查点】习性被识别？习性-位置关联？
   - 【禁止】忽略习性？习性本质化？
    ↓
5. 位置取用分析
   - 位置如何影响立场
   - 习性如何影响策略
   - 场域情境如何约束选择
   - 客观机会的主观期待
   - 【检查点】位置取用完整？非理性选择？
   - 【禁止】理性选择？忽略习性？
    ↓
6. 场域动力学追踪
   - 场域的历史演变
   - 位置的变化轨迹
   - 新行动者的进入
   - 场域边界的变动
   - 【检查点】历史完整？动态清晰？
   - 【禁止】静态观？忽略历史？
    ↓
7. 权力关系分析
   - 资本的权力效应
   - 符号暴力
   - 场域间权力关系
   - 合法化机制
   - 【检查点】权力关系清晰？符号暴力识别？
   - 【禁止】权力简化？忽略符号？
    ↓
8. 场域自主性评估
   - 场域的相对自主性
   - 外部资本的影响
   - 保卫自主性的策略
   - 【检查点】自主性评估？依赖关系分析？
   - 【禁止】认为场域独立？忽略场域间关系？
    ↓
【最终检查】完成度验证清单（20 项）
    ↓
【承诺宣读】确认未违反任何禁止原则
    ↓
报告完成（仅当全部验证通过）
```

## 📋 任务分解规则

### 分解原则

1. **粒度可控**：每个子任务必须能在一次会话中完成
2. **量化标准**：每个子任务必须有明确的完成标准
3. **独立验证**：每个子任务完成后必须独立验证
4. **数据清单**：子 agent 必须输出完整的数据清单

### 分解模板

```
总任务：[任务名称]
  ├─ 子任务 1: [具体描述]
  │   - 完成标准：[量化指标]
  │   - 输出清单：[位置清单/资本分布]
  │   - 验证方法：[关系性检查/反本质化检查]
  │
  ├─ 子任务 2: [具体描述]
  │   - 完成标准：[量化指标]
  │   - 输出清单：[文件列表]
  │   - 验证方法：[检查步骤]
  │
  └─ ...
```

## 🎓 质量标准

### 分析质量评估矩阵

| 维度 | 合格标准 | 检查方法 |
|------|---------|---------|
| **关系性** | 所有分析基于关系，反本质主义 | 本质化语言检查 |
| **资本多样性** | ≥3种资本，分布清晰 | 资本类型清单 |
| **场域自主性** | 自主性评估，场域间关系分析 | 依赖关系检查 |
| **动态性** | 历史、轨迹、变化分析 | 动态分析检查 |
| **习性分析** | 习性被识别，习性-位置关联 | 习性分析检查 |
| **权力关系** | 符号暴力、合法化被识别 | 权力分析检查 |
| **理论严谨** | 符合布迪厄理论，无简化 | 理论准确性检查 |
