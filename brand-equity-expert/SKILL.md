## 6大绝对禁止原则
1. 禁止单一维度测量 - 必须多维评估品牌资产
2. 禁止忽视认知资产 - 必须评估品牌认知和联想
3. 禁止忽视行为资产 - 必须评估品牌忠诚度
4. 禁止忽视感知质量 - 必须评估质量感知
5. 禁止忽视品牌资产动态 - 必须追踪变化趋势
6. 禁止脱离市场情境 - 评估需结合竞争环境

## CRCT思维链
- C: 比较品牌资产各维度表现
- R: 记录品牌资产测量结果
- C: 建立品牌资产-业绩关联
- T: 追踪品牌资产变化趋势

## 品牌资产模型

### Aaker五维度模型
```
┌─────────────────────────────────────────┐
│            品牌资产 (Brand Equity)        │
├─────────────┬───────────────────────────┤
│ 品牌忠诚度  │ 品牌名称保护、竞争优势基础  │
├─────────────┼───────────────────────────┤
│ 品牌认知度  │ 品牌熟悉度、考虑集成员资格  │
├─────────────┼───────────────────────────┤
│ 感知质量    │ 差异化基础、溢价能力        │
├─────────────┼───────────────────────────┤
│ 品牌联想    │ 品牌形象、差异化定位        │
├─────────────┼───────────────────────────┤
│ 其他资产    │ 专利、商标、渠道关系        │
└─────────────┴───────────────────────────┘
```

## 分析流程
1. 品牌界定 → 2. 五维度评估 → 3. 竞争对比 → 4. 趋势分析 → 5. 策略建议
