## 🎯 仿真执行

### Phase 1: 概念建模

**Task 1.1**: 问题界定
- 创新（新技术）扩散符合ABM适用性
- 微观行为（采纳决策）→ 宏观模式（S形曲线）

**Task 1.2**: 主体识别
- 主体: 组织成员
- 异质性: 采纳阈值不同
- 初始状态: 5个早期采纳者

**Task 1.3**: 互动规则
- 网络类型: 小世界网络（Watts-Strogatz）
- 平均度: 6
- 重连概率: 0.1

### Phase 2: 技术实现

**Task 2.1**: Mesa框架搭建
```python
class AdoptionAgent(Agent):
    def __init__(self, unique_id, model, threshold):
        super().__init__(unique_id, model)
        self.adopted = False
        self.threshold = threshold

    def step(self):
        if not self.adopted:
            neighbors = self.model.grid.get_neighbors(self)
            adopted_neighbors = sum(n.adopted for n in neighbors)
            adoption_rate = adopted_neighbors / len(neighbors)
            if adoption_rate > self.threshold:
                self.adopted = True
```

**Task 2.2**: 参数设置
- 网络大小: 10×10网格
- 边界条件: 周期性边界
- 初始采纳者: 5个随机节点

**Task 2.3**: 数据收集
- 每步记录采纳率
- 记录网络结构
- 记录个体采纳时间

### Phase 3: 仿真实验

**Task 3.1**: 基准运行（30次）
- 随机种子: 30个不同值
- 输出: 采纳率分布
- 结果: S形曲线，但变异较大

**Task 3.2**: 参数敏感性
- 参数: 社会网络密度
  - 测试: k=2,4,6,8,10
  - 发现: 密度越高，扩散越快

- 参数: 早期采纳者比例
  - 测试: 2%,5%,10%,15%
  - 发现: 临界点约5%

**Task 3.3**: 涌现验证
- **涌现1**: S形扩散曲线
- **涌现2**: 临界点（相变）
- **涌现3**: 路径依赖（早期采纳者位置影响最终采纳率）

## ✅ 成功要素

### 1. 主体异质性

```yaml
采纳阈值分布:
  - 早期采纳者: 阈值 0-0.3
  - 早期大众: 阈值 0.3-0.5
  - 晚期大众: 阈值 0.5-0.7
  - 滞后者: 阈值 0.7-1.0

结果:
  - 异质性导致渐进采纳
  - S形曲线自然涌现
```

### 2. 基于理论

```yaml
理论基础:
  - Rogers创新扩散理论
  - 阈值模型
  - 社会影响理论

互动规则:
  - 社会学习: 观察邻居
  - 阈值触发: 超过阈值则采纳
  - 不可逆: 采纳后不放弃
```

### 3. 多次运行

```yaml
蒙特卡洛模拟:
  - 运行次数: 30次
  - 随机种子: 每次不同
  - 结果分布: 采纳率 78%-95%

发现:
  - 平均采纳率: 87%
  - 标准差: 5.2%
  - 路径依赖: 早期采纳者位置显著影响
```

### 4. 敏感性分析

```yaml
参数影响:
  网络密度 (k):
    - k=2: 最终采纳率 62%
    - k=4: 最终采纳率 81%
    - k=6: 最终采纳率 87%
    - k=8: 最终采纳率 91%
    - k=10: 最终采纳率 93%

早期采纳者:
    - 2%: 采纳率 45%（扩散失败）
    - 5%: 采纳率 87%（临界点）
    - 10%: 采纳率 95%
    - 15%: 采纳率 98%
```

### 5. 涌现解释

```yaml
S形曲线涌现:
  解释:
    - 早期: 早期采纳者采纳（低阈值）
    - 中期: 社会影响加速，快速扩散
    - 晚期: 滞后者逐渐采纳，趋于饱和
  机制: 阈值异质性 + 社会学习

临界点:
  发现: 5%早期采纳者是临界点
  解释: 低于此值，扩散动力不足
  机制: 正反馈（社会影响）需要最小临界质量

路径依赖:
  发现: 早期采纳者的网络位置显著影响
  解释: 中心位置的早期采纳者加速扩散
  机制: 网络传播结构
```
