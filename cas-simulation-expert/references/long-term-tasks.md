# CAS长时研究项目指南

## 📋 研究周期：3-12个月

### 适用场景

- **社会扩散研究** (创新、谣言、疾病传播)
- **集体行为研究** (群众运动、市场恐慌)
- **组织演化研究** (企业生态、制度变迁)
- **城市系统仿真** (交通、住房、公共服务)
- **生态系统仿真** (资源管理、环境演化)

---

## 🎯 ABM项目分解模板

### Phase 1: 概念建模 (Month 1)

**Task 1.1**: 研究问题界定 (2周)
- **输出**: 问题陈述文档
- **验证**: 问题具有ABM适用性

**Task 1.2**: 主体识别 (2周)
- **输出**: 主体类型清单、属性定义
- **验证**: 异质性充分

**Task 1.3**: 互动规则设计 (2周)
- **输出**: 互动规则文档
- **验证**: 规则基于理论或实证

### Phase 2: 技术实现 (Months 2-3)

**Task 2.1**: 仿真环境搭建 (2周)
- **输出**: 可运行ABM代码
- **验证**: 代码可运行

**Task 2.2**: 主体行为编码 (4周)
- **输出**: 行为规则实现
- **验证**: 行为符合设计

**Task 2.3**: 互动机制实现 (2周)
- **输出**: 互动网络
- **验证**: 网络结构合理

### Phase 3: 校准与验证 (Months 4-6)

**Task 3.1**: 模式校准 (2个月)
- **输出**: 校准报告
- **验证**: 与真实数据拟合

**Task 3.2**: 参数敏感性 (2个月)
- **输出**: 敏感性分析报告
- **验证**: 多情景测试

**Task 3.3**: 涌现验证 (2个月)
- **输出**: 涌现机制解释
- **验证**: 理论解释充分
