# 案例：Z世代咖啡消费行为分析

## 目标群体界定
中国Z世代(1995-2009年出生)咖啡消费者。

---

## 影响因素分析

### 1. 文化因素
| 因素 | 分析 |
|------|------|
| 文化 | 追求品质生活、社交媒体文化 |
| 亚文化 | 咖啡爱好者社群、打卡文化 |
| 社会阶层 | 年轻白领、大学生为主 |

### 2. 社会因素
| 因素 | 分析 |
|------|------|
| 参照群体 | KOL推荐、朋友分享 |
| 家庭 | 同辈影响大于家庭 |
| 社交 | 社交场景消费为主 |

### 3. 个人因素
| 因素 | 分析 |
|------|------|
| 年龄 | 18-28岁，追求新鲜体验 |
| 经济 | 可支配收入有限但愿意为品质付费 |
| 生活方式 | 快节奏、数字化、社交化 |

### 4. 心理因素
| 因素 | 分析 |
|------|------|
| 动机 | 社交认同、身份表达 |
| 感知 | 品牌感知重于产品本身 |
| 学习 | 社交平台种草→拔草 |

---

## 购买决策旅程

### 阶段1: 问题识别
- 社交场景需求
- 下午茶/提神需求
- 社交媒体种草

### 阶段2: 信息搜索
- 小红书/抖音搜索
- 朋友推荐
- 门店位置查询

### 阶段3: 方案评估
- 品牌调性匹配
- 价格敏感度
- 拍照打卡价值

### 阶段4: 购买决策
- 线上下单/门店点单
- 季节限定偏好
- 会员优惠影响

### 阶段5: 购后行为
- 社交媒体分享
- 复购/品牌转换

---

## 关键洞察
1. 社交属性是核心驱动力
2. 品牌调性比价格更重要
3. 社交媒体影响全旅程
