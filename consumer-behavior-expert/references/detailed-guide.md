# 消费者行为分析详细指南

## 一、方法论概述

消费者行为分析是理解消费者"为什么买"的核心工具，通过系统分析影响因素和决策过程，为营销策略和产品设计提供依据。

## 二、影响因素四层次模型

### 1. 文化因素
| 因素 | 分析要点 |
|------|----------|
| 文化 | 价值观、习俗、规范 |
| 亚文化 | 民族、宗教、地理群体 |
| 社会阶层 | 收入、教育、职业分层 |

### 2. 社会因素
| 因素 | 分析要点 |
|------|----------|
| 参照群体 | 意见领袖、同伴影响 |
| 家庭 | 家庭决策角色 |
| 角色地位 | 社会角色期望 |

### 3. 个人因素
| 因素 | 分析要点 |
|------|----------|
| 年龄/生命周期 | 不同年龄段需求 |
| 职业/经济 | 购买能力约束 |
| 生活方式 | AIO分析 |
| 个性 | 品牌个性匹配 |

### 4. 心理因素
| 因素 | 分析要点 |
|------|----------|
| 动机 | Maslow需求层次 |
| 感知 | 选择性注意、理解 |
| 学习 | 经验形成行为 |
| 信念态度 | 品牌认知评价 |

## 三、购买决策五阶段

```
阶段1: 问题识别
    ↓ 需求触发
阶段2: 信息搜索
    ↓ 内外部搜索
阶段3: 方案评估
    ↓ 评价标准
阶段4: 购买决策
    ↓ 情境因素影响
阶段5: 购后行为
    ↓ 满意度形成
```

## 四、分析流程
1. 定义目标消费群体
2. 分析四层次影响因素
3. 绘制购买决策旅程
4. 识别关键触点
5. 提出营销建议
