## 第一章：内容分析基础

### 1.1 定义与发展

**经典定义**:
> "内容分析是对传播内容进行客观、系统和量化描述的研究方法。"
> — Berelson (1952)

**现代定义**:
> "内容分析是从数据到其情境的可复制和有效推断的研究技术。"
> — Krippendorff (2018)

**发展历程**:
```
1920s-1940s: 早期应用
├── 报纸内容分析
├── 宣传分析
└── 符号互动研究

1950s-1970s: 方法论成熟
├── Berelson经典定义
├── 编码方案标准化
└── 信度指标发展

1980s-2000s: 计算机化
├── 计算机辅助编码
├── 大规模数据应用
└── 新信度指标

2010s至今: 大数据时代
├── 自动内容分析
├── 机器学习方法
└── 社交媒体分析
```

### 1.2 核心特征

**客观性 (Objectivity)**:
- 编码过程独立于编码者主观判断
- 编码规则明确定义
- 结果可由他人复制

**系统性 (Systematic)**:
- 编码规则一致应用于全部数据
- 抽样方法科学
- 过程可重复

**量化性 (Quantitative)**:
- 结果以数值形式呈现
- 支持统计分析
- 可进行假设检验

### 1.3 适用研究类型

| 研究类型 | 示例 |
|---------|------|
| 描述性研究 | 媒体内容特征描述 |
| 比较研究 | 不同媒体内容比较 |
| 趋势研究 | 内容随时间变化 |
| 关系研究 | 内容与其他变量关联 |
| 评估研究 | 传播效果评估 |

## 第二章：研究设计

### 2.1 研究问题类型

**描述性问题**:
- 某类内容包含哪些特征？
- 这些特征的分布如何？

**比较问题**:
- 不同来源的内容有何差异？
- 不同时期的内容有何变化？

**关系问题**:
- 内容特征之间存在何种关联？
- 内容与外部变量有何关系？

### 2.2 分析单位

**单位类型**:

| 单位类型 | 定义 | 示例 |
|---------|------|------|
| 记录单位 | 编码的基本单位 | 文章、广告、帖子 |
| 语境单位 | 理解记录单位所需的语境 | 段落、整篇文章 |
| 计数单位 | 计算频次的单位 | 关键词出现次数 |

**单位选择原则**:
1. 与研究问题匹配
2. 可操作、可识别
3. 有足够变异
4. 便于编码

### 2.3 抽样设计

**抽样方法**:
- 简单随机抽样
- 系统抽样（如每隔n天）
- 分层抽样（按媒体类型、地区分层）
- 构造周抽样（传播研究常用）

**样本量考虑**:
- 效度：样本代表性
- 信度：编码稳定性
- 分析：统计功效
- 实践：资源限制

## 第三章：编码方案开发

### 3.1 类目设计原则

**排他性 (Mutual Exclusivity)**:
- 每个分析单位只能归入一个类目
- 类目之间不重叠

**穷尽性 (Exhaustiveness)**:
- 所有内容都能找到对应类目
- 不存在无法归类的内容

**独立性 (Independence)**:
- 一个单位的编码不影响其他单位
- 类目间无依赖关系

**理论基础 (Theoretical Grounding)**:
- 类目有理论依据
- 与研究概念对应

### 3.2 类目类型

**主题类目**:
```
新闻主题分类
├── 政治新闻
├── 经济新闻
├── 社会新闻
├── 文化新闻
└── 其他
```

**评价类目**:
```
报道倾向
├── 正面
├── 中性
├── 负面
└── 混合
```

**特征类目**:
```
消息来源
├── 官方来源
├── 专家来源
├── 民间来源
└── 未注明
```

### 3.3 编码手册编制

**编码手册结构**:
```
1. 概述
   - 研究目的
   - 编码原则
   - 术语定义

2. 类目体系
   - 类目名称
   - 类目定义
   - 操作指标
   - 编码示例

3. 编码程序
   - 编码步骤
   - 决策规则
   - 特殊情况处理

4. 编码表
   - 变量列表
   - 编码选项
   - 备注空间
```

## 第四章：信度与效度

### 4.1 信度检验

**信度类型**:

| 类型 | 定义 | 检验方法 |
|------|------|---------|
| 编码员间信度 | 不同编码者一致性 | 双编码 |
| 编码员内信度 | 时间稳定性 | 间隔重测 |
| 方案信度 | 编码方案稳定 | 多轮测试 |

**信度系数**:

**Cohen's Kappa (κ)**:
```
κ = (Po - Pe) / (1 - Pe)
Po: 观察一致率
Pe: 期望一致率
```

**Krippendorff's Alpha (α)**:
```
α = 1 - (Do / De)
Do: 观察不一致
De: 期望不一致
```
- 适用于任意测量水平
- 可处理缺失数据
- 多编码员可用

**Scott's Pi (π)**:
```
π = (Po - Pe) / (1 - Pe)
```

**信度标准**:

| 水平 | α/κ值 | 评价 |
|------|-------|------|
| 优秀 | ≥ 0.90 | 高质量研究 |
| 良好 | 0.80-0.89 | 可接受 |
| 最低 | 0.667-0.79 | 勉强可接受 |
| 不可接受 | < 0.667 | 需改进 |

### 4.2 效度检验

**内容效度**:
- 类目是否覆盖研究构念
- 专家评审
- 文献对照

**效标效度**:
- 与已知标准比较
- 预测外部效标
- 同时效度检验

**构念效度**:
- 因子分析检验类目结构
- 与相关构念的关系
- 理论一致性检验

## 第五章：数据分析

### 5.1 描述性分析

**频次分布**:
```
类目      频次    百分比    累积百分比
正面      150     30%       30%
中性      200     40%       70%
负面      150     30%       100%
总计      500     100%
```

**集中趋势**:
- 均值（连续变量）
- 中位数（顺序变量）
- 众数（类目变量）

**离散程度**:
- 标准差
- 范围
- 四分位距

### 5.2 推断性分析

**卡方检验**:
```python
from scipy.stats import chi2_contingency

# 观察频次
observed = [[100, 150], [200, 50]]

# 卡方检验
chi2, p, dof, expected = chi2_contingency(observed)
```

**关联强度**:
- Cramer's V: 多类目关联
- Φ系数: 2×2列联表
- Lambda: 预测关联

### 5.3 多变量分析

- Logistic回归
- 对数线性模型
- 对应分析

## 第六章：质量保证

### 6.1 编码员培训

**培训内容**:
1. 研究背景介绍
2. 编码方案讲解
3. 编码练习
4. 问题讨论
5. 一致性检验

**培训标准**:
- 完成练习编码
- 理解编码规则
- 达到初步一致

### 6.2 过程监控

**监控措施**:
- 定期信度检验
- 编码员会议
- 编码记录抽查
- 问题及时反馈

### 6.3 常见问题处理

| 问题 | 解决方案 |
|------|---------|
| 编码分歧 | 讨论、记录、更新手册 |
| 新情况 | 评估是否新类目，记录决策 |
| 信度下降 | 重新培训、检查方案 |
| 编码疲劳 | 合理分工、定期休息 |
