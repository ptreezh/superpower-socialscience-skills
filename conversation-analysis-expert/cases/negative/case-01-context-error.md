# 负面案例：脱离语境的错误解释

## 案例信息
- **案例ID**: case-01-context-error
- **类型**: 负面案例
- **错误类型**: 脱离语境解释
- **创建时间**: 2026-03-15

---

## 原分析问题

**原始片段**:
```
A: 你明天来吗
B: 我要想想
```

**错误分析**: "B拒绝A的邀请，表示不想来"

---

## 正确分析

### 语境因素
1. **序列位置**: 这是问题-回答序列
2. **偏好组织**: 非偏好回应需要延迟
3. **修复可能**: 可能有后续序列

### 正确解释
- "我要想想"是非偏好回应的延迟策略
- 不等于直接拒绝
- 可能预示后续谈判

---

## 教训

**核心原则**: 分析必须基于可观察证据，不能预设参与者意图
