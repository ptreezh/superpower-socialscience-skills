## 第一章：会话分析概论

### 1.1 学科渊源

会话分析(Conversation Analysis, CA)源于民族方法论(Ethnomethodology)，由Harvey Sacks、Emanuel Schegloff和Gail Jefferson于1960-70年代创立。

**核心理念**:
- 会话是社会行动
- 会话有系统组织
- 会话规则由参与者共建
- 分析应基于自然发生的互动

### 1.2 基本概念

| 概念 | 定义 |
|------|------|
| 话轮(Turn) | 一次发言机会 |
| TCU | 话轮构建单位 |
| TRP | 转换相关位置 |
| 邻接对 | 相邻的配对行为 |
| 序列 | 有组织的话语系列 |
| 修复 | 解决互动问题 |

## 第二章：话轮转换系统

### 2.1 话轮构建单位(TCU)

**类型**:
- 词汇完整: "我明天去北京"
- 语法完整: 句子结构完整
- 语调完整: 语调表示结束
- 实践完整: 行动已完成

**复合TCU**:
```
A: 我想说//第一个//你要注意安全
    └──复合TCU第一部分──┘└──第二部分──┘
```

### 2.2 转换相关位置(TRP)

**位置类型**:
- TCU末端
- 可能的完结点
- 语法-语调-实践三重完备点

### 2.3 话轮分配

**分配规则(Sacks et al., 1974)**:

```
规则1: 当前说话人选择下一说话人
    → 被选者有(并仅有)下一话轮权

规则2: 当前说话人未选择，下一说话人自选
    → 第一个自选者获得话轮

规则3: 当前说话人继续
    → 如果规则1和2都未发生
```

## 第三章：序列组织

### 3.1 邻接对

**特征**:
1. 两部分相邻
2. 由不同说话人产出
3. 有顺序关系(第一部分→第二部分)
4. 有类型关系(特定配对类型)

**常见类型**:
| 第一部分 | 第二部分 |
|---------|---------|
| 问候 | 问候 |
| 问题 | 回答 |
| 请求 | 接受/拒绝 |
| 邀请 | 接受/拒绝 |
| 抱怨 | 道歉/否认 |
| 评估 | 同意/不同意 |

### 3.2 偏好组织

**偏好结构**:
```
请求→接受(偏好回应)
    →拒绝(非偏好回应)

问题→直接回答(偏好)
    →非直接回答(非偏好)
```

**非偏好回应特征**:
- 延迟(停顿、前导语)
- 弱化(缓和语、修饰)
- 解释(说明理由)
- 替代方案

### 3.3 序列扩展

**类型**:
```
前扩展: 第一部分之前
    A: 你有空吗？
    B: 有啊，怎么了？
    A: 能帮我个忙吗？

插入扩展: 第一和第二部分之间
    A: 你明天来吗？
    B: 明天周几？
    A: 周三
    B: 来的

后扩展: 第二部分之后
    A: 谢谢
    B: 不客气
```

## 第四章：修复机制

### 4.1 修复发起

**发起位置**:
```
相同话轮内: 自我发起
    A: 我明天去//今天去北京

话轮转换位置: 自我/他人发起
    A: 我明天去北京
    B: 明天？(他人发起)

第三位置: 他人发起
    A: 我明天去北京
    B: 好的
    A: 不对，是后天
```

### 4.2 修复轨迹

```
问题 → 发起 → 修复 → 结果

示例:
A: 给我那个[杯子]         (问题)
B: 哪个？                  (他人发起)
A: 左边那个蓝色的          (修复)
B: 好的                    (结果)
```

### 4.3 修复偏好

**偏好顺序**:
1. 自我发起，自我修复(最偏好)
2. 他人发起，自我修复
3. 他人发起，他人修复(最不偏好)

## 第五章：机构互动

### 5.1 医患沟通

**特征**:
- 话轮分配不对称
- 医生提问主导
- 患者回答角色
- 诊断序列

**关键序列**:
```
病史采集序列
├── 开场
├── 主诉询问
├── 现病史
├── 既往史
└── 结束
```

### 5.2 课堂互动

**IRE模式**:
```
教师: 启动(提问)
学生: 回应(回答)
教师: 评价(反馈)
```

### 5.3 服务对话

**特征**:
- 目标导向
- 角色分工明确
- 标准化脚本
- 服务序列

## 第六章：转写系统

### 6.1 Jefferson系统

| 符号 | 含义 |
|------|------|
| (0.5) | 暂停秒数 |
| (.) | 微小停顿 |
| [ ] | 重叠开始/结束 |
| = | 连续话语 |
| ?↑ | 升调 |
| ?↓ | 降调 |
| - | 突然中断 |
| : | 声音延长 |
| .hh | 吸气 |
| hh. | 呼气 |
| ↑↓ | 音调升降 |
| < > | 加快/减慢 |
| °word° | 低声 |
| WORD | 高声 |
| *word* | 耳语 |
| (( )) | 研究者注释 |
| ( ) | 听不清内容 |
| (word) | 不确定转写 |

### 6.2 转写原则

1. 忠实于原始录音
2. 包含所有可听信息
3. 使用标准符号
4. 保持一致性和可读性
