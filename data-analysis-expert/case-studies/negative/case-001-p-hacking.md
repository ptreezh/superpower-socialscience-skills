# Case 001: p值崇拜与数据操纵 - 错误案例

**日期**: 2026-03-08
**分析类型**: 问卷数据分析（反面教材）
**问题**: p-hacking与不当数据分析
**状态**: ❌ 失败
**质量评分**: 1/5

---

## ⚠️ 案例背景

### 研究情境
一个研究者想要证明"社交媒体使用影响学业成绩"

### 数据
- 样本：N=50
- 变量：
  - 社交媒体使用时间（小时/天）
  - 学业成绩（GPA）

### 真实关系
- 相关系数：r = -0.18, p = 0.21
- **结论**: 无显著相关

---

## ❌ 错误做法分析

### 错误1: p-hacking

```yaml
错误行为:
  1. 第一次分析: r = -0.18, p = 0.21
     → 研究者: "不显著，再试试其他方法"

  2. 去掉3个"极端值": r = -0.25, p = 0.08
     → 研究者: "接近显著了"

  3. 再去掉2个"异常值": r = -0.31, p = 0.03
     → 研究者: "显著了！✓"

  4. 报告: "社交媒体使用显著负向预测学业成绩,
             r(45) = -0.31, p = .03"

问题:
  - 数据操纵: 为了得到显著结果而删减数据
  - 不透明: 不报告数据清洗过程
  - 不可复现: 别人无法复现"显著"结果
```

**正确做法**:
```yaml
✅ 预注册分析计划
   - 明确异常值处理规则
   - 明确分析方法
   - 不因结果不显著而改变

✅ 完全透明报告
   - 报告所有分析尝试
   - 报告数据清洗过程
   - 报告最终样本量

✅ 关注效应量
   - r = -0.18（小相关）
   - 即使p<0.05，实际意义也有限
```

---

### 错误2: 只报告p值

```yaml
错误报告:
  "社交媒体使用显著负向预测学业成绩,
   t(48) = -2.15, p = .036"

缺失信息:
  ❌ 没有效应量（Cohen's d）
  ❌ 没有置信区间（95% CI）
  ❌ 没有实际意义解释
  ❌ 没有可视化
```

**正确报告**:
```yaml
✅ 完整报告:
  "社交媒体使用与学业成绩呈小负相关，
   r(48) = -0.18, p = .21, 95%CI[-0.44, 0.10]。
   虽然方向为负，但相关系数较小且统计不显著，
   实际意义有限。"

包含:
  ✅ 相关系数（r）
  ✅ p值
  ✅ 置信区间（95%CI）
  ✅ 实际意义解释
  ✅ 可视化（散点图）
```

---

### 错误3: 忽视假设检验

```yaml
错误行为:
  直接运行相关分析:
    r = -0.31, p = .03 ✓

  问题:
    ❌ 没有检查线性假设
    ❌ 没有检查正态性
    ❌ 没有检查异常值影响
```

**真实情况**:
```yaml
假设检查后发现:
  1. 线性假设:
     - 散点图显示非线性关系
     - 建议: 考虑非线性模型

  2. 正态性:
     - Shapiro-Wilk: W=0.92, p=0.01 ✗
     - 违反正态性假设

  3. 异常值:
     - 有2个极端值（|Z|>3）
     - 影响相关系数

正确做法:
  1. 报告假设违反
  2. 使用非参数相关（Spearman）
  3. 或转换数据
```

---

### 错误4: 过度推断因果

```yaml
错误表述:
  "社交媒体使用导致学业成绩下降"

问题:
  - 观察研究，不能推断因果
  - 可能是反向因果（成绩差→多用社交媒体）
  - 可能有混淆变量（学习时间、动机）
```

**正确表述**:
```yaml
✅ 谨慎表述:
  "社交媒体使用与学业成绩呈负相关，
   但由于是观察研究，不能推断因果关系。
   可能的混淆变量包括学习时间、学习动机等。
   未来需要实验设计验证因果效应。"
```

---

### 错误5: 多重比较不校正

```yaml
情境: 研究者检验了10个相关

错误做法:
  不做校正，报告所有p<0.05的结果

问题:
  - 单次检验α=0.05
  - 10次检验→族错误率≈1-(1-0.05)^10≈0.40
  - 假阳性率40%！
```

**正确做法**:
```yaml
✅ Bonferroni校正:
  α' = 0.05 / 10 = 0.005

  报告: "进行了10个相关检验，
         使用Bonferroni校正，
         显著性水平调整为p<0.005"

✅ FDR校正:
  - Benjamini-Hochberg
  - 平衡发现与错误
```

## 🎯 教训总结

### 6大禁止原则违反

```yaml
1. ❌ 脱离数据
   - 为了显著结果操纵数据

2. ❌ p值崇拜
   - 只关注p<0.05
   - 不报告效应量

3. ❌ 忽视假设违反
   - 不检查正态性
   - 不检查线性

4. ❌ 混淆相关与因果
   - 观察研究却推断因果

5. ❌ 多重比较不校正
   - 多次检验但不校正

6. ❌ 不透明
   - 不报告数据清洗
   - 不报告所有分析
```

## ✅ 正确分析示例

如果正确分析这个数据：

```yaml
完整分析流程:

Step 1: 数据探索
  - 描述统计
  - 散点图 → 发现非线性
  - 正态性检验 → 违反

Step 2: 假设检验
  - Pearson相关: r=-0.18, p=.21
  - Spearman相关: ρ=-0.15, p=.29
  - 结论: 无显著相关

Step 3: 效应量
  - r=-0.18（小相关）
  - 实际意义有限

Step 4: 完整报告
  "社交媒体使用与学业成绩呈小负相关
   (r=-0.18)，但统计不显著(p=.21)，
   实际意义有限。"
```

## 📊 对比

| 维度 | ❌ 错误做法 | ✅ 正确做法 |
|------|-----------|-----------|
| **数据分析** | 删减数据直到显著 | 按计划分析所有数据 |
| **结果报告** | 只报告p值 | 报告p值+效应量+CI |
| **假设检验** | 不检查假设 | 检查所有假设 |
| **因果推断** | 过度推断因果 | 谨慎表述 |
| **多重比较** | 不校正 | Bonferroni/FDR校正 |
| **透明性** | 不报告过程 | 完全透明报告 |

## 🎓 学习要点

### 关键教训

1. **数据操纵是学术不端**
   - ❌ p-hacking
   - ❌ Cherry-picking
   - ❌ HARKing

2. **p值不是全部**
   - ✅ 关注效应量
   - ✅ 关注置信区间
   - ✅ 关注实际意义

3. **透明是关键**
   - ✅ 报告所有分析
   - ✅ 报告数据清洗
   - ✅ 报告局限性

4. **方法要适当**
   - ✅ 检查假设
   - ✅ 处理违反
   - ✅ 使用适当方法

## 🔍 识别警示信号

### 可疑分析的信号

```yaml
🚩 警示信号:
  - "接近显著" (p = .051) → 报告为"边缘显著"
  - "去掉异常值后显著"
  - 样本量与效应量不匹配
  - 不报告效应量和置信区间
  - 不报告数据清洗过程
  - 观察研究却用因果语言

如何验证:
  - 要求完整数据和分析代码
  - 检查预注册（如有）
  - 复现分析
  - 关注效应量而非p值
```
