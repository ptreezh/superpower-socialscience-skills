# 正面案例001: 问卷调查的完整分析

## ✅ 正确做法

### 研究问题

**社交媒体使用是否影响大学生的主观幸福感?**

**样本**: N=250名大学生

### 1. 数据探索（反脱离数据）

**步骤1: 描述统计**
```yaml
样本特征:
  - 年龄: M=20.5, SD=1.8, 范围[18,25]
  - 性别: 女性60%, 男性40%
  - 年级: 大一25%, 大二30%, 大三25%, 大四20%

变量分布:
  - 社交媒体使用: 正态分布（Shapiro-Wilk p=0.12）
  - 主观幸福感: 轻微左偏（转换后正态）

异常值:
  - 检测: 3个极端值（|Z|>3）
  - 决策: winsorize处理
```

**步骤2: 信度分析**
```yaml
量表信度:
  - 生活满意度量表: α=0.85（优秀）
  - 积极情感量表: α=0.78（良好）
  - 消极情感量表: α=0.82（良好）

结论: 量表可靠
```

### 2. 相关分析（谨慎因果推断）

**结果**:
| 变量 | 社交媒体使用 | 生活满意度 | 积极情感 | 消极情感 |
|------|-------------|-----------|---------|---------|
| 社交媒体使用 | 1 | -0.15 | -0.08 | 0.22** |
| 生活满意度 | -0.15 | 1 | 0.58** | -0.45** |

**注: *p<0.05, **p<0.01**

**报告**:
```yaml
"社交媒体使用与消极情感呈正相关（r=0.22, p<0.01, 95%CI[0.08,0.35]），
与生活满意度、积极情感无显著相关（r=-0.15, p=0.17; r=-0.08, p=0.45）。"

谨慎表述:
  ❌ "社交媒体导致消极情感"
  ✅ "社交媒体使用与消极情感相关，但无法确定因果方向"
```

### 3. 回归分析（控制混淆）

**模型1**: 基础模型
```yaml
Y = 消极情感
X = 社交媒体使用

结果:
  β = 0.22, SE = 0.08, t = 2.75, p = 0.006
  95% CI [0.06, 0.38]
  R² = 0.05
```

**模型2**: 添加控制变量
```yaml
Y = 消极情感
X = 社交媒体使用
控制: 年龄、性别、年级、家庭SES

结果:
  社交媒体使用: β = 0.18, SE = 0.09, t = 2.00, p = 0.047
  95% CI [0.00, 0.36]
  R² = 0.12

解释:
  - 控制混淆后，效应减弱但仍然显著
  - 增加控制变量提高了解释力（R²: 0.05 → 0.12）
```

**假设检验**:
```yaml
✅ 线性: 残差图无模式
✅ 独立性: Durbin-Watson = 1.95（接近2）
✅ 同方差: Breusch-Pagan p = 0.32（不显著，满足）
✅ 正态性: Q-Q图符合
✅ 多重共线性: VIF < 2（无问题）
```

### 4. 效应量与实际意义

**效应量**:
```yaml
β = 0.18 (标准化系数)

解释:
  - 社交媒体使用每增加1个标准差，消极情感增加0.18个标准差
  - 小到中等效应（Cohen's f² ≈ 0.07）

实际意义:
  - 统计显著（p<0.05）
  - 但实际效应中等
  - 需考虑是否值得干预
```

### 5. 透明报告

**完整报告**:
```yaml
方法:
  - 样本: N=250（年龄M=20.5, SD=1.8）
  - 测量: 7点Likert量表
  - 分析: OLS回归
  - 软件: R 4.3.0

结果:
  - 报告所有系数（包括不显著的控制变量）
  - 报告SE、t、p、95% CI
  - 报告效应量（R², β）

局限:
  - 横截面数据，无法确定因果
  - 自我报告，可能偏差
  - 便利样本，推广性有限

数据共享:
  - 数据和代码已上传至OSF
```
