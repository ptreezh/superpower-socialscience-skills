# 数据分析长时研究项目指南

## 场景1: 问卷调查分析（2-3周）

### Phase 1: 数据准备（Week 1）

**Task 1.1: 问卷设计回顾**
- 检查问卷结构
- 变量操作化
- 测量尺度确认

**Task 1.2: 数据录入与清洗**
- 数据录入（如纸质问卷）
- 双录入验证
- 清洗规则（反向题、缺失值）

**Task 1.3: 数据探索**
- 描述统计
- 分布检查
- 信度分析（Cronbach's α）

### Phase 2: 分析执行（Week 2）

**Task 2.1: 描述统计**
- 样本特征
- 变量分布
- 可视化

**Task 2.2: 推断统计**
- t检验（组间比较）
- 方差分析（多组比较）
- 相关分析（变量关系）

**Task 2.3: 回归分析**
- 简单回归
- 多元回归
- 调节/中介效应

### Phase 3: 结果报告（Week 3）

**Task 3.1: 表格生成**
- 描述统计表
- 推断统计表
- 回归结果表

**Task 3.2: 可视化**
- 直方图、箱线图
- 散点图、交互图

**Task 3.3: 研究报告**
- 方法部分
- 结果部分
- 讨论部分

## 场景3: 纵向数据分析（2-3个月）

### 数据类型

**面板数据** (Panel Data):
- 同一样本多次观测
- 例: PSID (Panel Study of Income Dynamics)

**队列研究** (Cohort Study):
- 特定群体追踪
- 例: 出生队列

### 分析方法

**成长模型** (Growth Model):
```
Yit = β0i + β1i(Time) + eit
β0i = γ00 + γ01(Xi) + u0i
β1i = γ10 + γ11(Xi) + u1i
```

**交叉滞后模型** (Cross-Lagged Panel):
- 检验因果关系方向
- X1 → Y2 vs Y1 → X2
