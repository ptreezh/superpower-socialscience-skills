# 社会科学统计方法详解

## 描述统计

### 集中趋势

**均值** (Mean):
- 定义: 所有值的平均
- 适用: 连续变量、正态分布
- 问题: 对异常值敏感

**中位数** (Median):
- 定义: 排序后中间值
- 适用: 有序变量、偏态分布
- 优点: 对异常值稳健

**众数** (Mode):
- 定义: 出现频率最高的值
- 适用: 名义变量

### 离散程度

**方差** (Variance):
```
σ² = Σ(xi - μ)² / N
```

**标准差** (Standard Deviation):
```
σ = √σ²
含义: 平均偏离程度
```

**四分位距** (IQR):
```
IQR = Q3 - Q1
优点: 对异常值稳健
```

### 分布形态

**偏度** (Skewness):
- 0: 对称
- >0: 右偏（正偏）
- <0: 左偏（负偏）

**峰度** (Kurtosis):
- 0: 正态峰度
- >0: 尖峰
- <0: 平峰

## 推断统计

### 假设检验框架

**步骤**:
1. 设定假设（H0, H1）
2. 选择检验统计量
3. 计算p值
4. 做出决策

**错误类型**:
- **第一类错误** (α): 假阳性（拒绝真H0）
- **第二类错误** (β): 假阴性（接受假H0）
- **统计功效** (1-β): 正确拒绝假H0的概率

### t检验

**单样本t检验**:
```
H0: μ = μ0
检验统计量: t = (M - μ0) / (SE)
```

**独立样本t检验**:
```
H0: μ1 = μ2
假设: 方差齐性
如违反: Welch's t检验
```

**配对样本t检验**:
```
H0: μd = 0 （差值均值）
用途: 前后测、匹配样本
```

### 卡方检验

**拟合优度**:
```
H0: 观察频数 = 期望频数
```

**独立性检验**:
```
H0: 两个变量独立
用途: 列联表分析
```

### 方差分析 (ANOVA)

**单因素ANOVA**:
```
H0: 所有组均值相等
事后检验: Tukey HSD
效应量: η²
```

**重复测量ANOVA**:
```
用途: 同一被试多次测量
假设: 球形性（Mauchly's test）
```

## 回归分析

### 简单线性回归

**模型**:
```
Y = β0 + β1X + ε
```

**假设**:
1. 线性
2. 独立性
3. 同方差性
4. 正态性

**诊断**:
- 残差图（检验线性、同方差）
- Q-Q图（检验正态性）
- VIF（检验多重共线性）

### 多元线性回归

**模型**:
```
Y = β0 + β1X1 + β2X2 + ... + βkXk + ε
```

**指标**:
- **R²**: 解释的方差比例
- **调整R²**: 考虑变量数
- **F统计量**: 整体显著性
- **t统计量**: 单个系数显著性

### 逻辑回归

**模型**:
```
log(P/(1-P)) = β0 + β1X1 + ... + βkXk
```

**输出**:
- **Odds Ratio (OR)**: exp(β)
- **95% CI**: OR的置信区间
- **Pseudo R²**: 类似R²

### 多层模型 (HLM)

**用途**:
- 嵌套数据（学生-班级-学校）
- 纵向数据（重复测量）

**模型**:
```
Level 1: Yij = β0j + β1jXij + eij
Level 2: β0j = γ00 + γ01Wj + u0j
```

## 效应量

### Cohen's d

**定义**: 两组均值差异的标准差单位
```
d = (M1 - M2) / SDpooled
```

**解释**:
- 0.2: 小
- 0.5: 中
- 0.8: 大

### 相关系数 (r)

**Pearson r**: 连续变量线性相关
**Spearman ρ**: 秩相关（非参数）

**解释**:
- 0.1: 小
- 0.3: 中
- 0.5: 大

### η² (Eta Squared)

**定义**: ANOVA中组间平方和占比
```
η² = SSbetween / SStotal
```

## 常见错误

### 错误1: p值崇拜
- ❌ "p<0.05，显著，重要"
- ✅ 报告效应量、置信区间、实际意义

### 错误2: 忽视假设
- ❌ "样本大，正态性不重要"
- ✅ 检验假设，违反时用替代方法

### 错误3: 过度推断
- ❌ "X与Y相关，所以X导致Y"
- ✅ 谨慎语言，讨论混淆变量
