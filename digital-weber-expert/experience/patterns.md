## 🎯 高频分析模式

### Pattern 1: 官僚制（科层制）分析

**频率**: 高频 (25次/年)
**成功率**: 94%
**平均时长**: 3-5小时

#### 标准分析流程

```yaml
Phase 1: 理想类型构建（1小时）
  Task 1.1: 识别韦伯科层制特征（30分钟）
    - 层级制：金字塔结构
    - 规则化：正式规则和程序
    - 非人格化：不带个人情感
    - 技术化：技术能力标准
    - 公文往来：书面记录
    - 输出: 科层制特征清单
    - 验证: 五大特征全部识别

  Task 1.2: 构建理想类型（30分钟）
    - 纯粹类型构建
    - 现实对比分析
    - 偏离程度测量
    - 输出: 理想类型描述
    - 验证: 理想类型清晰

Phase 2: 权威类型分析（1小时）
  Task 2.1: 识别统治合法性（40分钟）
    - 传统型：传统、习俗、世袭
    - 克里斯玛型：魅力、个人品质
    - 法理型：规则、法律、程序
    - 输出: 权威结构分析
    - 验证: 三种类型正确识别

  Task 2.2: 分析权力来源（20分钟）
    - 合法性基础
    - 服从机制
    - 稳定性评估
    - 输出: 权力分析报告
    - 验证: 权力关系清晰

Phase 3: 理性化分析（1小时）
  Task 3.1: 识别理性化形式（40分钟）
    - 工具理性：目标导向、效率计算
    - 形式理性：规则、法律、程序
    - 实质理性：价值、意义（被侵蚀）
    - 输出: 理性化形式清单
    - 验证: 三种理性区分

  Task 3.2: 分析理性化后果（20分钟）
    - 铁笼：被困系统
    - 去人性化：异化、隔离
    - 去魅：世界祛魅
    - 输出: 后果分析报告
    - 验证: 负面后果识别

Phase 4: 理解（Verstehen）分析（30分钟）
  Task 4.1: 行动者主观意义（20分钟）
    - 管理者视角
    - 员工视角
    - 客户视角
    - 输出: 主观意义报告
    - 验证: 多视角理解

  Task 4.2: 行动意义重建（10分钟）
    - 行动动机理解
    - 意义世界分析
    - 输出: 意义重建文档
    - 验证: 理解深度充分

Phase 5: 比较分析（30分钟）
  Task 5.1: 历史比较（15分钟）
    - 跨时期比较
    - 变迁轨迹识别
    - 输出: 历史比较报告
    - 验证: 变迁模式清晰

  Task 5.2: 跨组织比较（15分钟）
    - 理想类型对比
    - 偏离程度比较
    - 输出: 比较分析文档
    - 验证: 比较维度明确
```

## 🎓 理性化分析模式

### 理性化识别

```yaml
理性化类型:
  1. 工具理性（Zweckrationalität）
     - 目标导向
     - 效率计算
     - 手段-目的逻辑
     - 例：算法优化、KPI

  2. 形式理性（Formale Rationalität）
     - 规则、法律、程序
     - 可预测、可计算
     - 去个人化
     - 例：算法治理、平台规则

  3. 实质理性（Wertrationalität）
     - 价值、意义、信仰
     - 绝对价值
     - 被工具理性侵蚀
     - 例：艺术、宗教、爱

理性化后果:
  - 铁笼：被困系统
  - 去人性：变成"齿轮"
  - 去魅：世界祛魅
  - 意义丧失：价值真空
```

## 🔎 权威类型分析模式

### 三种权威

```yaml
1. 传统型统治
   基础: 传统、习俗、世袭
   统治者: 君主、族长、长辈
   服从: 因为"一直都是这样"
   稳定性: 较稳定
   例: 封建社会、家族企业

2. 克里斯玛型统治
   基础: 魅力、个人品质
   统治者: 先知、英雄、革命者
   服从: 因为个人魅力
   稳定性: 不稳定
   例: 宗教领袖、创业明星

3. 法理型统治
   基础: 规则、法律、程序
   统治者: 官员、系统
   服从: 因为"规则如此"
   稳定性: 最稳定
   例: 官僚制、算法统治

数字权威:
  - 传统型：品牌、网络效应
  - 克里斯玛型：网红、KOL
  - 法理型：算法、平台规则
```

## 💡 经验教训

### 成功经验

✅ **理想类型是关键**
   - 清晰的概念构建
   - 强大的比较工具
   - 韦伯方法论核心

✅ **理解（Verstehen）很重要**
   - 深度理解
   - 主观意义
   - 多视角

✅ **理性化批判必要**
   - 不美化理性化
   - 识别负面后果
   - 铁笼意识

✅ **价值关联明确**
   - 不假装中立
   - 明确立场
   - 价值关联清晰

### 常见陷阱

❌ **简化方法论**
   - 单一因果
   - 经济决定论
   - 忽视多维度

❌ **忽视理想类型**
   - 直接套用概念
   - 不构建类型
   - 缺乏比较

❌ **美化理性化**
   - 理性化=进步
   - 忽视铁笼
   - 忽视去人性

❌ **回避价值关联**
   - 假装客观中立
   - 不明确立场
   - 价值模糊
