## 成功模式

### 模式1：三维整合分析

**场景**: 任何CDA分析

**方法**:
- 从文本特征出发
- 通过话语实践解释
- 连接社会实践阐明意义
- 三个维度形成完整链条

**效果**: 确保分析既有语言基础，又有社会意义。

**适用条件**: 所有批判话语分析研究。

---

### 模式2：权力-意识形态双维分析

**场景**: 政治话语、媒体话语分析

**方法**:
1. 识别话语建构的群体身份
2. 分析群体间的权力关系
3. 揭示权力关系的意识形态基础
4. 阐明意识形态再生产机制

**效果**: 系统揭示话语的权力运作。

**适用条件**: 涉及群体关系的话语分析。

---

### 模式3：历史-共时结合分析

**场景**: 长期话语演变研究

**方法**:
1. 分析特定时期的共时特征
2. 追踪关键话语的历史演变
3. 分析演变的动力机制
4. 解释当前特征的由来

**效果**: 深化对话语的历史性理解。

**适用条件**: 需要历史视角的话语研究。

## 失败教训

### 教训1：技术主义分析

**场景**: 媒体话语分析

**错误做法**:
- 详细统计词汇频率
- 分析句法结构
- 但不连接社会实践
- 结论仅为描述性发现

**后果**:
- 分析无社会意义
- 不是真正的CDA
- 只是语言学描述

**正确做法**:
- 每个语言分析都问"为什么"和"有什么效果"
- 连接权力关系和意识形态
- 阐明话语的社会功能

**教训标签**: #技术主义 #脱离社会

---

### 教训2：预设批判立场

**场景**: 政治话语分析

**错误做法**:
- 先确定要批判的对象
- 只寻找支持批判的证据
- 忽视反对证据
- 分析服务于预设结论

**后果**:
- 分析缺乏客观性
- 可信度受损
- 违背CDA原则

**正确做法**:
- 基于文本证据分析
- 考虑替代解释
- 承认分析的局限
- 让证据引导结论

**教训标签**: #预设立场 #选择性证据

---

### 教训3：过度解读

**场景**: 广告话语分析

**错误做法**:
- 从简短文本推导复杂意识形态
- 无证据地推断"隐含意义"
- 将可能的解读当作确定的发现

**后果**:
- 分析缺乏可信度
- 被批评为"阴谋论"
- 损害CDA方法论声誉

**正确做法**:
- 分析必须有文本证据
- 区分明确意义和可能暗示
- 承认多种解读可能
- 限制分析的宣称范围

**教训标签**: #过度解读 #证据不足

## 质量标准基准

### 文本分析质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 语言分析详细，有社会意义解读，连接权力/意识形态 |
| 良好 | 语言分析基本完整，有一定的社会意义解读 |
| 及格 | 有语言分析，但社会意义解读不足 |
| 不及格 | 只有语言描述，无社会意义 |

### 话语实践分析质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 生产/消费分析完整，互文性分析详细 |
| 良好 | 有基本的生产/消费分析 |
| 及格 | 有部分话语实践分析 |
| 不及格 | 完全忽视话语实践 |

### 社会实践分析质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 权力分析深入，意识形态揭示清晰，有变革分析 |
| 良好 | 有权力或意识形态分析 |
| 及格 | 有社会实践层面的提及 |
| 不及格 | 完全脱离社会实践 |
