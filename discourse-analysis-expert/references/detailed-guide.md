## 第一章：话语分析概述

### 1.1 什么是话语分析

话语分析是研究语言在社会生活中使用的方法论体系，关注：
- 语言如何建构社会现实
- 话语中的权力关系
- 意识形态如何在话语中运作

### 1.2 主要流派

| 流派 | 代表人物 | 核心关切 |
|------|---------|---------|
| 批判话语分析(CDA) | Fairclough, van Dijk, Wodak | 权力、意识形态、社会变革 |
| 话语心理学 | Potter, Wetherell | 心理现象的话语建构 |
| 会话分析 | Sacks, Schegloff | 日常互动的组织 |
| 福柯式话语分析 | Foucault | 话语与权力/知识 |

### 2.2 文本层面分析

#### 词汇分析

**分类系统**：
```
分析要点：
1. 如何命名社会行动者？
   - "非法移民" vs "无证移民" vs "外来劳工"
   - 不同的命名建构不同的现实

2. 如何分类社会现象？
   - 分类标准是什么？
   - 分类边界如何划定？
   - 边缘群体如何被分类？
```

**评价性词汇**：
```
评价维度：
- 判断：好/坏、对/错
- 情感：喜欢/不喜欢
- 欣赏：美丽/丑陋
- 安全：安全/危险

分析工具：
- 态度系统 (Appraisal)
- 评价框架 (Evaluation)
```

**隐喻分析**：
```
概念隐喻理论 (Lakoff & Johnson):

常见隐喻系统：
- 战争隐喻："抗击疫情"、"战胜疾病"
- 建筑隐喻："构建和谐社会"、"夯实基础"
- 流动隐喻："人才流动"、"信息流"
- 疾病隐喻："社会病"、"经济癌症"

意识形态功能：
- 隐喻建构我们对议题的理解方式
- 隐喻凸显某些方面，遮蔽其他方面
```

#### 语法分析

**及物性系统**：
```
过程类型：
| 类型 | 定义 | 示例 | 功能 |
|------|------|------|------|
| 物质 | 行动/做 | "政府出台了政策" | 突出行动 |
| 心理 | 感知/思考 | "人民相信..." | 内在状态 |
| 关系 | 存在/属性 | "这是正确的决定" | 定义/分类 |
| 言语 | 说 | "发言人表示..." | 信息传递 |
| 行为 | 生理行为 | "他在哭泣" | 行为表现 |
| 存在 | 存在 | "存在问题" | 存在状态 |
```

**情态系统**：
```
认识情态：
- 高：一定、必然
- 中：可能、也许
- 低：或许、大概

道义情态：
- 高：必须、一定
- 中：应该、需要
- 低：可以、可能

分析要点：
- 情态选择反映说话者的立场
- 高情态表达确定性立场
- 低情态表达谨慎或回避
```

**转换（语态）分析**：
```
主动语态：
- "政府出台了政策"
- 施事明确，责任清晰

被动语态：
- "政策被出台了"
- 施事隐含，责任模糊

无施事被动：
- "问题被解决了"
- 施事缺失，责任转移

名词化：
- "政策的出台促进了发展"
- 过程变成事物，去历史化
```

#### 文本结构分析

**体裁结构**：
```
常见体裁结构：
- 新闻报道：标题-导语-主体-背景-结尾
- 学术论文：引言-文献-方法-结果-讨论
- 政治演讲：开场-问题陈述-解决方案-号召

分析要点：
- 文本遵循什么体裁规约？
- 是否混合多种体裁？
- 结构选择有什么功能？
```

**衔接手段**：
```
词汇衔接：
- 重复：同一词汇重复出现
- 同义：同义词替代
- 上下义：类别与成员

语法衔接：
- 指代：人称代词、指示词
- 替代：do, so, not
- 省略：成分省略
- 连接：连接词

分析要点：
- 衔接如何建构文本连贯性？
- 衔接选择有什么意识形态效果？
```

### 2.3 话语实践层面

**互文性分析**：
```
明显互文性 (Manifest Intertextuality):
- 直接引用："正如马克思所说..."
- 间接引用：转述他人观点

构成互文性 (Constitutive Intertextuality):
- 话语类型的混合
- 学术语言与日常语言的混合
- 不同文体的混合

分析要点：
- 话语中吸收了哪些其他话语？
- 互文性有什么功能？
- 互文性建构了什么权威？
```

**话语链分析**：
```
话语链：
原始事件 → 媒体报道 → 公众讨论 → 政策回应

分析要点：
- 话语如何在不同场域间流动？
- 话语在流动中如何被转换？
- 谁控制话语的流动？
```

### 2.4 社会实践层面

**意识形态分析**：
```
意识形态的定义：
- 支配性社会群体的利益表达
- 自然化的"常识"
- 维持权力关系的信念系统

分析要点：
1. 话语建构了什么"常识"？
2. 这些"常识"服务于谁的利益？
3. 什么是被边缘化或沉默的？
4. 话语如何自然化权力关系？
```

**权力分析**：
```
权力的定义：
- 不是静态资源，而是关系性力量
- 通过话语生产、再生产或挑战

分析要点：
1. 谁有权说话？谁被排除？
2. 话语如何分配社会位置？
3. 话语如何建构"我们"与"他们"？
4. 存在什么抵抗或挑战的可能性？
```

## 第三章：van Dijk社会认知方法

### 3.1 方法概述

```
分析层次：
宏观结构 (Macrostructure)
├── 主题 (Themes)
└── 图式 (Schemas)
    │
微观结构 (Microstructure)
├── 局部语义 (Local Semantics)
├── 局部语法 (Local Syntax)
└── 修辞 (Rhetoric)
    │
认知维度
├── 心智模型 (Mental Models)
├── 知识结构 (Knowledge Structures)
└── 意识形态 (Ideologies)
```

### 3.2 宏观结构分析

**主题分析**：
```
主题提取方法：
1. 自下而上：从命题中概括
2. 自上而下：识别宏观命题

示例：
文本："失业率上升...企业裁员...政府采取措施..."
主题："经济危机及其应对"

意识形态功能：
- 主题组织读者对文本的理解
- 主题选择反映意识形态立场
```

**图式分析**：
```
新闻图式：
├── 标题
├── 导语
├── 主要事件
├── 背景
├── 后果
├── 评论
└── 结论

分析要点：
- 信息在不同图式位置有何不同权重？
- 正面信息放在什么位置？
- 负面信息如何处理？
```

### 3.3 微观结构分析

**局部语义**：
```
命题分析：
- 命题的结构
- 命题间的关系（因果、条件等）

连贯分析：
- 局部连贯：相邻句子
- 整体连贯：与主题的关系

分析要点：
- 什么信息被连接？
- 什么信息被分离？
```

**局部语法**：
```
词序分析：
- 信息焦点
- 前景/背景

句法选择：
- 复杂句/简单句
- 主从关系

分析要点：
- 句法选择有什么效果？
- 如何影响读者的处理？
```

### 3.4 认知维度

**心智模型**：
```
定义：对特定情境的心理表征

分析要点：
- 话语唤起什么心智模型？
- 话语建构什么心智模型？
- 模型如何被操控？
```

**意识形态**：
```
意识形态的定义：
- 基本社会信念
- 组织社会认知
- 维持群体利益

意识形态与话语：
- 意识形态影响话语生产
- 话语再生产意识形态
```

## 第四章：Wodak话语策略框架

### 4.1 五大策略

**1. 指称策略 (Referential Strategy)**：
```
功能：命名与分类社会行动者

分析要点：
- 内群体如何命名？（我们）
- 外群体如何命名？（他们）
- 命名建构什么身份？
- 分类边界如何划定？
```

**2. 谓语策略 (Predicational Strategy)**：
```
功能：赋予特质、定性评价

分析要点：
- 正面定性：美德、能力
- 负面定性：缺陷、威胁
- 定性如何被证明？
```

**3. 论证策略 (Argumentation Strategy)**：
```
功能：正当化论点

论据类型：
- 诉诸权威
- 诉诸因果
- 诉诸历史
- 诉诸数量
- 诉诸类比

分析要点：
- 使用什么论据？
- 论据是否有效？
- 存在什么谬误？
```

**4. 视角策略 (Perspectivization Strategy)**：
```
功能：表达立场

表现形式：
- 显性立场："我认为..."
- 隐性立场：通过词汇选择
- 报道框架：如何引述他人

分析要点：
- 立场如何表达？
- 立场如何被合法化？
```

**5. 强化/弱化策略**：
```
强化手段：
- 强化词："绝对"、"完全"
- 重复
- 排比

弱化手段：
- 模糊语："可能"、"似乎"
- 条件句
- 被动语态

分析要点：
- 什么被强化？
- 什么被弱化？
```

## 第五章：分析实践

### 5.1 分析步骤

```
步骤1：确定分析对象
├── 选择有社会意义的文本
├── 确定分析焦点
└── 收集背景资料

步骤2：文本层面分析
├── 词汇分析
├── 语法分析
└── 结构分析

步骤3：话语实践分析
├── 互文性分析
├── 话语链分析
└── 话语消费分析

步骤4：社会实践分析
├── 意识形态分析
├── 权力分析
└── 社会变革分析

步骤5：整合与解释
├── 三个层面的关联
├── 发现的理论意义
└── 研究局限
```

### 5.2 质量标准

**透明性**：
- 详细描述分析过程
- 提供分析示例
- 说明分析决策

**证据性**：
- 分析有文本证据
- 避免过度解读
- 考虑替代解释

**批判性**：
- 关注权力维度
- 揭示意识形态
- 不预设政治立场
