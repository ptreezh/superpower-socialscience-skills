# 负面案例：文档现实主义分析

## 案例信息
- **分析对象**: 组织年报
- **分析方法**: 文档现实主义
- **问题**: 将文档等同于现实
- **时间**: 2026-03-18

## 错误分析示范

### 研究问题
"组织年报如何反映组织绩效？"

### 错误分析
"根据公司年报数据显示，公司2024年营收增长20%，净利润增长15%。这表明公司经营状况良好，管理效率提高。年报还显示员工满意度达95%，说明人力资源管理卓有成效。"

## 问题诊断

| 问题 | 表现 |
|------|------|
| 文档现实主义 | 直接采信年报数据 |
| 忽视生产语境 | 不分析年报制作过程 |
| 忽视选择性 | 不分析年报省略什么 |
| 忽视受众 | 不分析年报写给谁看 |
| 忽视权力 | 不分析年报的权力维度 |

## 正确分析示范

### 生产语境
```
年报生产过程
├── 谁制作？财务部门+公关部门
├── 为何制作？合规要求+投资者关系
├── 有何压力？股价压力+监管要求
└── 如何选择？选择性呈现数据
```

### 选择性分析
| 包含 | 排除 |
|------|------|
| 营收增长 | 成本压力 |
| 利润数据 | 员工流失率 |
| 正面事件 | 负面舆情 |
| 未来展望 | 风险披露 |

### 受众分析
```
年报受众
├── 投资者: 关注财务数据
├── 监管机构: 关注合规性
├── 媒体: 关注亮点新闻
└── 员工: 关注企业前景
```

### 权力关系
- 公司控制叙事权
- 选择性呈现有利于公司的信息
- 构建"成功"的组织形象

### 综合解读
"年报确实显示营收和利润增长，但需要注意：
1. 年报是公司自我呈现的工具
2. 增长数据可能是选择性呈现
3. 需要交叉验证其他来源
4. 员工满意度数据来源和测量方式需审视
5. 年报建构了积极的组织形象"

## 教训总结

1. ❌ 文档不等同于现实
2. ❌ 必须分析生产语境
3. ❌ 必须分析选择性
4. ❌ 必须分析受众
5. ✅ 批判性阅读是底线

---

**警示等级**: ⚠️ 典型错误
