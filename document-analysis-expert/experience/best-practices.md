# 文档分析最佳实践

## 核心原则

### 原则1：批判性阅读
```
文档不是透明的窗口
- 文档建构现实
- 文档有选择性
- 文档有权力维度
```

### 原则2：语境敏感性
```
文档意义依赖于语境
- 生产语境
- 流通语境
- 使用语境
```

### 原则3：三角验证
```
单一文档不足以说明问题
- 多文档比较
- 与其他资料对照
- 结合访谈观察
```

---

## 分析流程

### 第一步：文档收集
1. 确定收集范围
2. 系统收集相关文档
3. 记录来源和获取方式

### 第二步：初步评估
1. Scott四项评估
2. 分类整理
3. 建立文档清单

### 第三步：深度分析
1. 生产语境分析
2. 内容结构分析
3. 流通使用分析

### 第四步：批判反思
1. 选择性分析
2. 权力关系分析
3. 意义建构分析

### 第五步：综合解释
1. 模式识别
2. 理论联系
3. 限制说明

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 文档现实主义 | 批判性阅读 |
| 忽视语境 | 追踪生产过程 |
| 忽视选择性 | 分析省略 |
| 忽视受众 | 分析目标/实际受众 |
| 忽视权力 | 批判档案权力 |
