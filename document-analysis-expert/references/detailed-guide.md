## 第一章：文档分析概论

### 1.1 学科定义

文档分析是社会学中研究各类文档（政策文件、组织文档、个人文献等）的方法，关注文档的社会生命和现实建构功能。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 早期 | Scott | 文档证据理论 |
| 发展 | Prior | 文档社会生命 |
| 建构主义 | Atkinson & Coffey | 文档现实建构 |
| 当代 | Silverman | 文档质性分析 |

### 1.3 核心概念

```
文档分析体系
├── 研究对象
│   ├── 官方文件
│   ├── 组织文档
│   ├── 个人文献
│   └── 媒体文本
├── 分析维度
│   ├── 内容分析
│   ├── 形式分析
│   ├── 语境分析
│   └── 功能分析
└── 理论视角
    ├── 现实描述
    ├── 现实建构
    └── 现实调节
```

## 第二章：文档类型与方法

### 2.1 Scott文档评估

**真实性评估**:
```
问题：
- 文档是否为原件？
- 是否经过篡改？
- 来源是否可靠？
```

**可信性评估**:
```
问题：
- 内容是否准确？
- 是否有偏见？
- 与其他证据是否一致？
```

**代表性评估**:
```
问题：
- 是否典型？
- 是否有系统性？
- 是否覆盖研究对象？
```

### 2.2 文档类型分析

| 类型 | 来源 | 特征 | 方法 |
|------|------|------|------|
| 政策文件 | 政府 | 规范性、权威性 | 话语分析 |
| 组织文档 | 机构 | 内部性、功能性 | 内容分析 |
| 私人文献 | 个人 | 主观性、私密性 | 叙事分析 |
| 媒体文本 | 媒体 | 公开性、传播性 | 框架分析 |

### 2.3 分析步骤

```
文档分析流程
├── 1. 文档收集
│   ├── 确定收集范围
│   ├── 系统收集
│   └── 记录来源
├── 2. 初步处理
│   ├── 分类整理
│   ├── 真实性检验
│   └── 背景调查
├── 3. 深度分析
│   ├── 内容分析
│   ├── 形式分析
│   └── 语境分析
└── 4. 综合解释
    ├── 模式识别
    ├── 理论联系
    └── 意义建构
```

## 第三章：文档的社会生命

### 3.1 生产语境

**分析问题**:
- 谁生产了文档？
- 为何生产？
- 在何条件下生产？
- 有何限制和压力？

### 3.2 流通过程

**分析问题**:
- 文档如何传播？
- 目标受众是谁？
- 实际受众是谁？
- 流通中有何变化？

### 3.3 使用与效果

**分析问题**:
- 如何被使用？
- 有何效果？
- 如何被解读？
- 如何影响实践？

### 3.4 存档与遗忘

**分析问题**:
- 为什么被保存？
- 谁决定保存？
- 什么被遗忘？
- 档案有何权力？
