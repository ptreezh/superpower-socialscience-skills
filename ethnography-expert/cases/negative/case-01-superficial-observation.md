# 负面案例：快餐式田野调查

## 案例概述

**研究主题**: 某企业组织文化研究
**田野时间**: 2周 (宣称"民族志研究")
**研究者**: 张研究员
**问题类型**: 方法论违规

---

## 一、问题背景

某商业咨询公司受委托对一家企业进行"民族志研究"，声称研究组织文化，为企业转型提供依据。实际田野工作仅2周，却宣称使用了民族志方法。

---

## 二、违规行为分析

### 违规1: 田野时间不足

**行为**:
- 仅在企业停留2周
- 每天工作时间不足4小时
- 总田野时间约40小时

**问题**:
```
❌ 违反Malinowski传统: 民族志要求长期田野(6个月以上)
❌ 无法获得深度理解: 浅层观察无法揭示文化意义
❌ 无法建立信任关系: 短期停留难以深入
```

**后果**:
- 研究发现停留在表面
- 忽视深层文化模式
- 结论缺乏说服力

### 违规2: 肤浅观察

**行为**:
- 仅观察正式会议和工作流程
- 不参与非正式互动
- 不了解员工日常经验

**问题**:
```
❌ 违反"禁止肤浅观察"原则
❌ 未进行真正的参与观察
❌ 只看到组织展示的一面
```

**遗漏内容**:
| 应观察 | 实际观察 |
|--------|---------|
| 茶水间闲聊 | ❌ |
| 加班时的氛围 | ❌ |
| 非正式小团体 | ❌ |
| 领导不在时的行为 | ❌ |
| 正式会议 | ✓ |

### 违规3: 单一视角

**行为**:
- 主要采访管理层
- 员工访谈仅3人
- 未获得不同层级视角

**问题**:
```
❌ 违反"禁止单一视角"原则
❌ 缺少主位视角(普通员工的声音)
❌ 研究结论偏向管理层立场
```

**视角对比**:
| 视角 | 来源 | 是否获得 |
|------|------|---------|
| 高层管理者 | 5次访谈 | ✓ |
| 中层管理者 | 3次访谈 | ✓ |
| 普通员工 | 3次访谈 | ⚠ 不足 |
| 一线工人 | 0次访谈 | ❌ |

### 违规4: 预设结论

**行为**:
- 研究前已确定"企业文化需要变革"结论
- 选择性收集支持结论的证据
- 忽视反对证据

**问题**:
```
❌ 违反"禁止预设结论"原则
❌ 让田野材料服务于预设结论
❌ 违背民族志发现的基本精神
```

**证据使用对比**:
| 证据类型 | 是否使用 |
|---------|---------|
| 支持预设的 | ✓ |
| 反对预设的 | ❌ 忽视 |
| 中性的 | ❌ 忽视 |

### 违规5: 脱离情境

**行为**:
- 不了解企业历史
- 不了解行业背景
- 不了解员工生活境况

**问题**:
```
❌ 违反"禁止脱离情境"原则
❌ 文化意义需要情境理解
❌ 脱离情境的解释是空洞的
```

### 违规6: 薄描写作

**行为**:
- 报告以数据和图表为主
- 缺少具体场景描述
- 不解释行为的文化意义

**问题**:
```
❌ 违反"禁止单薄描述"原则
❌ 未能进行深描
❌ 仅提供功能性和统计性描述
```

**薄描示例**:
> "调查显示，78%的员工对工作环境不满意。"

**应有的深描**:
> "在开放式办公区的角落，小李每天都在同一张桌子前工作，尽管周围人来人往，他却很少与人交流。他说：'以前我们有自己的格子间，虽然小，但是有归属感。现在像是在公共场所，总是感觉不自在。'这种对空间的重新安排，表面上是提高效率，实际上却改变了一种关于'我的工作空间'的文化理解..."

---

## 三、后果与影响

### 学术后果

- 研究被学术期刊拒稿
- 方法论受到同行批评
- 损害民族志方法声誉

### 实践后果

- 企业转型建议不切实际
- 员工对研究产生不信任
- 浪费研究资源

### 伦理后果

- 参与者时间被浪费
- 研究真实性受到质疑
- 违反研究诚信

---

## 四、正确做法对比

| 方面 | 错误做法 | 正确做法 |
|------|---------|---------|
| 田野时间 | 2周 | 至少6个月 |
| 观察深度 | 正式场合 | 全面参与 |
| 视角多样性 | 管理层主导 | 多层级视角 |
| 结论形成 | 预设结论 | 田野引导 |
| 情境理解 | 忽视背景 | 深入情境 |
| 描述方式 | 数据罗列 | 深描叙事 |

---

## 五、教训总结

### 方法论教训

1. **时间投入不可妥协**
   - 民族志的核心是长期田野工作
   - 没有时间就没有深度

2. **参与观察不可替代**
   - 不是旁观者的观察
   - 需要真正的参与

3. **视角多元不可忽视**
   - 不能只听一方声音
   - 主位/客位需要对话

### 伦理教训

1. **诚实面对局限**
   - 承认研究不足
   - 不夸大研究发现

2. **尊重参与者**
   - 认真对待他们的时间
   - 真实呈现他们的声音

### 实践教训

1. **选择适合的方法**
   - 不是所有研究都适合民族志
   - 短期调研应使用其他方法

2. **坚持方法论标准**
   - 不能因客户压力妥协
   - 方法论标准是底线

---

## 六、正确案例参考

如果需要进行此类研究，应：

1. **调整时间预期**: 至少6个月田野
2. **扩展观察范围**: 包括非正式互动
3. **多元视角**: 系统采访各层级员工
4. **开放心态**: 让田野材料引导发现
5. **深入情境**: 了解企业历史和行业背景
6. **深描写作**: 用叙事呈现文化意义
