## 一、田野进入经验模式

### 成功模式

1. **渐进式融入**
   - 初期以学习者身份进入
   - 参与社区日常活动
   - 通过中介人建立信任
   - 逐步深入核心活动

2. **礼物交换策略**
   - 了解当地礼赠规范
   - 适度回馈社区
   - 避免过度补偿造成依赖

3. **语言学习**
   - 学习基本问候和常用语
   - 表达对当地文化的尊重
   - 通过语言建立亲密关系

### 失败教训

| 情境 | 问题 | 教训 |
|------|------|------|
| 过度介入 | 改变社区动态 | 保持观察者角色平衡 |
| 承诺过多 | 无法兑现 | 谨慎承诺，诚实沟通 |
| 文化误解 | 造成冒犯 | 事先了解文化禁忌 |

## 二、田野笔记经验模式

### 高效记录策略

1. **分层记录法**
   ```
   田野笔记结构:
   ├── 即时笔记 (关键词、速记)
   ├── 扩展笔记 (当日完善)
   ├── 分析笔记 (每周整理)
   └── 理论笔记 (持续反思)
   ```

2. **四类笔记平衡**
   - 描述性: 40% (事实记录)
   - 方法论: 20% (研究过程)
   - 理论性: 25% (分析思考)
   - 情感性: 15% (个人反思)

3. **田野速记技巧**
   - 使用缩写系统
   - 关键词捕捉
   - 录音辅助(需许可)
   - 当日整理不拖延

### 常见记录错误

- ❌ 过度依赖记忆
- ❌ 描述与分析混淆
- ❌ 忽视研究者情感反应
- ❌ 单一来源确认

## 三、关键报道人管理经验

### 选择标准

| 类型 | 特点 | 适合角色 |
|------|------|---------|
| 文化专家 | 深谙传统知识 | 文化解释 |
| 边缘人 | 跨越多个世界 | 比较视角 |
| 普通成员 | 代表日常经验 | 典型案例 |
| 批评者 | 对文化有反思 | 另类视角 |

### 关系维护原则

1. **互惠原则**
   - 提供知识回馈
   - 尊重对方时间
   - 适度物质回报

2. **透明原则**
   - 明确研究目的
   - 分享研究发现
   - 获得知情同意

3. **尊重原则**
   - 保护隐私
   - 尊重文化边界
   - 不强迫回答

## 四、文化模式识别经验

### 模式发现路径

```
观察 → 初步命名 → 验证观察 → 修正命名 → 深描
```

### 模式类型库

| 类型 | 特征 | 示例 |
|------|------|------|
| 行为模式 | 反复出现的行动 | 问候仪式 |
| 语言模式 | 重复的话语结构 | 故事讲述 |
| 关系模式 | 固定的互动方式 | 长幼关系 |
| 空间模式 | 空间使用规律 | 公私领域 |
| 时间模式 | 时间组织方式 | 节庆周期 |

### 模式验证方法

1. **频率检验**: 是否持续出现？
2. **分布检验**: 是否在不同情境出现？
3. **意义检验**: 当地人是否认同？
4. **比较检验**: 是否区别于其他文化？

## 五、深描写作经验

### 深描层次

```
第一层: 行为描述 (发生了什么)
第二层: 情境说明 (在什么情况下)
第三层: 意义解释 (对当地人意味着什么)
第四层: 理论对话 (与更大社会结构的关联)
```

### 深描检验清单

- [ ] 是否描述了具体情境？
- [ ] 是否呈现了当地人视角？
- [ ] 是否解释了文化意义？
- [ ] 是否使用了当地术语？
- [ ] 是否展示了复杂性？
- [ ] 是否避免了过度简化？

### 写作禁忌

- ❌ 薄描: 只描述行为不解释意义
- ❌ 过度概括: 忽视个案差异
- ❌ 客观伪装: 隐藏研究者位置
- ❌ 文化本质化: 忽视文化变迁

## 六、伦理困境处理经验

### 常见伦理问题

| 问题 | 处理原则 |
|------|---------|
| 知情同意 | 持续的同意过程，不只是签字 |
| 隐私保护 | 匿名化处理，情境伪装 |
| 利益分配 | 研究成果回馈社区 |
| 敏感信息 | 谨慎处理，可能不发表 |
| 退出机制 | 尊重参与者随时退出权利 |

### 伦理决策框架

```
识别利益相关者 → 评估潜在影响 → 咨询伦理委员会 → 
选择最小伤害方案 → 记录决策过程 → 反思与调整
```

## 七、反身性实践经验

### 反身性维度

1. **情境反身性**
   - 研究者在场如何影响情境？
   - 如何最小化干扰？

2. **个人反身性**
   - 研究者的身份如何影响理解？
   - 个人偏见如何影响分析？

3. **认识论反身性**
   - 研究预设如何影响发现？
   - 理论框架是否限制视野？

### 反身性记录模板

```
日期: 
田野情境: 
我的角色: 
我的感受: 
可能的偏见: 
对研究的影响: 
反思与调整: 
```
