## 第一部分：民族志基础

### 1.1 什么是民族志

民族志(Ethnography)是一种质性研究方法，通过长期田野调查深入理解社会文化现象。其核心特征包括：

- **长期田野工作**: 通常6-24个月
- **参与观察**: 深入日常生活
- **文化解释**: 理解行为的意义
- **整体视角**: 关注文化整体性

### 1.2 历史发展

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 早期 | Boas, Malinowski | 确立田野工作传统 |
| 经典时期 | Evans-Pritchard, Geertz | 发展解释性方法 |
| 反思时期 | Clifford, Marcus | 批判与实验民族志 |
| 当代 | Hammersley, Atkinson | 方法论规范化 |

### 1.3 核心概念

**文化**
- 共享的意义系统
- 行为的解释框架
- 地方性知识

**深描(Thick Description)**
- Geertz的核心概念
- 不仅描述行为，更解释意义
- 揭示行为背后的文化逻辑

**主位/客位(Emic/Etic)**
- 主位: 当地人视角
- 客位: 分析者视角
- 需要：两种视角的对话

## 第二部分：田野工作准备

### 2.1 研究问题发展

**问题类型**:
- 描述性问题: 这个群体的文化是怎样的？
- 解释性问题: 为什么他们这样做？
- 过程性问题: 这种实践如何演变？

**问题特征**:
- 开放性
- 情境敏感性
- 理论相关性

### 2.2 文献回顾

**回顾范围**:
1. 理论文献
2. 方法论文献
3. 地区/群体研究
4. 相关主题研究

**回顾成果**:
- 理论框架
- 研究缺口
- 方法借鉴
- 比较基准

### 2.3 进入策略

**进入途径**:
| 途径 | 优势 | 风险 |
|------|------|------|
| 正式渠道 | 合法性强 | 可能被限制 |
- 非正式渠道 | 灵活性高 | 合法性不确定 |
- 中介人 | 快速建立信任 | 可能被边缘化 |
- 自我介绍 | 独立性强 | 建立信任慢 |

**进入原则**:
- 诚实说明研究目的
- 尊重当地规范
- 建立互惠关系
- 保持灵活性

### 2.4 伦理准备

**伦理审查**:
- 机构审查委员会(IRB)审批
- 知情同意设计
- 风险评估
- 数据管理计划

**伦理原则**:
1. 尊重自主性
2. 不伤害原则
3. 利益原则
4. 公正原则

## 第三部分：田野工作实施

### 3.1 参与观察

**观察层次**:
```
Level 1: 描述性观察 (发生了什么)
Level 2: 聚焦观察 (关注特定现象)
Level 3: 选择性观察 (验证初步发现)
```

**观察内容**:
- 空间: 物理环境如何组织
- 行动者: 谁参与其中
- 活动: 他们在做什么
- 物品: 使用什么物品
- 行动: 具体行为是什么
- 事件: 发生了什么事件
- 时间: 活动如何组织
- 目标: 试图达成什么
- 感受: 参与者的情感

**参与程度选择**:

| 角色 | 描述 | 适用 |
|------|------|------|
| 完全观察者 | 不参与，只观察 | 公共场所 |
| 观察者即参与者 | 主要观察，有限参与 | 特定活动 |
| 参与者即观察者 | 主要参与，同时观察 | 深入研究 |
| 完全参与者 | 完全融入 | 隐蔽研究 |

### 3.2 田野笔记

**笔记类型**:

1. **描述性笔记**
   - 事实记录
   - 感官细节
   - 直接引语
   - 时间顺序

2. **方法论笔记**
   - 研究过程记录
   - 观察策略调整
   - 进入问题记录

3. **理论笔记**
   - 初步分析
   - 概念发展
   - 模式识别

4. **情感笔记**
   - 研究者感受
   - 反身性记录
   - 偏见意识

**记录流程**:
```
田野速记 (即时) → 扩展笔记 (当日) → 
编码整理 (每周) → 分析备忘录 (持续)
```

**笔记原则**:
- 具体而非抽象
- 区分观察与推论
- 记录研究者的角色
- 保存原始语言

### 3.3 深度访谈

**访谈类型**:
- 口述史: 生命历程叙事
- 半结构化: 有框架但灵活
- 焦点小组: 群体互动视角
- 随访访谈: 追踪变化

**访谈技巧**:
1. 开放式问题
2. 追问和澄清
3. 沉默的运用
4. 情感回应
5. 使用当地术语

**访谈记录**:
- 录音(需同意)
- 笔记
- 非语言观察
- 访谈后立即总结

### 3.4 关键报道人

**选择标准**:
- 对文化有深入了解
- 愿意分享知识
- 有代表性
- 有特殊视角

**关系维护**:
- 建立信任
- 互惠互利
- 尊重边界
- 长期联系

## 第四部分：文化分析

### 4.1 文化模式识别

**模式类型**:

| 类型 | 特征 | 示例 |
|------|------|------|
| 行为模式 | 反复出现的行为 | 问候仪式 |
| 语言模式 | 重复的话语结构 | 故事讲述 |
| 关系模式 | 固定的互动方式 | 长幼关系 |
| 空间模式 | 空间使用规律 | 公私领域 |
| 时间模式 | 时间组织方式 | 节庆周期 |

**识别过程**:
```
观察 → 初步命名 → 验证观察 → 修正命名 → 深描
```

**验证方法**:
1. 频率检验
2. 分布检验
3. 成员检验
4. 比较检验

### 4.2 主位/客位分析

**主位分析**:
- 使用当地人的概念
- 理解当地人的分类
- 探索当地的逻辑
- 保持内部一致性

**客位分析**:
- 使用学科概念
- 建立理论联系
- 进行跨文化比较
- 发展理论贡献

**对话原则**:
- 不让任何一方独占
- 展示两种视角的张力
- 寻找理解的桥梁
- 保持开放性

### 4.3 深描写作

**深描层次**:
```
第一层: 行为描述 (发生了什么)
第二层: 情境说明 (在什么情况下)
第三层: 意义解释 (对当地人意味着什么)
第四层: 理论对话 (与更大社会结构的关联)
```

**深描要素**:
- 具体场景
- 行动者视角
- 文化意义
- 社会背景
- 理论对话

**避免薄描**:
- 不只是行为罗列
- 不只是功能说明
- 不只是数字统计
- 不只是客观描述

### 4.4 三角验证

**验证类型**:

| 类型 | 方法 | 目的 |
|------|------|------|
| 数据三角 | 多来源数据 | 减少偏见 |
| 方法三角 | 多种收集方法 | 交叉验证 |
| 研究者三角 | 多研究者 | 多元视角 |
| 理论三角 | 多理论解释 | 避免单一 |

## 第五部分：质量与伦理

### 5.1 质量标准

**Guba & Lincoln标准**:

| 标准 | 定义 | 实现策略 |
|------|------|---------|
| 可信性 | 真实性 | 三角验证、成员检验 |
| 可转移性 | 可应用性 | 深描、情境描述 |
| 可依赖性 | 一致性 | 审计追踪 |
| 可确认性 | 客观性 | 反思性、透明度 |

**可信性策略**:
1. 长期田野
2. 三角验证
3. 同行讨论
4. 成员检验
5. 反例分析
6. 深描

### 5.2 伦理考量

**核心伦理问题**:

1. **知情同意**
   - 持续的同意过程
   - 语言和文化的适应
   - 对无文字群体的考量

2. **隐私与匿名**
   - 情境匿名化
   - 个体匿名化
   - 识别风险

3. **利益分配**
   - 研究成果回馈
   - 知识产权
   - 经济利益

4. **敏感信息**
   - 处理原则
   - 发表决策
   - 保护措施

**伦理决策框架**:
```
识别问题 → 分析利益相关者 → 评估选项 → 
选择行动 → 实施与监控 → 反思与调整
```

### 5.3 反身性

**反身性维度**:

1. **情境反身性**
   - 研究者在场的影响
   - 如何最小化干扰
   - 研究关系的性质

2. **个人反身性**
   - 身份对理解的影响
   - 个人偏见
   - 情感反应

3. **认识论反身性**
   - 研究预设
   - 理论框架的限制
   - 知识的边界

**反身性实践**:
- 保持反思日志
- 定期自我审视
- 寻求同行反馈
- 在写作中呈现

## 第六部分：民族志类型

### 6.1 传统民族志

**特征**:
- 长期田野(12-24个月)
- 单一地点
- 深入整体文化
- 经典写作模式

**适用场景**:
- 文化描述研究
- 社区研究
- 组织文化研究

### 6.2 快速民族志

**特征**:
- 缩短田野时间(2-6个月)
- 聚焦特定问题
- 使用多种方法加速
- 团队合作

**适用场景**:
- 评估研究
- 需求分析
- 政策研究

### 6.3 多点民族志

**特征**:
- 跨地点追踪
- 关注流动和联结
- 非单一地点
- Marcus传统

**追踪策略**:
| 策略 | 方法 |
|------|------|
| 跟随人 | 追踪个体移动 |
| 跟随物 | 追踪物品流通 |
| 跟随隐喻 | 追踪概念变迁 |
| 跟随生命 | 追踪生命历程 |
| 跟随冲突 | 追踪争议发展 |

### 6.4 数字民族志

**特征**:
- 在线田野
- 数字媒体研究
- 虚拟社区
- 混合方法

**方法适应**:
| 传统方法 | 数字方法 |
|---------|---------|
| 参与观察 | 潜水观察/参与互动 |
| 深度访谈 | 在线访谈 |
| 田野笔记 | 截图/数字档案 |
| 关键报道人 | 社区活跃成员 |

## 第七部分：写作与发表

### 7.1 民族志写作

**写作风格**:
- 深描优先
- 叙事性呈现
- 研究者可见
- 理论对话

**结构要素**:
```
引言: 研究背景和问题
文献回顾: 理论框架
方法: 田野工作过程
发现: 文化描述和分析
讨论: 理论贡献和意义
结论: 总结和展望
```

### 7.2 伦理写作

**写作伦理**:
- 保护参与者身份
- 情境匿名化
- 获得发表许可
- 考虑后果

**成员检验**:
- 发现反馈
- 解释验证
- 修正建议
- 发表同意
