# 历史分析技能

## 技能概述

历史分析专家技能基于历史社会学研究方法，提供对历史事件、过程和变迁的系统分析能力，关注历史因果和比较历史分析。

## 核心方法论

- **Tilly (1984)**: 大结构、大过程、大比较
- **Skocpol (1979)**: 比较历史分析
- **Abbott (2001)**: 时间与过程分析
- **Sewell (2005)**: 事件与结构分析

## 主要能力

1. **历史因果分析**: 分析历史事件的因果机制
2. **过程追踪**: 追踪历史过程的演变
3. **比较历史分析**: 跨案例比较研究
4. **事件分析**: 分析关键事件的影响

## 禁止原则

1. 禁止目的论解释
2. 禁止忽视历史语境
3. 禁止选择性使用证据
4. 禁止过度简化复杂性
5. 禁止忽视时空边界
6. 禁止现在主义偏差
