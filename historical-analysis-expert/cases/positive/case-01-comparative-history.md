# 正面案例：比较历史分析研究

## 案例信息
- **研究对象**: 社会革命比较
- **分析方法**: 比较历史+过程追踪
- **时间**: 2026-03-18

## 研究设计

### 研究问题
"为什么某些国家发生社会革命，而其他国家没有？"

### 比较策略
采用**最相似系统设计**：
- 选择经济、文化相似的国家
- 因变量变异：革命发生 vs 未发生
- 目标：识别关键差异因素

## 案例选择

| 案例 | 结果 | 经济水平 | 国际压力 |
|------|------|---------|---------|
| 国家A | 发生革命 | 中等 | 高 |
| 国家B | 发生革命 | 中等 | 高 |
| 国家C | 未发生革命 | 中等 | 高 |
| 国家D | 未发生革命 | 中等 | 高 |

## 过程追踪

### 革命国家(国家A)因果链
```
国际压力 → 国家财政危机 → 国家能力削弱
→ 精英分裂 → 政治机会开放 → 农民动员
→ 革命爆发
```

**关键机制**:
- 国家财政危机：战争失败导致财政崩溃
- 精英分裂：改革派与保守派对立
- 政治机会：国家镇压能力下降

### 非革命国家(国家C)对比
```
国际压力 → 国家财政压力 → 改革应对
→ 精英妥协 → 政治稳定 → 渐进变革
```

**差异机制**:
- 国家应对：成功进行财政改革
- 精英关系：达成政治妥协
- 农民状态：未能形成跨区域联盟

## 关键发现

### 必要条件分析
| 条件 | 革命案例 | 非革命案例 | 是否必要 |
|------|---------|-----------|---------|
| 国家危机 | ✓ | ✓ | - |
| 精英分裂 | ✓ | ✗ | 可能必要 |
| 农民动员 | ✓ | ✗ | 可能必要 |

### 因果机制
**国家崩溃 → 政治机会 → 农民动员 → 革命**

## 限制说明

### 结论边界
- 适用于类似历史时期的案例
- 不适用于当代政治变迁分析
- 结论为概率性，非确定性
