# 历史分析最佳实践

## 核心原则

### 原则1：反目的论
```
历史没有预定目标
- 不假设必然结果
- 分析多种可能性
- 承认偶然性
```

### 原则2：语境敏感
```
历史事件必须置于语境中理解
- 理解时代背景
- 考量当时条件
- 避免时代错置
```

### 原则3：证据透明
```
证据使用必须透明
- 系统收集
- 全面报告
- 批判评估
```

---

## 分析流程

### 第一步：问题定义
1. 明确研究问题
2. 确定时空范围
3. 选择分析框架

### 第二步：案例选择
1. 确定比较策略
2. 选择案例
3. 说明选择理由

### 第三步：证据收集
1. 多源收集
2. 批判评估
3. 系统整理

### 第四步：因果分析
1. 识别因果链
2. 追踪机制
3. 检验替代解释

### 第五步：比较分析
1. 找出相似/差异
2. 识别关键因素
3. 检验稳健性

### 第六步：限制说明
1. 证据限制
2. 方法限制
3. 结论边界

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 目的论 | 分析多种可能 |
| 忽视语境 | 历史地理解 |
| 选择性证据 | 全面收集报告 |
| 过度简化 | 保留复杂性 |
| 忽视边界 | 明确适用范围 |
| 现在主义 | 避免时代错置 |
