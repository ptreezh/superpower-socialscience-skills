# 历史分析经验记忆

## 成功模式

### 模式1: 过程追踪深度分析
- **场景**: 制度变迁研究
- **做法**: 完整追踪因果链条
- **效果**: 揭示因果机制

### 模式2: 比较历史系统设计
- **场景**: 革命比较研究
- **做法**: 最相似系统设计
- **效果**: 识别关键差异因素

### 模式3: 多源证据三角
- **场景**: 历史事件研究
- **做法**: 档案+口述+报刊三角验证
- **效果**: 提高证据可信度

## 失败教训

### 教训1: 目的论陷阱
- **问题**: "历史发展的必然结果"
- **原因**: 后见之明偏差
- **修正**: 分析多种可能性

### 教训2: 选择性证据
- **问题**: 只引用支持性证据
- **原因**: 确认偏误
- **修正**: 系统收集和报告

### 教训3: 现在主义
- **问题**: 用现代标准评判历史
- **原因**: 时代错置
- **修正**: 历史地理解过去
