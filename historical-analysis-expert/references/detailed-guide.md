## 第一章：历史分析概论

### 1.1 学科定义

历史分析是历史社会学的核心方法，旨在通过对历史事件和过程的系统分析，揭示社会变迁的因果机制和规律。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 经典 | Weber, Marx | 历史比较方法 |
| 结构主义 | Skocpol, Tilly | 比较历史分析 |
| 过程转向 | Abbott | 时间序列分析 |
| 事件转向 | Sewell | 事件与结构 |

### 1.3 核心概念

```
历史分析体系
├── 分析对象
│   ├── 历史事件
```

## 第二章：因果分析方法

### 2.1 因果类型

**链式因果**:
```
A → B → C → 结果
每一步都是必要条件
可通过过程追踪验证
```

**并发因果**:
```
A ─┐
B ─┼→ 结果
C ─┘
多因素共同作用
需比较分析识别
```

**结构因果**:
```
制度结构 → 行为约束 → 历史结果
强调结构性因素
需制度分析
```

### 2.2 过程追踪

**方法类型**:
| 类型 | 目的 | 证据需求 |
|------|------|---------|
| 理论检验 | 检验因果机制 | 链条各环证据 |
| 理论建构 | 发现新机制 | 开放性收集 |
| 因果解释 | 解释特定结果 | 完整证据链 |

**证据类型**:
- 足迹证据: 机制运行的痕迹
- 诊断证据: 排除替代解释
- 模式证据: 序列一致性

### 2.3 反事实推理

```
反事实问题：
如果X没有发生，Y是否仍会发生？

评估方法：
1. 历史合理性
2. 理论一致性
3. 证据支持度
```

## 第三章：比较历史分析

### 3.1 比较策略

**最相似系统设计**:
```
案例选择：在关键方面相似
分析目标：找出差异原因

例子：
国家A (革命) ─┐
              ├→ 差异：？
国家B (无革命) ─┘
控制：经济发展、国际压力
```

**最不同系统设计**:
```
案例选择：在多方面不同
分析目标：找出共同因素

例子：
国家A (革命) ─┐
              ├→ 共同：？
国家B (革命) ─┘
差异：文化、经济、地理
```

### 3.2 案例选择

| 标准 | 说明 |
|------|------|
| 因变量变异 | 结果有差异 |
| 可比性 | 案例可比较 |
| 信息可得性 | 证据可获取 |
| 理论相关性 | 与理论相关 |

## 第四章：事件与结构分析

### 4.1 Sewell事件理论

**核心概念**:
```
结构：相对稳定的制度和文化模式
事件：打破常规的历史时刻
转化：事件引发的结构变迁
```

**事件分析框架**:
1. 事件的发生条件
2. 事件的展开过程
3. 事件的结构效应

### 4.2 关键节点分析

```
关键节点特征：
├── 高不确定性
├── 多种可能性
├── 长期影响
└── 路径依赖起点
```
