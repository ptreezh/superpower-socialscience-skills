# 负面案例：忽视伦理的网络研究

## 案例信息
- **研究对象**: 社交媒体用户
- **分析方法**: 数据抓取
- **问题**: 忽视伦理和隐私
- **时间**: 2026-03-18

## 错误研究示范

### 研究问题
"社交媒体用户如何讨论敏感话题？"

### 错误做法

#### 数据收集
```python
# 直接抓取用户内容
data = scrape_platform("敏感话题")
# 不做任何处理直接使用
save_data(data, "raw_data.json")
```

#### 分析呈现
```
"用户@JohnDoe在2024年3月15日发帖说：
'我正在经历心理健康问题...'"
```

## 问题诊断

| 问题 | 表现 |
|------|------|
| 无知情同意 | 未获取参与者同意 |
| 隐私侵犯 | 直接引用用户名和内容 |
| 风险忽视 | 敏感话题可能造成伤害 |
| 平台违规 | 违反平台使用条款 |

## 后果分析

### 对研究对象
- 隐私泄露风险
- 潜在污名化
- 心理伤害可能

### 对研究质量
- 数据可信度受质疑
- 研究伦理审查不通过
- 学术声誉受损

## 正确做法示范

### 伦理审查
```
1. 评估研究风险
2. 制定隐私保护方案
3. 申请伦理委员会批准
```

### 数据收集
```python
# 获取知情同意
consent = get_informed_consent(participants)
# 匿名化处理
data = scrape_platform("话题")
data = anonymize(data)  # 移除用户名等标识
```

### 呈现方式
```
"一位匿名用户分享道：
'我正在经历心理健康问题...'"
```

## 教训总结

1. ❌ 伦理不是可选项
2. ❌ 公开不等于可用于研究
3. ❌ 敏感话题需特别谨慎
4. ✅ 研究前必须伦理审查
5. ✅ 隐私保护是底线
