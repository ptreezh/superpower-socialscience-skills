# 互联网研究最佳实践

## 核心原则

### 原则1：伦理优先
```
伦理不是可选项，是底线。
- 知情同意
- 隐私保护
- 风险最小化
```

### 原则2：方法多元
```
单一方法不足以理解网络现象。
- 质性+量化结合
- 多数据源验证
- 多方法三角
```

### 原则3：批判反思
```
保持对技术和平台的批判视角。
- 平台不是中立的
- 数据有偏见
- 技术在变化
```

---

## 研究流程

### 第一步：确定研究问题
1. 明确研究焦点
2. 评估可行性
3. 识别伦理问题

### 第二步：选择方法
1. 匹配问题性质
2. 考虑数据可及性
3. 评估资源需求

### 第三步：伦理审查
1. 风险评估
2. 知情同意设计
3. 隐私保护方案
4. 申请伦理批准

### 第四步：数据收集
1. 遵守平台规则
2. 记录收集过程
3. 安全存储数据

### 第五步：数据分析
1. 选择分析方法
2. 质量控制
3. 限制说明

### 第六步：呈现结果
1. 匿名化呈现
2. 透明披露
3. 伦理考量说明

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 忽视伦理 | 先做伦理审查 |
| 单一方法 | 多方法三角 |
| 数据决定论 | 讨论数据偏见 |
| 忽视平台权力 | 批判分析平台 |
| 技术过时 | 定期更新方法 |
| 忽视数字鸿沟 | 讨论代表性 |
