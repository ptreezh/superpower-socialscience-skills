## 第一章：互联网研究概论

### 1.1 学科定义

互联网研究是对网络空间、网络行为和网络文化的跨学科研究，结合社会学、传播学、人类学和信息科学的方法。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 早期 | Turkle | 网络身份研究 |
| 发展 | Hine | 虚拟民族志 |
| 数据化 | Rogers | 数字方法 |
| 当代 | Bruns | 社交媒体研究 |

### 1.3 核心概念

```
互联网研究体系
├── 研究对象
│   ├── 网络空间
│   ├── 网络行为
│   └── 网络文化
├── 研究方法
│   ├── 虚拟民族志
│   ├── 数字方法
│   └── 在线访谈
└── 研究伦理
    ├── 知情同意
    ├── 隐私保护
    └── 数据安全
```

## 第二章：虚拟民族志

### 2.1 方法论基础

**核心假设**:
- 网络空间是文化场所
- 在线行为有社会意义
- 可以进行参与观察

**研究设计**:
```
1. 选择研究场所
   └── 社区、平台、游戏空间
2. 进入田野
   └── 注册、观察、参与
3. 数据收集
   └── 观察、访谈、文档收集
4. 分析与书写
   └── 文化解读、理论建构
```

### 2.2 数据收集

| 方法 | 描述 | 考量 |
|------|------|------|
| 潜水观察 | 不参与的观察 | 公开性考量 |
| 参与观察 | 参与式研究 | 角色披露 |
| 网络访谈 | 在线交流 | 同步/异步 |
| 文档收集 | 截图、存档 | 隐私保护 |

### 2.3 分析策略

- 内容分析
- 话语分析
- 叙事分析
- 符号互动分析

## 第三章：数字方法

### 3.1 Rogers数字方法

**核心原则**:
- 跟随媒介：利用平台原生方法
- 平台作为来源：平台提供研究素材
- 数字痕迹：利用自然产生的数据

**方法工具**:
```
数字方法工具箱
├── 搜索查询分析
├── 链接分析
├── 社交网络分析
├── 地理映射
└── 时间分析
```

### 3.2 数据收集

| 来源 | 方法 | 工具 |
|------|------|------|
| 社交媒体 | API | Twitter API, Reddit API |
| 网站 | 爬虫 | Scrapy, Beautiful Soup |
| 搜索引擎 | 查询 | Google Trends |
| 元数据 | 抓取 | 自定义脚本 |

### 3.3 分析技术

- 社会网络分析
- 文本挖掘
- 情感分析
- 主题建模

## 第四章：研究伦理

### 4.1 核心原则

| 原则 | 含义 |
|------|------|
| 知情同意 | 参与者知情并同意 |
| 隐私保护 | 保护身份和数据 |
| 最小风险 | 不造成伤害 |
| 透明性 | 披露研究目的 |

### 4.2 特殊考量

**公开vs私密**:
- 公开论坛：可能不需同意
- 私密群组：需要明确同意
- 边界模糊：谨慎判断

**匿名化**:
- 移除标识符
- 考虑可识别性
- 搜索引擎问题
