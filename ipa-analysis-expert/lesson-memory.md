## 成功模式

### 模式1：三类注释并行策略

**场景**: 任何IPA分析

**方法**:
- 同时进行描述性、语言性、概念性注释
- 使用三列表格保持对照
- 不跳过任何一类注释

**效果**: 确保分析的深度和全面性，捕捉经验的多维度。

**适用条件**: 所有IPA单案例分析。

---

### 模式2：自反性对话

**场景**: 研究者可能带有前理解的分析

**方法**:
1. 在分析前记录自己的前理解
2. 在概念性注释中明确标注研究者思考
3. 定期回顾自反日志
4. 邀请同侪挑战自己的诠释

**效果**: 提升分析的透明度和可信度。

**适用条件**: 研究者对研究主题有个人经验时。

---

### 模式3：主题涌现追踪

**场景**: 主题发展阶段

**方法**:
- 记录每个主题的"诞生"过程
- 保留从注释到主题的转变记录
- 使用引语标注主题来源

**效果**: 增强分析的透明性和可追溯性。

**适用条件**: 所有IPA主题发展。

---

## 失败教训

### 教训1：跳过注释直接命名主题

**场景**: 时间压力下的分析

**错误做法**:
- 看到原文就想到"主题"
- 跳过系统注释过程
- 使用已有概念标签

**后果**:
- 分析表面化
- 参与者声音被遮蔽
- 违背IPA方法论原则

**正确做法**:
- 必须逐行注释
- 让主题从注释中涌现
- 保持与原文的连接

**教训标签**: #跳过注释 #预设主题

---

### 教训2：使用诊断术语命名主题

**场景**: 临床相关研究

**错误做法**:
- 使用症状标签作为主题
- 用诊断术语替代心理学术语
- 如"抑郁症状"、"焦虑表现"

**后果**:
- 重复已有知识
- 无法贡献新理解
- 不是真正的IPA主题

**正确做法**:
- 使用心理学术语
- 关注经验而非症状
- 如"受限自我的体验"、"时间停滞感"

**教训标签**: #诊断术语 #表面分析

---

### 教训3：跨案例分析忽视个体差异

**场景**: 多参与者研究

**错误做法**:
- 只寻找共同主题
- 忽视个体独特主题
- 用"所有参与者都..."概括

**后果**:
- 丢失IPA的核心价值
- 遮蔽经验复杂性
- 违背个体关注原则

**正确做法**:
- 保持个体差异展示
- 独特主题与共同主题并列
- 使用"大多数参与者"而非"所有"

**教训标签**: #过度概括 #忽视差异

## 质量标准基准

### 注释质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 三类注释完整，与原文有清晰对应，概念性注释深化理解 |
| 良好 | 三类注释基本完整，有一定深度 |
| 及格 | 有三类注释，但深度不足 |
| 不及格 | 缺失注释类型，或注释脱离原文 |

### 主题质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 使用心理学术语，有多个引语支撑，从注释中涌现 |
| 良好 | 使用心理学术语，有引语支撑 |
| 及格 | 主题基本合理，但证据不足 |
| 不及格 | 使用诊断术语，或无证据支撑 |

### 结构质量标准

| 评分 | 标准 |
|------|------|
| 优秀 | 主从关系清晰，覆盖完整经验，有充分证据 |
| 良好 | 有主从结构，基本覆盖经验 |
| 及格 | 有结构，但关系不够清晰 |
| 不及格 | 结构混乱或缺失 |
