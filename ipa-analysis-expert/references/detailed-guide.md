## 第一章：IPA认识论基础

### 1.1 哲学渊源

IPA植根于两个哲学传统：

**现象学**：
- 关注"事物本身"
- 悬置自然态度
- 描述经验本质

**诠释学**：
- 理解总是诠释
- 前理解不可避免
- 诠释循环

### 1.2 双重建构

```
┌─────────────────────────────────────────────────────┐
│                    IPA认识论模型                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│    参与者的经验世界                                  │
│           │                                         │
│           ▼                                         │
│    参与者的诠释 (第一层建构)                         │
│    "我感到被困住了"                                  │
│           │                                         │
│           ▼                                         │
│    研究者的诠释 (第二层建构)                         │
│    "受限自我的体验"                                  │
│           │                                         │
│           ▼                                         │
│    研究文本 (知识产物)                               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 1.3 IPA的核心承诺

| 承诺 | 含义 | 实践体现 |
|------|------|---------|
| 深度 | 关注经验细节 | 逐行注释 |
| 情境 | 理解经验背景 | 情境描述 |
| 过程 | 追踪经验变化 | 时间线索 |
| 反思 | 研究者自反性 | 自反日志 |

## 第二章：研究设计

### 2.1 研究问题

**适合IPA的问题类型**：
- "...是如何体验的？"
- "...对当事人意味着什么？"
- "...如何理解...？"

**示例**：
- 慢性疼痛患者如何体验身体自我的变化？
- 初次成为父亲对男性意味着什么？
- 乳腺癌幸存者如何理解"康复"？

**不适合IPA的问题**：
- 描述性问题："患者有什么症状？"
- 因果性问题："什么导致了抑郁？"
- 频率问题："多少人经历过...？"

### 2.2 抽样策略

**同质样本原则**：
- 选择经验相似的人
- 便于识别共同模式
- 保持个体差异可见

**样本量参考**：
| 研究类型 | 建议样本量 |
|---------|-----------|
| 学生项目 | 3-6 |
| 硕士论文 | 6-12 |
| 博士论文 | 10-20 |
| 发表论文 | 6-10（深度优先） |

### 2.3 访谈设计

**半结构化访谈原则**：
- 准备开放性问题
- 允许参与者主导
- 灵活追问

**访谈提纲示例**：
```
研究问题：慢性疼痛患者如何体验身体自我的变化？

开场：
"请您谈谈您的疼痛经历。"

主题探索：
"疼痛对您的日常生活有什么影响？"
"您如何理解您现在的身体？"
"这对您意味着什么？"

深化追问：
"您能举个例子吗？"
"那时候您有什么感受？"
"这让我想到...您能多说一点吗？"

结束：
"还有什么您觉得重要但我没问到的吗？"
```

## 第三章：数据收集

### 3.1 访谈技巧

**建立关系**：
- 热情、非评判的态度
- 清晰解释研究目的
- 强调没有"正确"答案

**追问技术**：
| 类型 | 示例 | 目的 |
|------|------|------|
| 澄清 | "您能详细说说吗？" | 明确含义 |
| 深化 | "那时候您有什么感受？" | 情感维度 |
| 具体化 | "能举个例子吗？" | 具体情境 |
| 连接 | "这和您之前说的...有关系吗？" | 建立联系 |

### 3.2 转录

**IPA转录要求**：
- 逐字转录
- 保留语言特征（停顿、强调、犹豫）
- 标注非语言表达

**转录符号**：
```
(3s) - 3秒停顿
(.) - 短暂停顿
[叹气] - 非语言表达
[笑] - 笑声
... - 犹豫
"..." - 强调
(??) - 听不清
```

## 第四章：单案例分析

### 4.1 阅读与重读

**第一遍**：
- 完整阅读，获得整体印象
- 记录初步感受
- 不做任何标记

**第二遍**：
- 关注情感基调
- 识别关键片段
- 记录困惑

**第三遍及以后**：
- 开始系统注释
- 尝试理解

### 4.2 注释实践

**三类注释并行**：

**示例分析**：

原文：
> "我觉得...[叹气]...就像被困在一个盒子里，哪儿也去不了，我想要出去但是...就是...出不去。"

注释：
```
┌───────────────────────────────────────────────────┐
│ 描述性注释：                                       │
│ 参与者描述被困的感觉，想要逃离但做不到              │
├───────────────────────────────────────────────────┤
│ 语言性注释：                                       │
│ - "我觉得..." 犹豫的开场                          │
│ - "就像..." 隐喻引入                              │
│ - "盒子" 空间限制意象                             │
│ - "哪儿也去不了" 绝对化表达                       │
│ - "就是...出不去" 重复、无力感                    │
│ - [叹气] 无奈情感                                 │
├───────────────────────────────────────────────────┤
│ 概念性注释：                                       │
│ - 被困感、受限                                    │
│ - 自我与他者/环境的边界                           │
│ - 无力感、控制丧失                                │
│ - 可能与"受限自我"概念相关                        │
└───────────────────────────────────────────────────┘
```

### 4.3 发展主题

**从注释到主题**：
```
注释 → 新兴主题 → 心理学术语命名

示例：
注释："被困在盒子里"
    ↓
新兴主题："被困的感觉"
    ↓
心理学术语："受限自我的体验"
```

**主题命名原则**：
1. 使用心理学术语
2. 反映经验本质
3. 与原文有清晰连接
4. 区分主从关系

### 4.4 构建主从结构

```
主主题: 身体自我的丧失
│
├── 从主题1: 身体作为陌生人
│   ├── "这不再是我的身体"
│   └── "我认不出我自己"
│
├── 从主题2: 身体作为障碍
│   ├── "身体拖累了我"
│   └── "想做但做不到"
│
└── 从主题3: 努力重建身体自我
    ├── "我在学习接受"
    └── "找到新的方式"
```

## 第五章：跨案例分析

### 5.1 分析策略

**逐案例分析原则**：
- 每个案例单独分析
- 不预设主题
- 允许新主题涌现

**跨案例整合**：
```
案例1主题 ←→ 案例2主题 ←→ 案例3主题
      ↓              ↓              ↓
      └──────────────┴──────────────┘
                     ↓
              共同主题模式
              个体差异保持
```

### 5.2 模式识别

**共同模式**：
- 多个参与者出现的主题
- 核心经验维度

**个体差异**：
- 每个参与者的独特性
- 不同经验轨迹

### 5.3 最终主从主题表

```
主主题1: [共同主题]
├── 从主题1.1
│   ├── 案例1引语
│   ├── 案例2引语
│   └── 案例3引语
├── 从主题1.2
│   └── ...
└── 从主题1.3
    └── ...

主主题2: [共同主题]
└── ...

个体差异展示：
案例4独特主题: [...]
案例5独特主题: [...]
```

## 第六章：撰写报告

### 6.1 报告结构

```
引言
├── 研究背景
├── 文献回顾
└── 研究问题

方法
├── 参与者
├── 数据收集
├── 分析过程
└── 质量保证

发现
├── 主主题1
│   ├── 从主题分析
│   └── 引语展示
├── 主主题2
└── 个体差异

讨论
├── 与理论对话
├── 研究贡献
└── 局限性
```

### 6.2 引语使用

**原则**：
- 大量使用原文引语
- 引语与分析交织
- 引语支撑主题

**示例**：
```
从主题"身体作为陌生人"反映了参与者对自己身体的疏离感。
正如Sarah描述："这不再是我的身体，它做了自己的决定"。
这种描述表明参与者体验到自己与身体之间的断裂...
```

## 第七章：质量保证

### 7.1 Yardley四原则

| 原则 | 具体体现 | 如何实现 |
|------|---------|---------|
| 环境敏感性 | 情境描述 | 详细描述参与者背景、访谈情境 |
| 承诺与严谨 | 深度分析 | 逐行注释、案例展示 |
| 透明性与连贯性 | 过程清晰 | 方法描述详细、论证有据 |
| 影响与重要性 | 贡献明确 | 理论对话、实践意义 |

### 7.2 技术策略

**三角验证**：
- 多参与者
- 多数据源（访谈、观察、文档）
- 多研究者

**成员检验**：
- 将转录稿返回参与者
- 将分析结果返回确认
- 获取参与者反馈

**同侪审查**：
- 独立研究者审查主题
- 讨论分析分歧
- 达成共识

**审计追踪**：
- 保存所有分析记录
- 注释→主题的转变过程
- 研究决策文档

### 7.3 自反性实践

**自反日志**：
```
日期: [日期]
转录: [案例/段落]
我的反应: [...]
这可能如何影响我的分析: [...]
我需要进一步探索的: [...]
```
