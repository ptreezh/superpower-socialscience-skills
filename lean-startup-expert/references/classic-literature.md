# 精益创业经典文献

## 核心文献

### 1. Eric Ries - The Lean Startup (2011)
**《精益创业：新创企业的成长思维》**

核心概念：
- MVP (Minimum Viable Product)：最小可行产品
- Build-Measure-Learn：构建-度量-学习循环
- Validated Learning：经证实的认知
- Pivot or Persevere：转型或坚持
- Innovation Accounting：创新核算

引用规范：
> "The only way to win is to learn faster than anyone else." - Eric Ries

### 2. Steve Blank - The Four Steps to the Epiphany (2005)
**《四步创业法》**

核心概念：
- Customer Discovery：客户发现
- Customer Validation：客户验证
- Customer Creation：客户创建
- Company Building：公司建设

### 3. Alexander Osterwalder - Business Model Generation (2010)
**《商业模式新生代》**

与精益创业的关系：
- 商业模式画布与精益创业结合
- 假设测试与商业模式验证

## 关键方法论

### Build-Measure-Learn 循环

```
Idea → Build → Product → Measure → Data → Learn → Insights → Idea
```

**时间优化目标**：最小化循环时间

### 假设类型

1. **价值假设 (Value Hypothesis)**
   - 客户是否会使用/购买产品？
   - 产品是否解决真实问题？

2. **增长假设 (Growth Hypothesis)**
   - 产品如何增长？
   - 增长引擎是什么？

### 转型类型 (Pivot Types)

| 转型类型 | 描述 |
|---------|------|
| Zoom-in | 单一功能成为产品 |
| Zoom-out | 产品成为功能 |
| Customer Segment | 更换目标客户 |
| Customer Need | 解决不同问题 |
| Platform | 应用转平台或相反 |
| Business Model | 改变盈利模式 |
| Channel | 更换销售渠道 |
| Technology | 更换技术栈 |

## 衡量指标

### 可执行指标 (Actionable Metrics)
- 转化率
- 留存率
- 客户生命周期价值

### 虚荣指标 (Vanity Metrics) - 避免使用
- 总注册用户数
- 总页面浏览量
- 累计下载量
