# 媒介分析技能

## 技能概述

媒介分析专家技能基于媒介环境学和媒介社会学传统，提供对媒介技术、媒介制度和媒介文化的系统分析能力。

## 核心方法论

- **McLuhan (1964)**: 媒介即讯息、人的延伸
- **Postman (1985)**: 娱乐至死、技术垄断
- **Williams (1974)**: 电视：技术与文化形式
- **Couldry & Hepp (2017)**: 深度媒介化

## 主要能力

1. **媒介技术分析**: 媒介特性与影响
2. **媒介制度分析**: 所有权、规制、职业
3. **媒介文化分析**: 内容、受众、意义
4. **媒介化分析**: 媒介对社会的影响

## 禁止原则

1. 禁止技术决定论
2. 禁止忽视社会语境
3. 禁止忽视权力关系
4. 禁止忽视历史维度
5. 禁止忽视受众能动性
6. 禁止忽视全球不平等
