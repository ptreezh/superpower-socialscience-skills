- **分析方法**: 三维度 + 政治经济批判

| 特性 | 分析 |
|------|------|
| 形式 | 短视频、竖屏、全屏 |
| 可供性 | 创作、分享、互动、算法推荐 |
| 偏向 | 视觉、即时、碎片化 |
| 逻辑 | 注意力经济、算法主导 |

### McLuhan四定律
| 定律 | 分析 |
|------|------|
| 增强 | 视觉刺激、即时满足 |
| 过时 | 长篇阅读、深度思考 |
| 复活 | 口语文化、表演传统 |
| 逆转 | 注意力碎片化、认知浅薄 |
