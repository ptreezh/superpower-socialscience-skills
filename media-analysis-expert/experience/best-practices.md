# 媒介分析最佳实践

## 核心原则

### 原则1：三维度综合
```
分析框架：
技术 → 制度 → 文化
```
任一维度不可遗漏。

### 原则2：反技术决定论
- 技术提供可能性，不决定效果
- 效果取决于社会因素
- 避免线性因果思维

### 原则3：权力批判视角
- 媒介涉及权力分配
- 分析谁控制、谁受益
- 关注不平等和排斥

---

## 分析流程

### 第一步：界定分析对象
1. 确定媒介类型
2. 收集基础资料
3. 明确分析目的

### 第二步：技术分析
1. 媒介特性
2. 可供性
3. 偏向
4. McLuhan四定律

### 第三步：制度分析
1. 所有权结构
2. 规制框架
3. 经济模式
4. 职业组织

### 第四步：文化分析
1. 内容特征
2. 受众实践
3. 意义生产
4. 文化影响

### 第五步：媒介化分析
1. 媒介化程度
2. 媒介化过程
3. 社会影响

### 第六步：权力批判
1. 平台权力
2. 国家权力
3. 资本权力
4. 用户能动性

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 技术决定论 | 分析社会因素 |
| 忽视制度 | 增加政治经济分析 |
| 忽视受众 | 分析用户实践 |
| 去历史化 | 增加历史维度 |
| 忽视全球 | 增加全球视角 |
