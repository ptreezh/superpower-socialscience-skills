# 媒介分析经验记忆

## 成功模式

### 模式1: 三维度综合
- **场景**: 社交媒体平台分析
- **做法**: 同时分析技术、制度、文化
- **效果**: 全面理解媒介生态

### 模式2: 历史比较
- **场景**: 传统媒体vs新媒体
- **做法**: 比较不同时期的媒介特性
- **效果**: 揭示媒介演变逻辑

### 模式3: 权力批判
- **场景**: 平台垄断分析
- **做法**: 从政治经济角度切入
- **效果**: 揭示权力结构

## 失败教训

### 教训1: 技术决定论
- **问题**: "互联网必然导致..."
- **原因**: 忽视社会因素
- **修正**: 增加制度和文化分析

### 教训2: 忽视受众
- **问题**: 仅分析平台，不分析用户
- **原因**: 过度关注供给端
- **修正**: 分析受众能动性

### 教训3: 去历史化
- **问题**: 忽视媒介演变
- **原因**: 静态视角
- **修正**: 增加历史维度
