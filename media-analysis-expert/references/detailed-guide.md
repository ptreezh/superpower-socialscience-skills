## 第一章：媒介研究概论

### 1.1 学科定义

媒介分析研究媒介技术、媒介制度和媒介文化之间的复杂关系，关注媒介如何塑造社会、文化和个体经验。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 早期 | Innis | 传播偏向理论 |
| 经典 | McLuhan | 媒介环境学 |
| 批判 | Postman | 技术批判 |
| 文化 | Williams | 文化唯物主义 |
| 当代 | Couldry, Hepp | 深度媒介化 |

### 1.3 核心概念

```
媒介分析体系
├── 媒介技术
│   ├── 物质性
│   ├── 可供性
│   └── 偏向
├── 媒介制度
│   ├── 所有权
│   ├── 规制

├── 媒介文化
│   ├── 内容

    ├── 过程

    └── 效果
```

## 第二章：媒介环境学

### 2.1 McLuhan核心概念

**媒介即讯息**:
- 媒介的形式比内容更重要
- 媒介塑造感知和思维模式
- 每种媒介创造特定的环境

**人的延伸**:
| 媒介 | 延伸的能力 |
|------|-----------|
| 印刷 | 视觉/线性思维 |
| 电子媒介 | 中枢神经 |
| 数字媒介 | 认知/记忆 |

**媒介四定律**:
1. 增强: 媒介增强了什么？
2. 过时: 媒介使什么过时？
3. 复活: 媒介复活了什么？
4. 逆转: 媒介推向极致会变成什么？

### 2.2 Postman技术批判

- 媒介垄断: 技术主导文化
- 媒介意识形态: 隐含的价值取向
- 技术变化的社会后果
