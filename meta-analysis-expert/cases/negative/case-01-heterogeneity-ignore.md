# 负面案例：忽视异质性的错误合并

## 案例信息
- **案例ID**: case-01-heterogeneity-ignore
- **类型**: 负面案例
- **错误类型**: 忽视临床和统计异质性
- **创建时间**: 2026-03-15

---

## 一、案例背景

### 原研究声称
"运动干预对所有类型抑郁症均有效，合并效应量为0.8"

### 原研究问题
运动干预对抑郁症的疗效

---

## 二、方法学错误分析

### 错误1: 临床异质性未考虑

**原研究纳入标准**:
- 任何运动干预
- 任何抑郁症类型
- 任何对照

**问题**:
| 研究 | 抑郁症类型 | 运动类型 | 强度 |
|------|-----------|---------|------|
| A | 轻度 | 有氧运动 | 低 |
| B | 重度 | 抗阻训练 | 高 |
| C | 产后抑郁 | 瑜伽 | 低 |
| D | 双相抑郁 | 混合训练 | 中 |
| E | 季节性抑郁 | 户外步行 | 低 |

**临床异质性极大**:
- 抑郁类型不同(单相、双相、产后等)
- 运动类型不同(有氧、抗阻、瑜伽)
- 强度差异大

### 错误2: 强行使用固定效应模型

**原研究统计结果**:
```
固定效应模型
合并效应: d = 0.80 (95%CI: 0.65, 0.95)

异质性:
Q = 156.8, df = 15, p < 0.0001
I² = 90.4%
```

**问题**: I²=90.4%仍使用固定效应模型!

**正确做法**:
- I²>75%应使用随机效应模型
- 且应探索异质性来源
- 可能需要分开分析

### 错误3: 忽视漏斗图不对称

**原研究漏斗图**:
```
        ●
      ●   ●
    ●       ●
  ●           ●
●               ●
●                 ●
●●●●●●●●●●●●●●●●●●
```

明显左偏，但未报告发表偏倚检验

### 错误4: 结论过度推广

**原结论**:
"运动干预对所有抑郁症患者都有效，应作为首选治疗"

**问题**:
- 不同类型抑郁症效应可能不同
- 未区分运动类型和强度
- 忽视个体差异

---

## 三、正确做法示范

### 正确做法1: 评估临床同质性

**应在纳入前评估**:
- 是否可以合理合并？
- 是否需要分开分析？

**建议分组**:
- 轻中度单相抑郁
- 重度抑郁
- 特殊类型(产后、季节性等)

### 正确做法2: 正确处理异质性

```r
# 使用随机效应模型
library(metafor)
rma(yi, vi, data = data, method = "REML")

# 探索异质性来源
rma(yi, vi, mods = ~ depression_type, data = data)
```

### 正确做法3: 分开分析

| 亚组 | 研究数 | 效应量 | I² |
|------|--------|--------|-----|
| 轻中度抑郁 | 8 | 0.65 | 45% |
| 重度抑郁 | 4 | 0.32 | 52% |
| 产后抑郁 | 3 | 0.78 | 38% |

**结论修正**: 运动对轻中度和产后抑郁效果较好，对重度抑郁效果有限

### 正确做法4: 报告发表偏倚

```
Egger检验: t = 3.21, p = 0.006
存在显著发表偏倚
Trim-and-Fill校正后效应: d = 0.52
```

---

## 四、后果分析

### 对研究的后果
- 结论不可靠
- 无法复制
- 引用误导临床决策

### 对临床的后果
- 可能对重度抑郁患者建议无效治疗
- 忽视个体化需求

### 对学科的后果
- 降低元分析可信度
- 浪费研究资源

---

## 五、教训总结

### 核心原则

1. **临床异质性优先评估**
   - 纳入前评估研究相似性
   - 不强行合并不相似的研究

2. **统计异质性必须报告**
   - 常规报告I²和Q检验
   - I²>50%使用随机效应模型

3. **发表偏倚必须检验**
   - 研究数≥10时必须检验
   - 报告检验结果和校正

4. **结论谨慎推广**
   - 明确适用范围
   - 承认局限性

### 避免清单

- [ ] 不评估临床异质性就合并
- [ ] 高异质性使用固定效应模型
- [ ] 忽视漏斗图不对称
- [ ] 结论过度推广
- [ ] 不报告异质性指标
- [ ] 不做敏感性分析
