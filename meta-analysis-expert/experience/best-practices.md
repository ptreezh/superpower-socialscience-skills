## 一、方案设计

### 1.1 预注册必要性

**强烈建议预注册**:
- PROSPERO (系统综述)
- Cochrane (Cochrane综述)
- OSF (其他研究)

**注册内容**:
- 研究问题
- 纳入标准
- 检索策略
- 分析计划

### 1.2 PICO设计原则

**清晰明确**:
- 每个要素都有操作定义
- 避免模糊表述
- 考虑可行性

**示例对比**:
```
❌ 差: P = 抑郁症患者
✓ 好: P = 符合DSM-5诊断标准的18-65岁单相抑郁症患者

❌ 差: I = 心理治疗
✓ 好: I = 标准化认知行为治疗(8-20次，每周1次)

❌ 差: C = 对照组
✓ 好: C = 等待名单对照或常规治疗
```

|------|-----------|

| 心理学 | PsycINFO |
| 教育 | ERIC |
| 社会科学 | SSCI |

### 2.3 检索策略验证

- 检查是否包含已知研究
- 测试检索敏感度
- 记录检索历史

## 三、研究筛选

### 3.1 筛选流程

```
标准流程
├── 去重
│   └── 记录去重数量
├── 标题/摘要筛选
│   ├── 双人独立筛选
│   ├── 记录排除原因
│   └── 计算一致性
├── 全文筛选
│   ├── 双人独立评估
│   ├── 记录排除原因
│   └── 计算一致性
└── 最终纳入
```

### 3.2 筛选一致性

**Kappa值评价**:
- κ > 0.80: 几乎完全一致
- κ 0.60-0.80: 高度一致
- κ 0.40-0.60: 中度一致
- κ < 0.40: 一致性差，需讨论

## 四、数据提取

### 4.1 提取原则

**标准化**:
- 预先设计提取表
- 明确各字段定义
- 提供提取示例

**双人独立**:
- 两名提取者独立提取
- 比较结果差异
- 讨论达成一致

### 4.2 常见数据问题

| 问题 | 解决方案 |
|------|---------|
| 数据缺失 | 联系作者 |
| 数据不一致 | 优先使用预先指定 |
| 单位不同 | 统一转换 |
| 多时间点 | 选择主要时间点 |

### 4.3 效应数据提取

**连续变量**:
```
需要: 均值、SD、样本量
注意: 区分干预组和对照组
特殊: 如有基线数据，考虑使用变化值
```

**二分类变量**:
```
需要: 事件数、总数
注意: 明确分子/分母定义
```

## 五、质量评价

### 5.1 工具选择

| 研究类型 | 推荐工具 |
|---------|---------|
| RCT | Cochrane RoB 2.0 |
| 观察性(干预) | ROBINS-I |
| 诊断准确性 | QUADAS-2 |
| 非随机 | Newcastle-Ottawa |

### 5.2 评价原则

**独立评价**:
- 至少两名评价者
- 盲法评价(如可能)

**判断依据**:
- 基于报告内容
- 未报告≠低风险
- 必要时联系作者

### 5.3 质量影响分析

**敏感性分析**:
- 排除高风险研究
- 比较结果差异

**亚组分析**:
- 按质量分组
- 比较效应差异

## 六、统计分析

### 6.1 效应量选择

**连续变量**:
| 情况 | 选择 |
|------|------|
| 相同测量工具 | MD (均数差) |
| 不同测量工具 | SMD (标准化均数差) |
| 小样本(<50) | Hedges' g |

**二分类变量**:
| 情况 | 选择 |
|------|------|
| 常见事件(>10%) | OR或RR |
| 罕见事件(<10%) | Peto OR |
| 需要解释风险 | RR或RD |

### 6.2 模型选择

```
决策树
├── 临床异质性评估
│   ├── 研究相似 → 考虑固定效应
│   └── 研究差异 → 随机效应
├── I²评估
│   ├── I² < 25% → 固定效应可行
│   └── I² ≥ 25% → 随机效应推荐
└── 推荐
    └── 默认使用随机效应模型
```

### 6.3 异质性处理

**步骤**:
1. 量化异质性(I², τ²)
2. 探索来源(亚组、元回归)
3. 判断是否可合并
4. 必要时分开分析

**注意**:
- I²高不一定不可合并
- I²低不一定可合并
- 临床判断优先

## 七、发表偏倚

### 7.1 检验条件

**最低研究数**:
- 漏斗图: ≥10个研究
- Egger检验: ≥10个研究
- Trim-and-Fill: ≥10个研究

### 7.2 结果解释

**漏斗图不对称原因**:
- 发表偏倚
- 真实异质性
- 小样本效应
- 选择性报告

**综合判断**:
- 多种方法检验
- 结合领域知识
- 报告校正后结果

## 八、证据质量

### 8.1 GRADE框架

**降级因素**:
1. 偏倚风险
2. 不一致性
3. 间接性
4. 不精确性
5. 发表偏倚

**升级因素**:
1. 大效应量
2. 剂量反应关系
3. 负面混杂

### 8.2 质量报告

**建议格式**:
```
证据质量评估
├── 主要结局: 中等质量 ⊕⊕⊕○
│   ├── 偏倚风险: 不降级 (敏感性分析一致)
│   ├── 不一致性: 降1级 (I²=65%)
│   └── 不精确性: 不降级 (CI较窄)
└── 次要结局: 低质量 ⊕⊕○○
    └── ...
```

## 九、报告规范

### 9.1 PRISMA 2020要点

**标题**:
- 明确为系统综述或元分析

**摘要**:
- 结构化摘要(背景、方法、结果、结论)

**方法**:
- 方案注册信息
- 完整检索策略
- 明确纳入标准
- 质量评价方法

**结果**:
- PRISMA流程图
- 研究特征表
- 偏倚风险图
- 森林图
- 漏斗图

**讨论**:
- 证据质量评估
- 局限性
- 与其他证据比较

### 9.2 图表规范

**森林图要素**:
- 各研究效应量及CI
- 合并效应量及CI
- 异质性指标
- 权重信息

**漏斗图要素**:
- 效应量vs精度
- 对称性参考线
- 可能缺失研究

## 十一、常见问题

### Q1: 研究数太少怎么办？

**建议**:
- 考虑定性描述
- 不强行合并
- 增加检索范围

### Q2: 异质性过高怎么办？

**建议**:
- 检查临床异质性
- 进行亚组分析
- 必要时放弃合并

### Q3: 数据格式不同怎么办？

**建议**:
- 使用统一转换公式
- 报告转换方法
- 进行敏感性分析
