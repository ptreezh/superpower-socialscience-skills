## 成功模式

### 模式001: 完整PRISMA流程
**场景**: 系统性文献综述与元分析
**成功要素**:
- 预注册方案
- 多数据库检索
- 双人独立筛选
- 标准化数据提取表
- Cochrane RoB评估
- PRISMA流程图

**效果**: 高质量证据等级

### 模式002: 异质性处理策略
**场景**: 发现高异质性(I²>50%)
**成功策略**:
1. 使用随机效应模型
2. 进行亚组分析探索来源
3. 敏感性分析
4. 报告预测区间

**避免**: 忽视异质性强行合并

### 模式003: 发表偏倚处理
**场景**: 研究数≥10篇
**成功策略**:
- 绘制漏斗图
- Egger检验
- Trim-and-Fill校正
- 失安全系数计算

### 模式004: 小样本研究处理
**场景**: 纳入研究样本量小
**成功策略**:
- 使用Hedges' g而非Cohen's d
- 报告精确CI
- 进行敏感性分析

## 失败模式

### 失败001: 预设结论偏倚
**问题**: 根据期望结果调整纳入标准
**后果**: 证据可信度降低
**预防**:
- 预注册方案
- 客观PICO框架
- 透明报告所有决策

### 失败002: 忽视临床异质性
**问题**: 合并临床差异大的研究
**后果**: 结果无临床意义
**预防**:
- 详细评估研究特征
- 必要时分开分析
- 报告临床相关性

### 失败003: 过度亚组分析
**问题**: 进行过多亚组分析导致假阳性
**后果**: 结论不可靠
**预防**:
- 预先设定亚组
- 限制亚组数量
- 校正多重比较

### 失败004: 忽视研究质量
**问题**: 不区分高质量和低质量研究
**后果**: 偏倚影响合并结果
**预防**:
- 必须做质量评价
- 敏感性分析排除低质量研究
- GRADE评估证据质量

## 关键决策记录

### 决策001: 固定vs随机效应模型
**条件**:
- I²<25% 且临床同质 → 固定效应
- I²≥25% 或临床异质 → 随机效应

**理由**: 随机效应模型更保守，适用于大多数情况

### 册策002: 效应量选择
| 数据类型 | 推荐效应量 |
|---------|-----------|
| 连续变量(相同量表) | MD |
| 连续变量(不同量表) | SMD |
| 二分类(罕见事件) | Peto OR |
| 二分类(常见事件) | OR或RR |
| 时间-事件 | HR |

### 决策003: 研究数量要求
| 分析类型 | 最低要求 |
|---------|---------|
| 简单合并 | ≥2个研究 |
| 亚组分析 | ≥3个研究/组 |
| 元回归 | ≥10个研究/变量 |
| 发表偏倚检验 | ≥10个研究 |

## 方法论笔记

### 异质性来源
1. **临床异质性**: 人群、干预、对照、结局差异
2. **方法学异质性**: 设计、偏倚风险差异
3. **统计异质性**: 效应量变异超过抽样误差

### 质量评价指标
- Cochrane RoB 2.0 (RCT)
- ROBINS-I (观察性研究)
- QUADAS-2 (诊断研究)
- Newcastle-Ottawa (非随机研究)

### GRADE证据等级
- 高质量: RCT, 无降级因素
- 中等质量: RCT, 有1个降级因素
- 低质量: RCT, 有2个降级因素
- 极低质量: 观察性研究或多个降级因素

## 统计公式参考

### 效应量计算
```
SMD = (M₁ - M₂) / SD_pooled
SD_pooled = √((SD₁² + SD₂²) / 2)

OR = (a×d) / (b×c)
ln(OR)的SE = √(1/a + 1/b + 1/c + 1/d)
```

### 异质性指标
```
I² = (Q - df) / Q × 100%
τ²(DL法) = (Q - (k-1)) / C

C = Σw_i - Σw_i²/Σw_i
```

### 合并效应
```
固定效应: ES = Σ(w_i × ES_i) / Σw_i
随机效应: ES = Σ(w_i* × ES_i) / Σw_i*
w_i* = 1 / (v_i + τ²)
```
