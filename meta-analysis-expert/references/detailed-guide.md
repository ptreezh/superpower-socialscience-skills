## 第一章：元分析概论

### 1.1 元分析定义

元分析(Meta-Analysis)是一种定量合成方法，通过统计技术整合多个独立研究的结果，得出更精确、更可靠的总体效应估计。由Gene Glass于1976年首次提出。

**核心特征**:
- 系统性文献检索
- 明确的纳入/排除标准
- 效应量计算
- 定量合成
- 异质性评估
- 发表偏倚检验

### 1.2 元分析类型

| 类型 | 描述 | 适用情况 |
|------|------|---------|
| 干预元分析 | 评估干预效果 | RCT研究合成 |
| 观察性元分析 | 合并观察研究结果 | 队列/病例对照研究 |
| 诊断准确性 | 评估诊断方法 | 敏感性/特异性合成 |
| 剂量反应 | 剂量-效应关系 | 流行病学研究 |
| 网络元分析 | 多干预比较 | 多种治疗对比 |
| 个体患者数据 | IPD元分析 | 原始数据可用 |

### 1.3 元分析步骤

```
标准流程 (PRISMA框架)
├── 1. 问题明确化
│   └── PICO/PECO框架
├── 2. 文献检索
│   ├── 数据库检索
│   └── 其他来源
├── 3. 筛选与选择
│   ├── 标题/摘要筛选
│   └── 全文筛选
├── 4. 数据提取
│   ├── 研究特征
│   └── 效应数据
├── 5. 质量评价
│   └── 偏倚风险评估
├── 6. 数据分析
│   ├── 效应量计算
│   ├── 异质性检验
│   └── 模型选择
├── 7. 发表偏倚
│   └── 漏斗图/Egger检验
└── 8. 结果解释
    └── 证据质量评估
```

## 第二章：研究问题与检索策略

### 2.1 PICO框架

| 要素 | 描述 | 示例 |
|------|------|------|
| P (Population) | 研究人群 | 抑郁症患者 |
| I (Intervention) | 干预措施 | 认知行为治疗 |
| C (Comparison) | 对照组 | 常规治疗/等待名单 |
| O (Outcome) | 结局指标 | 抑郁症状评分 |

### 2.2 检索策略

**数据库选择**:
- 生物医学: PubMed, Embase, Cochrane Library
- 心理学: PsycINFO
- 社会科学: SSCI, ERIC
- 中文: CNKI, 万方

**检索词设计**:
```
(Population terms) AND
(Intervention terms) AND
(Study design terms)

示例:
(depression OR depressive) AND
(cognitive behavioral therapy OR CBT) AND
(randomized controlled trial OR RCT)
```

### 2.3 PRISMA流程图

```
检索记录 (n=)
    ↓
去重后 (n=)
    ↓
标题/摘要筛选 (n=) → 排除 (n=)
    ↓
全文评估 (n=) → 排除 (n=, 原因)
    ↓
纳入研究 (n=)
    ↓
纳入定性合成 (n=)
    ↓
纳入定量合成 (n=)
```

## 第三章：数据提取与质量评价

### 3.1 数据提取内容

**研究特征**:
- 作者、年份
- 研究地点
- 样本量
- 研究设计
- 纳入/排除标准

**人群特征**:

## 第四章：效应量计算

### 4.1 效应量类型

**连续变量**:
```
标准化均数差 (SMD)
SMD = (M₁ - M₂) / SD_pooled

Hedges' g (小样本校正)
g = SMD × (1 - 3/(4N-9))
```

**二分类变量**:
```
比值比 (OR)
OR = (a/b) / (c/d)

相对风险 (RR)
RR = (a/(a+b)) / (c/(c+d))

风险差 (RD)
RD = a/(a+b) - c/(c+d)
```

### 4.2 效应量解释

| SMD值 | 解释 |
|-------|------|
| 0.2 | 小效应 |
| 0.5 | 中效应 |
| 0.8 | 大效应 |

| OR/RR值 | 解释 |
|---------|------|
| 1.0 | 无效应 |
| 1.5 | 小效应 |
| 2.0 | 中效应 |
| 3.0+ | 大效应 |

## 第五章：统计模型与分析

### 5.1 固定效应模型

**假设**: 所有研究估计同一真实效应

**权重计算**:
```
w_i = 1 / v_i

合并效应 = Σ(w_i × ES_i) / Σw_i
SE = √(1 / Σw_i)
```

**适用条件**:
- 低异质性
- 研究同质
- 理论基础支持

### 5.2 随机效应模型

**假设**: 真实效应服从某种分布

**权重计算**:
```
w_i* = 1 / (v_i + τ²)

τ²: 研究间方差
DerSimonian-Laird估计
```

**适用条件**:
- 存在异质性
- 研究方法差异
- 人群差异

### 5.3 异质性评估

**Q检验**:
```
Q = Σw_i(ES_i - ES_pooled)²

H₀: 所有研究效应相同
```

**I²统计量**:
```
I² = (Q - df) / Q × 100%

解释:
- 25%: 低异质性
- 50%: 中等异质性
- 75%: 高异质性
```

**预测区间**:
```
PI = ES_pooled ± t_α × √(τ² + v_pooled)
```

## 第六章：亚组分析与元回归

### 6.1 亚组分析

**目的**: 探索异质性来源

**常见亚组**:
- 研究设计
- 人群特征
- 干预类型
- 地区/文化

**统计检验**:
```
Q_between = Σn_j(ES_j - ES_pooled)²

Q_total = Q_within + Q_between
```

### 6.2 元回归

**单变量回归**:
```
ES_i = β₀ + β₁X_i + e_i

权重: w_i = 1 / (v_i + τ²)
```

**多变量回归**:
```
ES_i = β₀ + β₁X₁ + β₂X₂ + ... + e_i
```

**注意事项**:
- 研究数量要求 (≥10个/变量)
- 共线性问题
- 过拟合风险

## 第七章：发表偏倚

### 7.1 漏斗图

**对称性评估**:
- 效应量 vs 精度 (SE或1/SE)
- 目视对称性检验
- 注意: 非发表偏倚独有

### 7.2 统计检验

**Egger检验**:
```
回归: ES_i/SE_i = α + β × (1/SE_i) + e

H₀: α = 0 (无偏倚)
```

**Begg秩相关检验**:
```
Kendall's τ between ES and SE
```

**Trim-and-Fill**:
- 估计缺失研究数
- 填充后重新估计

### 7.3 失安全系数

```
N_fs = (ΣZ_i)² / Z_α² - k

解释: 需要多少未发表研究使结果不显著
```

## 第十章：常见问题与解决方案

### 10.1 异质性处理

| 问题 | 解决方案 |
|------|---------|
| 高I² | 使用随机效应模型 |
| 异质性来源不明 | 亚组分析/元回归 |
| 临床异质 | 分开分析或放弃合成 |

### 10.2 数据转换

| 原始数据 | 转换方法 |
|---------|---------|
| F/t值 | ES = t/√df 或 F→t |
| p值 | 需要方向信息 |
| 中位数/范围 | 近似转换公式 |

### 10.3 特殊情况

**零事件研究**:
- 连续性校正
- Peto OR
- 排除 (说明理由)

**小样本研究**:
- Hedges' g校正
- 精确方法
