## 第一部分：叙事研究基础

### 1.1 什么是叙事研究

叙事研究(Narrative Research)是一种关注人们如何通过讲述故事来建构意义和身份的研究方法。其核心假设是：人类通过叙事来理解和组织经验。

**核心特征**:
- 关注故事如何被讲述
- 理解叙事的意义建构功能
- 重视讲述者的视角
- 关注叙事的社会文化语境

### 1.2 叙事转向

**从实证主义到建构主义**:
- 知识不再是客观发现
- 现实通过话语建构
- 叙事是意义建构的核心方式

**跨学科发展**:
| 学科 | 叙事应用 |
|------|---------|
| 心理学 | 叙事认同、叙事治疗 |
| 社会学 | 生活故事、传记研究 |
| 人类学 | 民族志叙事 |
| 教育学 | 教师叙事、教育故事 |
| 医学 | 疾病叙事、医患叙事 |

### 1.3 核心概念

**叙事(Narrative)**
- 有组织的事件序列
- 具有时间结构
- 包含意义建构
- 在社会语境中讲述

**故事(Story)**
- 叙事的内容
- 被讲述的事件
- 情节和人物

**讲述(Telling)**
- 叙事的行为
- 表演维度
- 语境依赖

## 第二部分：Labov叙事结构分析

### 2.1 六要素模型

**完整叙事的结构**:

```
┌─────────────────────────────────────┐
│ 摘要(Abstract)                      │
│ ────────────────────────            │
│ 故事的简要概括                       │
│ 获取听众注意力                       │
├─────────────────────────────────────┤
│ 定向(Orientation)                   │
│ ────────────────────────            │
│ 时间、地点、人物、情境               │
│ 建立故事背景                         │
├─────────────────────────────────────┤
│ 纠葛(Complicating Action)           │
│ ────────────────────────            │
│ 核心事件发生                         │
│ 冲突和问题出现                       │
├─────────────────────────────────────┤
│ 评估(Evaluation)                    │
│ ────────────────────────            │
│ 讲述者的态度和评价                   │
│ 故事的意义阐释                       │
├─────────────────────────────────────┤
│ 结果(Resolution)                    │
│ ────────────────────────            │
│ 事件的最终结局                       │
│ 问题如何解决                         │
├─────────────────────────────────────┤
│ 结尾(Coda)                          │
│ ────────────────────────            │
│ 回归当下时刻                         │
│ 故事的最终意义                       │
└─────────────────────────────────────┘
```

### 2.2 结构分析示例

**原文**:
> "面试那天，我提前半小时到了公司楼下。天气特别热，我在星巴克买了杯咖啡等着。到了约定时间，我走进办公室，结果前台说面试官临时有事改期了。我当时特别生气，觉得白跑了。不过后来想想，也许这是老天在考验我的耐心吧。"

**结构分析**:

| 要素 | 内容 | 功能 |
|------|------|------|
| 定向 | 面试那天，提前半小时，天气热，星巴克 | 建立背景 |
| 纠葛 | 面试官临时改期 | 核心事件 |
| 评估 | 特别生气，觉得白跑了 | 情感评价 |
| 结果 | (未明确交代) | - |
| 结尾 | 老天考验耐心 | 意义阐释 |

### 2.3 评估的类型

**内部评估**: 嵌入叙述中
> "我当时心里很紧张"

**外部评估**: 明确的评价陈述
> "这件事让我意识到..."

**评估手段**:
- 情感表达
- 价值判断
- 比较对照
- 重复强调

## 第三部分：McAdams叙事认同理论

### 3.1 自我即故事

**核心观点**:
- 身份是通过叙事建构的
- 自我理解通过讲故事实现
- 人生故事是身份的核心

### 3.2 叙事认同三层次

**层次一：叙事基调**

| 基调类型 | 特征 | 心理意义 |
|---------|------|---------|
| 救赎性 | 困境→转变→成长 | 心理健康、适应力 |
| 污染性 | 好事被坏事破坏 | 抑郁倾向 |
| 进步性 | 持续向上发展 | 乐观、能动性 |
| 退步性 | 持续下降 | 悲观、无助感 |

**层次二：叙事意象**

核心情节类型：
1. **高潮情节** - 成功与成就
2. **低谷情节** - 失败与痛苦
3. **转折情节** - 人生方向改变
4. **留恋情节** - 深刻记忆

**层次三：叙事意识形态**

- 价值观
- 人生哲学
- 信念系统
- 道德原则

### 3.3 生活故事访谈

**访谈框架**:

```
1. 人生章节
   └── "如果把人生分成几个章节..."

2. 关键事件
   ├── 高潮事件
   ├── 低谷事件
   ├── 转折点
   └── 最早记忆

3. 重要他人
   └── 人生剧本中的关键角色

4. 未来脚本
   └── 想象中的未来

5. 压力与挑战
   └── 人生障碍与应对

6. 人生哲学
   └── 核心信念和价值观
```

## 第四部分：Gee诗意结构分析

### 4.1 分析单位

**诗行(Line)**
- 一个思想单元
- 一个语调单位
- 通常以停顿标记

**诗节(Stanza)**
- 相关诗行集合
- 围绕一个主题
- 通常2-4行

**部分(Part)**
- 相关诗节集合
- 叙事的一个阶段
- 通常多个诗节

### 4.2 分析示例

**原文**:
> "我在医院工作，已经五年了。那时候刚毕业，什么都不会。现在好多了，能独当一面了。"

**诗意结构分析**:

```
第1部分: 背景
┌─────────────────────────────┐
│ 第1诗节: 当前状态             │
│   第1行: 我在医院工作         │
│   第2行: 已经五年了           │
└─────────────────────────────┘

第2部分: 变化
┌─────────────────────────────┐
│ 第2诗节: 过去状态             │
│   第1行: 那时候刚毕业         │
│   第2行: 什么都不会           │
└─────────────────────────────┘

┌─────────────────────────────┐
│ 第3诗节: 现在状态             │
│   第1行: 现在好多了           │
│   第2行: 能独当一面了         │
└─────────────────────────────┘
```

## 第五部分：Riessman三种分析模式

### 5.1 主题分析

**关注点**: 叙事内容

**步骤**:
```
1. 阅读所有叙事
2. 识别重复出现的主题
3. 编码相关内容
4. 跨叙事比较
5. 发展主题类别
6. 理论对话
```

**适用场景**:
- 探索共同经验
- 理解群体特征
- 发展理论概念

### 5.2 结构分析

**关注点**: 叙事形式

**步骤**:
```
1. 转录叙事
2. 识别结构要素
3. 分析组织方式
4. 理解形式-意义关系
5. 比较结构差异
```

**适用场景**:
- 理解叙事能力
- 分析叙事技巧
- 探索文化差异

### 5.3 表演分析

**关注点**: 叙事讲述

**关注要素**:
- 讲述情境
- 听众角色
- 讲述目的
- 非语言表达

**分析问题**:
- 这个故事为何在此刻讲述？
- 听众如何影响讲述？
- 叙事实现了什么功能？

## 第六部分：数据收集方法

### 6.1 叙事访谈

**开放式叙事访谈**:
```
指导语: "请讲讲你...的经历"
         "能详细说说吗？"
         "然后发生了什么？"
```

**生活故事访谈**:
- 结构化框架
- 深度探索
- 多次进行

### 6.2 焦点叙事

**主题聚焦**:
- 围绕特定主题
- 收集相关故事
- 深度分析

**示例主题**:
- 职业转型故事
- 疾病经历叙事
- 创伤与康复

### 6.3 书面叙事

**类型**:
- 自传
- 日记
- 信件
- 网络日志

**优势**:
- 完整性
- 反思性
- 自然性

## 第七部分：质量与伦理

### 7.1 质量标准

**叙事效度**:
- 内在一致性
- 情节连贯性
- 人物一致性

**对应效度**:
- 与其他资料对应
- 三角验证

**批判效度**:
- 挑战既有假设
- 揭示权力关系

### 7.2 伦理考量

**知情同意**:
- 说明研究目的
- 解释叙事使用
- 获得持续同意

**隐私保护**:
- 匿名化处理
- 识别信息删除
- 情境伪装

**情感保护**:
- 敏感话题谨慎
- 提供支持资源
- 尊重讲述边界
