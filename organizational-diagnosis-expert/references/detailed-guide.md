# 组织诊断详细指南

## 一、方法论概述

组织诊断是评估组织健康状况、识别问题和改进机会的系统方法，通过多维度分析组织结构、流程、文化等要素，为组织变革提供依据。

## 二、Weisbord六盒详解

### 1. 目的/目标
| 维度 | 问题 |
|------|------|
| 目标清晰度 | 目标是否明确？ |
| 目标认同度 | 员工是否认同？ |
| 目标一致性 | 各层级目标是否一致？ |

### 2. 结构
| 维度 | 问题 |
|------|------|
| 结构适配 | 结构是否支持目标？ |
| 职责清晰 | 职责是否明确？ |
| 协调机制 | 部门协调是否有效？ |

### 3. 关系
| 维度 | 问题 |
|------|------|
| 人际关系 | 关系质量如何？ |
| 冲突管理 | 冲突如何处理？ |
| 协作水平 | 跨部门协作如何？ |

### 4. 领导
| 维度 | 问题 |
|------|------|
| 领导风格 | 风格是否适配？ |
| 决策质量 | 决策是否有效？ |
| 沟通效果 | 沟通是否畅通？ |

### 5. 激励
| 维度 | 问题 |
|------|------|
| 激励公平 | 激励是否公平？ |
| 激励有效性 | 激励是否有效？ |
| 需求匹配 | 是否满足需求？ |

### 6. 机制
| 维度 | 问题 |
|------|------|
| 决策机制 | 决策是否高效？ |
| 控制机制 | 控制是否适当？ |
| 信息机制 | 信息是否畅通？ |

## 三、分析流程
```
步骤1: 组织界定
    ↓
步骤2: 六盒诊断
    ↓
步骤3: 一致性分析
    ↓
步骤4: 问题诊断
    ↓
步骤5: 改进建议
```
