# PEST/PESTEL分析详细指南

## 一、方法论概述

### 1.1 PEST/PESTEL分析的定义

PEST/PESTEL分析是一种宏观环境分析工具，通过系统识别政治(Political)、经济(Economic)、社会(Social)、技术(Technological)、环境(Environmental)、法律(Legal)六大维度的环境因素，为战略决策提供外部环境洞察。

### 1.2 方法论价值

| 维度 | 价值说明 |
|------|----------|
| 战略规划 | 为长期战略提供环境洞察 |
| 风险识别 | 提前识别环境变化带来的风险 |
| 机会发现 | 发现环境变化带来的新机会 |
| 投资决策 | 为投资决策提供环境评估 |

### 1.3 适用场景

- 战略规划环境扫描
- 市场进入评估
- 投资决策支持
- 行业分析
- 政策影响评估

## 二、六维度详解

### 2.1 Political（政治因素）

**定义**：影响组织的政府政策和政治环境因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 政府政策 | 产业政策、财政政策 | 产业扶持政策、减税政策 |
| 政治稳定性 | 政府更迭、政治风险 | 选举周期、地缘政治 |
| 贸易政策 | 关税、贸易协定 | 进出口限制、自贸协定 |
| 外交关系 | 国际关系、制裁 | 贸易战、外交摩擦 |

**分析问题清单**：
- 当前政府的政策重点是什么？
- 近期有哪些政策变化？
- 政治稳定性如何？
- 贸易政策有什么变化？

### 2.2 Economic（经济因素）

**定义**：影响组织经济环境的宏观经济因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 经济增长 | GDP增长率、经济周期 | 经济扩张、衰退 |
| 利率汇率 | 利率水平、汇率变化 | 加息、货币贬值 |
| 通胀水平 | 通胀率、价格水平 | 高通胀、通缩风险 |
| 就业收入 | 失业率、居民收入 | 就业市场、消费能力 |

**分析问题清单**：
- 经济处于什么周期阶段？
- 利率和汇率趋势如何？
- 通胀水平对成本有何影响？
- 消费者购买力变化趋势？

### 2.3 Social（社会因素）

**定义**：影响组织的社会文化和人口因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 人口结构 | 年龄分布、人口增长 | 老龄化、人口红利 |
| 文化趋势 | 价值观、生活方式 | 健康意识、消费升级 |
| 教育水平 | 教育普及、人才供给 | 高等教育普及率 |
| 社会流动性 | 城市化、社会分层 | 城镇化率、中产阶级 |

**分析问题清单**：
- 人口结构有什么变化趋势？
- 社会价值观有什么变化？
- 教育水平如何影响人才供给？
- 生活方式变化带来什么机会？

### 2.4 Technological（技术因素）

**定义**：影响组织的技术创新和变革因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 技术创新 | 新技术、新应用 | AI、大数据、区块链 |
| 研发投入 | 行业研发水平 | 研发支出占比 |
| 技术替代 | 颠覆性技术 | 新技术替代旧技术 |
| 知识产权 | 专利、技术壁垒 | 技术垄断、开放标准 |

**分析问题清单**：
- 有哪些颠覆性技术正在兴起？
- 行业研发投入趋势如何？
- 技术替代风险有多大？
- 知识产权格局如何？

### 2.5 Environmental（环境因素）

**定义**：影响组织的自然环境和可持续发展因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 气候变化 | 碳排放、极端天气 | 碳中和目标 |
| 环保法规 | 环境保护政策 | 排放标准、环保税 |
| 资源稀缺 | 能源、原材料 | 能源转型、资源约束 |
| 可持续发展 | ESG要求 | 绿色供应链 |

**分析问题清单**：
- 气候变化政策对企业有何影响？
- 环保法规有什么新要求？
- 资源稀缺性如何影响成本？
- ESG要求有什么变化？

### 2.6 Legal（法律因素）

**定义**：影响组织的法律法规和合规要求因素

**分析要点**：
| 因素类型 | 分析内容 | 示例 |
|----------|----------|------|
| 行业监管 | 行业准入、经营许可 | 牌照管理、资质要求 |
| 劳动法规 | 劳动保护、用工规定 | 最低工资、工时规定 |
| 消费者保护 | 消费者权益、产品质量 | 产品责任、消费者赔偿 |
| 数据隐私 | 数据保护、网络安全 | GDPR、数据本地化 |

**分析问题清单**：
- 行业监管有什么新变化？
- 劳动法规对企业用工有何影响？
- 数据隐私法规有什么要求？
- 合规成本有何变化？

## 三、分析流程

### 3.1 标准流程

```
步骤1: 确定分析范围
    │
    ↓
步骤2: 六维度扫描
    │
    ↓
步骤3: 因素识别与分类
    │
    ↓
步骤4: 影响评估
    │
    ↓
步骤5: 交叉影响分析
    │
    ↓
步骤6: 战略启示提炼
```

### 3.2 因素评估方法

**影响程度评分**：
| 分值 | 描述 |
|------|------|
| 5 | 决定性影响 |
| 4 | 重大影响 |
| 3 | 中等影响 |
| 2 | 轻微影响 |
| 1 | 微不足道 |

**发生概率评分**：
| 分值 | 描述 |
|------|------|
| 5 | 几乎确定（>90%） |
| 4 | 很可能（70-90%） |
| 3 | 可能（50-70%） |
| 2 | 不太可能（30-50%） |
| 1 | 很不可能（<30%） |
