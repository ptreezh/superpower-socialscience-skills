# 波特五力模型详细指南

## 一、方法论概述

波特五力模型是分析行业竞争结构和盈利能力的核心工具，通过评估五种竞争力量，判断行业吸引力和战略定位方向。

## 二、五力详解

### 1. 现有竞争者竞争程度
**决定因素**：
- 竞争者数量和规模分布
- 行业增长率
- 固定成本/存储成本
- 产品差异化程度
- 退出壁垒

**评估标准**：
- 高强度：价格战、广告战、产品创新战
- 中强度：有限竞争、非价格竞争
- 低强度：差异化明显、市场稳定

### 2. 潜在进入者威胁
**进入壁垒**：
- 规模经济
- 产品差异化
- 资本需求
- 转换成本
- 分销渠道
- 政府政策
- 绝对成本优势

**评估标准**：
- 高威胁：壁垒低、利润吸引
- 中威胁：有一定壁垒
- 低威胁：壁垒高、难以进入

### 3. 替代品威胁
**威胁因素**：
- 替代品性价比
- 转换成本
- 买家对替代品偏好
- 替代品行业利润率

**评估标准**：
- 高威胁：替代品性价比高、转换容易
- 中威胁：替代品有限
- 低威胁：难以替代

### 4. 供应商议价能力
**影响因素**：
- 供应商集中度
- 替代投入品存在性
- 行业对供应商重要性
- 投入品差异化
- 转换成本
- 向前整合威胁

**评估标准**：
- 高能力：供应商强势、可提价
- 中能力：相对平衡
- 低能力：买方强势

### 5. 买方议价能力
**影响因素**：
- 买方集中度
- 买方购买量
- 产品差异化
- 转换成本
- 买方利润率
- 向后整合威胁
- 产品对买方成本重要性

**评估标准**：
- 高能力：买方强势、压低价格
- 中能力：相对平衡
- 低能力：卖方强势

## 三、分析流程

```
步骤1: 行业界定
    ↓
步骤2: 五力分别评估
    ↓
步骤3: 力量强度评分(1-5)
    ↓
步骤4: 行业吸引力判断
    ↓
步骤5: 战略建议
```
