# 负面案例001: 机械校准的错误

## ❌ 错误做法

### 机械校准（统计思维套用）

**研究问题**: 哪些因素导致民主转型?

**错误分析: 使用统计分位数校准**

**步骤1: 数据收集**
```yaml
数据集: 100个国家
变量:
  - GDP per capita
  - Gini coefficient
  - NGO density
  - Polity score
```

**步骤2: 错误的校准**
```yaml
❌ 机械校准:

GDP per capita:
  0.0: 33%分位数 (< $1,234)
  0.5: 50%分位数 (中位数)
  1.0: 67%分位数 (>$5,678)

Gini coefficient:
  0.0: 33%分位数 (<0.32)
  0.5: 50%分位数 (0.41)
  1.0: 67%分位数 (>0.48)

问题:
  - 没有理论依据
  - 不同样本不可比较
  - 分位数随样本变化
  - "中等收入"在非洲样本和全球样本不同
```

**步骤3: MSQCA分析**
```yaml
真值表分析 → 布尔最小化 → 解

解: GDP*~Gini + NGO → 民主
```

**步骤4: 错误的结论**
```yaml
结论:
  - "经济发展是民主的必要条件"
  - "低不平等导致民主"

问题:
  - 基于错误的校准
  - 结果不可复现
  - 理论含义模糊
```

## ✅ 正确做法

### 理论指导的校准

**同样的研究问题，正确的做法**

**步骤1: 理论定义**
```yaml
基于世界银行和联合国定义:

GDP per capita:
  0.0: 低收入国家定义（<$1,045）
  0.5: 中等收入国家定义（$4,095）
  1.0: 高收入国家定义（>$12,695）

Gini coefficient:
  0.0: 高平等（北欧水平，<0.30）
  0.5: 中等不平等（0.40）
  1.0: 高不平等（>0.50，南非水平）

NGO density:
  0.0: 无公民社会（<1 NGO per 100,000）
  0.5: 中等密度（10 NGOs per 100,000）
  1.0: 高密度（>50 NGOs per 100,000，北欧水平）
```

**步骤2: 使用校准函数**
```yaml
使用logistic函数:
  membership = 1 / (1 + e^(-α(x - midpoint)))

其中:
  - midpoint: 0.5的锚点
  - α: 调整陡峭程度
```

**步骤3: MSQCA分析**
```yaml
真值表分析 → 布尔最小化 → 解

解可能完全不同!
因为校准标准不同
```

**步骤4: 理论驱动的结论**
```yaml
结论:
  - "当经济发展达到高收入水平时，
      结合低不平等或强公民社会，
      可能导致民主"

  - "经济发展不是必要条件，
      但在充分条件中重要"

  - "多条路径可导致民主（等效性）"
```

## 🎯 关键教训

### 机械校准的问题

1. **不可比较**
   - 不同样本的分位数不同
   - 同一国家在不同样本中隶属度不同
   - 无法跨研究比较

2. **无理论依据**
   - 不知道"高收入"是什么
   - 不知道"高不平等"是什么
   - 理论含义模糊

3. **结果不稳定**
   - 增加案例→分位数变化→校准变化→结果变化
   - 无法保证稳健性

### 理论指导校准的优势

1. **可比较**
   - 使用固定的理论锚点
   - 不同样本可比较
   - 跨研究可复制

2. **理论清晰**
   - 知道每个集合的定义
   - 理论含义明确
   - 可理论讨论

3. **结果稳健**
   - 锚点不随样本变化
   - 结果更稳定
