# 正面案例001: 社会运动成功因素的MSQCA分析

## ✅ 正确做法

### 研究问题

**哪些因果组合导致社会运动成功?**

**案例**: 30个社会运动（20世纪后期）

### 1. 理论指导的校准（反机械校准）

**条件与结果**:
```yaml
条件C1: 政治机会（开放性）
  理论: Tarrow (1998) 政治机会结构理论
  锚点:
    0.0: 完全威权（无抗议空间）
    0.5: 部分开放（有限抗议空间）
    1.0: 完全民主（完全抗议空间）

条件C2: 资源动员（财务资源）
  理论: McCarthy & Zald (1977) 资源动员理论
  锚点:
    0.0: 无外部资金
    0.5: 中等资金（可组织活动）
    1.0: 充足资金（可大规模动员）

条件C3: 文化框架（共鸣性）
  理论: Snow & Benford (1988) 框架理论
  锚点:
    0.0: 无共鸣（边缘议题）
    0.5: 部分共鸣（部分支持）
    1.0: 高共鸣（主流议题）

条件C4: 组织强度（网络密度）
  理论: Diani (2003) 社会运动组织理论
  锚点:
    0.0: 无组织
    0.5: 松散网络
    1.0: 强组织（密集网络）

结果Y: 运动成功
  操作化:
    0.0: 完全失败（无影响）
    0.5: 部分成功（部分目标达成）
    1.0: 完全成功（所有目标达成）
```

**正确做法**:
- ✅ 每个条件有明确理论依据
- ✅ 锚点有理论或实证定义
- ✅ 校准标准在报告中详细说明

**错误做法**（避免）:
- ❌ 使用33%, 67%分位数
- ❌ 使用均值±标准差
- ❌ 无理论依据

### 2. 必要性分析

**问题**: 是否有运动成功的必要条件?

**结果**:
| 条件 | 一致性 | 覆盖度 | 必要性? |
|------|--------|--------|--------|
| 政治机会 | 0.65 | - | 否 |
| 资源动员 | 0.72 | - | 否 |
| 文化框架 | 0.88 | - | 接近（≥0.90才算必要） |
| 组织强度 | 0.58 | - | 否 |

**发现**: 没有严格必要条件，但文化框架接近必要。

**解释**:
- 社会运动可以通过多种路径成功
- 没有单一条件是必须的
- 等效性（equifinality）存在

### 3. 充分性分析

**真值表** (简化):
| C1 | C2 | C3 | C4 | 案例数 | 一致性 | 逻辑行 |
|----|----|----|----|--------|--------|--------|
| 1 | 1 | 1 | 1 | 5 | 0.98 | ✓ |
| 1 | 0 | 1 | 1 | 3 | 0.95 | ✓ |
| 0 | 1 | 1 | 1 | 2 | 0.92 | ✓ |
| 1 | 1 | 0 | 1 | 1 | 0.75 | ✓（边缘）|
| ... | ... | ... | ... | ... | ... | ... |

**布尔最小化结果**:

**复杂解**:
```
成功 = C1*C3*C4 + C1*~C2*C3*C4 + ~C1*C2*C3*C4 + C1*C2*~C3*C4
```

**中间解**（推荐）:
```
成功 = C1*C3*C4 + ~C1*C2*C3
```

**解释**:
- **路径1** (C1*C3*C4): 政治机会 + 文化框架 + 组织强度
  - 案例: 民主国家的高共鸣运动（如美国民权运动）
  - 占比: 原始覆盖度 0.55

- **路径2** (~C1*C2*C3): 威权体制下需要资源+文化框架
  - 案例: 威权国家的资源丰富运动（如南非反种族隔离）
  - 占比: 原始覆盖度 0.25

**总覆盖度**: 0.80（80%的成功案例被解释）

### 4. 案例归属与深入分析

**路径1的案例** (C1*C3*C4):
- 美国民权运动 (1960s)
- 北欧环保运动
- 台湾民主运动

**深入案例: 美国民权运动**
```yaml
条件组合:
  政治机会: 0.8（部分开放的民主）
  资源动员: 0.7（中等资金）
  文化框架: 0.9（高共鸣：种族平等）
  组织强度: 0.95（强组织：教会、SCLC）

因果机制:
  1. 高政治开放度 → 抗议空间
  2. 高文化共鸣 → 公众支持
  3. 强组织 → 有效动员

过程追踪:
  - 1955: Montgomery Bus Boycott（教会组织）
  - 1963: March on Washington（跨组织联盟）
  - 1964: Civil Rights Act（政策成功）
```

**路径2的案例** (~C1*C2*C3):
- 南非反种族隔离运动
- 波兰团结工会
- 韩国民主化运动

**深入案例: 南非反种族隔离运动**
```yaml
条件组合:
  政治机会: 0.1（威权）
  资源动员: 0.9（高资金：国际支持）
  文化框架: 0.85（高共鸣：人权）
  组织强度: 0.8（强组织：ANC）

因果机制:
  1. 威权体制 → 需要资源对抗
  2. 高资源 → 维持长期运动
  3. 高文化共鸣 → 国际压力

过程追踪:
  - 1960s: Sharpeville Massacre（镇压）
  - 1970s: 国际制裁（资源+共鸣）
  - 1990s: Mandela释放（成功）
```

### 5. 稳健性检验

**参数敏感性**:
| 一致性阈值 | 解 | 变化 |
|-----------|----|----|
| 0.75 | C1*C3*C4 + ~C1*C2*C3 | 基准 |
| 0.80 | C1*C3*C4 + ~C1*C2*C3 | 稳定 |
| 0.85 | C1*C3*C4 | 路径2消失 |

**校准敏感性**:
```yaml
原始校准（理论锚点）:
  解: C1*C3*C4 + ~C1*C2*C3

替代校准（±10%锚点）:
  解: C1*C3*C4 + ~C1*C2*C3

结论: 稳健
```

### 6. 理论贡献

**发现**:
1. **等效性**: 两条路径可导致成功
   - 路径1: 民主+高共鸣+强组织
   - 路径2: 威权+高资源+高共鸣

2. **文化框架的核心性**:
   - 出现在所有路径
   - 接近必要条件（一致性0.88）

3. **组织强度的重要性**:
   - 路径1需要，路径2不一定
   - 因为路径2有资源补偿

**理论含义**:
- 支持资源动员理论（路径2）
- 支持政治机会理论（路径1）
- 支持框架理论（所有路径）
- **整合**: 不同条件需要不同组合
