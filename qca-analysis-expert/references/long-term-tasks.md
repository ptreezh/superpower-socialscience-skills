# MSQCA长时研究项目指南

## 场景1: 中小样本比较研究（2-3周）

### Phase 1: 理论与数据准备（Week 1）

**Task 1.1: 研究设计（2天）**
- 选择研究问题
- 理论框架构建
- 条件选择（3-6个）
- 结果定义

**Task 1.2: 数据收集（3天）**
- 案例选择（10-50个）
- 变量操作化
- 数据收集
- 数据质量检查

### Phase 2: 校准与分析（Week 2）

**Task 2.1: 校准（2天）**
- 每个条件的理论定义
- 锚点设定（0.0, 0.5, 1.0）
- 执行校准
- 校准质量检查

**Task 2.2: 必要性分析（1天）**
- 计算必要一致性
- 识别必要条件
- 必要条件报告

**Task 2.3: 充分性分析（2天）**
- 构建真值表
- 布尔最小化
- 解选择（复杂/简化/中间）

### Phase 3: 案例与稳健性（Week 3）

**Task 3.1: 案例归属（2天）**
- 每个组合的案例列表
- 深入案例分析
- 跨案例比较

**Task 3.2: 稳健性检验（2天）**
- 参数敏感性（3种设置）
- 校准敏感性（2种方案）
- 稳健性报告

**Task 3.3: 最终报告（1天）**
- 解的表达式
- 案例分布
- 理论含义
- 政策建议

## 场景2: 跨国比较研究（1-2个月）

### 研究问题: 哪些因果组合导致民主转型?

**案例**: 50-100个国家

**条件**:
- 经济发展（人均GDP）
- 不平等（基尼系数）
- 公民社会（NGO密度）
- 外部压力（国际贸易依赖）
- 政治制度（前政体类型）

**结果**: 民主转型（是/否，或模糊集）

### Phase 1: 理论框架（2周）

**文献综述**:
- 现代化理论（经济发展）
- 阶级理论（不平等）
- 公民社会理论
- 依赖理论
- 制度主义

**理论假设**:
- H1: 经济发展 → 民主
- H2: 低不平等 → 民主
- H3: 强公民社会 → 民主
- H4: 组合效应？

### Phase 2: 数据收集与校准（2周）

**数据来源**:
- World Bank（GDP）
- Standardized World Income Inequality Database（基尼系数）
- Varieties of Democracy（公民社会、民主）
- Polity IV（政体类型）

**校准标准**:
```yaml
经济发展:
  0.0: <$1,000 per capita
  0.5: $4,000 per capita
  1.0: >$12,000 per capita

不平等:
  0.0: 基尼<0.3（平等）
  0.5: 基尼=0.4
  1.0: 基尼>0.5（不平等）

公民社会:
  0.0: 无NGO
  0.5: 中等密度
  1.0: 高密度（北欧水平）
```

### Phase 3: 分析与发现（3周）

**必要性分析**:
- 是否有民主转型的必要条件?
- 例如: "经济发展是否必要?"

**充分性分析**:
- 哪些组合导致民主?
- 例如: "高经济发展 + 低不平等 → 民主"
- 例如: "高经济发展 + 强公民社会 → 民主"

**等效路径**:
- 路径1: 经济发展 + 低不平等
- 路径2: 经济发展 + 强公民社会
- 路径3: ...

### Phase 4: 案例深入研究（2周）

**每个路径的代表性案例**:
- 路径1: 韩国、台湾
- 路径2: 印度、哥斯达黎加
- ...

**过程追踪**:
- 验证因果机制
- 排除反向因果

### Phase 5: 稳健性与理论（1周）

**稳健性检验**:
- 不同一致性阈值（0.75, 0.80, 0.85）
- 不同校准标准
- 不同案例集

**理论贡献**:
- 支持哪些理论?
- 反对哪些理论?
- 新的理论发现?

## 场景3: 混合方法研究（2-3个月）

### 设计: QCA + 深入案例研究

**Phase 1: QCA分析（1个月）**
- 完整MSQCA分析
- 识别关键路径
- 选择代表性案例

**Phase 2: 深入案例研究（1个月）**
- 2-4个代表性案例
- 过程追踪
- 历史分析
- 访谈/档案

**Phase 3: 理论整合（1个月）**
- QCA结果 + 案例研究
- 因果机制提炼
- 理论生成
