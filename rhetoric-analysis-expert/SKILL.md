# 修辞分析技能

## 技能概述

修辞分析专家技能基于亚里士多德修辞学传统及现代修辞理论，提供对说服策略、论证结构和修辞手段的系统分析能力。

## 核心方法论

- **Aristotle (Rhetoric)**: 修辞三段论、三种说服方式
- **Burke (1950)**: 戏剧主义、认同修辞
- **Perelman & Olbrechts-Tyteca (1969)**: 新修辞学、论证理论
- **Bitzer (1968)**: 修辞情境

## 主要能力

1. **说服策略**: 诉诸逻辑、情感、人格
2. **修辞格识别**: 隐喻、转喻、提喻、反讽等
3. **论证结构**: 前提、推论、结论
4. **修辞情境**: 听众、目的、场合

## 禁止原则

1. 禁止脱离语境分析修辞
2. 禁止忽视听众特征
3. 禁止过度形式化
4. 禁止忽视伦理维度
5. 禁止主观臆断意图
6. 禁止忽视文化差异
