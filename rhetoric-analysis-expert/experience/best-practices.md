# 修辞分析最佳实践

## 核心原则

### 原则1：情境优先
```
分析顺序：
修辞情境 → 说服策略 → 修辞手段 → 效果评估
```

### 原则2：三要素协同
- 不孤立分析单一要素
- 关注logos, pathos, ethos的配比
- 分析三者之间的互动关系

### 原则3：功能导向
- 修辞格不仅是形式特征
- 每个修辞手段都有功能
- 分析"为什么"比"是什么"更重要

---

## 分析流程

### 第一步：理解语境
1. 确定文本类型和体裁
2. 了解历史背景
3. 明确分析目的

### 第二步：分析修辞情境
1. 识别急迫性
2. 描述目标听众
3. 列出限制因素

### 第三步：评估说服策略
1. Logos分析
2. Pathos分析
3. Ethos分析
4. 综合评估

### 第四步：识别修辞手段
1. 比喻类修辞
2. 结构类修辞
3. 音韵类修辞

### 第五步：评估效果
1. 说服效果
2. 审美效果
3. 伦理效果

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 孤立识别修辞格 | 结合语境分析功能 |
| 忽视听众 | 分析目标听众特征 |
| 主观臆断意图 | 基于文本证据分析 |
| 忽视伦理 | 评估修辞的伦理维度 |
| 文化盲视 | 考虑文化背景差异 |
