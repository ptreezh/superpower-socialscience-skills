# 修辞分析经验记忆

## 成功模式

### 模式1: 三要素协同分析
- **场景**: 政治演说分析
- **做法**: 同时分析logos, pathos, ethos三者的配比和互动
- **效果**: 全面把握说服策略

### 模式2: 情境优先
- **场景**: 广告修辞分析
- **做法**: 先分析修辞情境，再分析具体策略
- **效果**: 避免脱离语境

### 模式3: 读者反应
- **场景**: 文学修辞分析
- **做法**: 关注修辞对不同读者的影响
- **效果**: 理解修辞的多元效果

## 失败教训

### 教训1: 孤立修辞格
- **问题**: 仅识别修辞格，不分析功能
- **原因**: 过度形式化倾向
- **修正**: 必须分析修辞格的功能和效果

### 教训2: 忽视听众
- **问题**: 只分析说话人策略，不分析听众
- **原因**: 单向度思维
- **修正**: 修辞是双向互动过程

### 教训3: 主观臆断
- **问题**: 无依据推测说话人"真实意图"
- **原因**: 过度解读
- **修正**: 基于文本证据分析
