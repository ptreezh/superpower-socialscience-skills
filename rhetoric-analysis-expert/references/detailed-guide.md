## 第一章：修辞学概论

### 1.1 学科定义

修辞学是研究说服艺术的学科，关注如何有效地使用语言影响他人。从古希腊发展至今，形成了丰富的理论体系。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 古典时期 | Aristotle | 修辞三要素、三段论 |
| 罗马时期 | Cicero, Quintilian | 五艺、演说术 |
| 中世纪 | Augustine | 基督教修辞 |
| 文艺复兴 | Ramus | 修辞与逻辑分离 |
| 现代 | Burke, Perelman | 新修辞学 |

### 1.3 核心概念

```
修辞学体系
├── 修辞情境
│   ├── 急迫性
│   ├── 听众
│   └── 限制
├── 说服方式
│   ├── Logos(逻辑)
│   ├── Pathos(情感)
│   └── Ethos(人格)
├── 修辞手段
│   ├── 比喻类
│   ├── 结构类
│   └── 音韵类
└── 论证结构
    ├── 前提
    ├── 推论
    └── 结论
```

## 第二章：说服策略

### 2.1 Logos(逻辑诉求)

**论证类型**:
- 演绎论证: 从一般到特殊
- 归纳论证: 从特殊到一般
- 类比论证: 从相似到相似

**修辞三段论**:
```
大前提: 所有X都是Y
小前提: Z是X
结论: 因此Z是Y

修辞三段论(enthymeme):
前提(隐含): [共同认知]
陈述: A
结论: 因此B
```

### 2.2 Pathos(情感诉求)

**情感类型**:
| 情感 | 修辞策略 |
|------|---------|
| 恐惧 | 威胁、危险描述 |
| 愤怒 | 不公正展示 |
| 同情 | 苦难叙述 |
| 希望 | 美好愿景 |

### 2.3 Ethos(人格诉求)

**品格维度**:
- 智慧
- 品德
- 善意

## 第三章：修辞格

### 3.1 比喻类修辞

**隐喻分析**:
1. 识别本体和喻体
2. 分析映射关系
3. 评估修辞效果

**转喻类型**:
- 作者代作品: 读鲁迅
- 容器代内容: 喝两瓶
- 地点代事件: 华盛顿决定

### 3.2 结构类修辞

**排比类型**:
- 词语排比
- 句子排比
- 段落排比

**对偶特征**:
- 字数相等
- 结构相同
- 意义相对或相关
