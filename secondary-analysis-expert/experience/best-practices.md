# 二手数据分析最佳实践

## 核心原则

### 原则1：尊重原始设计
```
理解数据来源是分析的前提
- 审查原始研究目的
- 理解抽样设计
- 了解测量工具
```

### 原则2：质量优先
```
数据质量决定分析边界
- 评估数据质量
- 识别数据限制
- 谨慎得出结论
```

### 原则3：限制透明
```
所有限制必须明确说明
- 设计限制
- 测量限制
- 推断限制
```

---

## 分析流程

### 第一步：问题定义
1. 明确研究问题
2. 确定变量需求
3. 列出假设

### 第二步：数据搜索
1. 搜索合适数据源
2. 比较各来源优劣
3. 选择最佳来源

### 第三步：质量评估
1. 审查原始研究
2. 评估数据质量
3. 判断适用性

### 第四步：数据处理
1. 选择变量
2. 重构变量
3. 检验等价性

### 第五步：分析执行
1. 执行分析
2. 检验假设
3. 稳健性检验

### 第六步：限制说明
1. 列出所有限制
2. 讨论对结论影响
3. 提出未来研究建议

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 忽视原始设计 | 仔细审查 |
| 忽视数据限制 | 明确说明 |
| 过度解释 | 区分相关因果 |
| 忽视等价性 | 检验测量等价 |
| 忽视样本偏差 | 评估代表性 |
| 忽视伦理 | 遵守规定 |
