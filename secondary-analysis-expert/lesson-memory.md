# 二手数据分析经验记忆

## 成功模式

### 模式1: 多源数据三角
- **场景**: 社会趋势分析
- **做法**: 整合多个调查数据源
- **效果**: 提高结论可靠性

### 模式2: 趋势追踪
- **场景**: 长期变化研究
- **做法**: 利用多年调查数据
- **效果**: 揭示发展趋势

### 模式3: 限制透明
- **场景**: 数据质量参差
- **做法**: 明确说明所有限制
- **效果**: 提高研究可信度

## 失败教训

### 教训1: 忽视设计限制
- **问题**: 用数据回答原设计不支持的问题
- **原因**: 未审查原始研究
- **修正**: 仔细审查原始设计

### 教训2: 过度解释
- **问题**: 相关当作因果
- **原因**: 忽视因果限制
- **修正**: 明确说明相关关系

### 教训3: 测量不等价
- **问题**: 直接比较不同测量
- **原因**: 忽视问卷变化
- **修正**: 检验测量等价性
