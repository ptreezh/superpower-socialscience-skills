## 第一章：二手数据分析概论

### 1.1 学科定义

二手数据分析是对他人收集的数据进行再分析的研究方法，是高效利用现有数据资源的重要途径。

### 1.2 发展历程

| 时期 | 代表人物 | 贡献 |
|------|---------|------|
| 早期 | Kiecolt & Nathan | 方法论奠基 |
| 发展 | Stewart & Kamins | 质量评估框架 |
| 数据化 | Riley | 大型调查分析 |
| 当代 | Smith | 比较研究应用 |

### 1.3 核心概念

```
二手分析体系
├── 数据类型
│   ├── 调查数据
│   ├── 行政数据
│   ├── 档案数据
│   └── 大数据
├── 分析任务
│   ├── 描述分析
│   ├── 解释分析
│   ├── 比较分析
│   └── 趋势分析
└── 质量控制
    ├── 来源评估
    ├── 变量验证
    └── 限制说明
```

## 第二章：数据来源评估

### 2.1 数据来源类型

**政府调查数据**:
```
特点：
- 大样本代表性
- 系统性收集
- 公开可获取
- 标准化测量

例子：人口普查、劳动力调查
```

**学术调查数据**:
```
特点：
- 理论驱动设计
- 详细测量
- 专业执行
- 需申请获取

例子：GSS、CGSS、WVS
```

### 2.2 质量评估框架

| 维度 | 问题 | 指标 |
|------|------|------|
| 概念效度 | 测量是否有效？ | 效度检验 |
| 样本代表性 | 是否有代表性？ | 抽样方法 |
| 数据完整性 | 是否有缺失？ | 缺失率 |
| 测量可靠性 | 是否可靠？ | 信度系数 |
| 文档完整性 | 文档是否充分？ | 文档清单 |

### 2.3 适用性判断

```
适用性评估问题：
1. 数据是否涵盖研究问题？
2. 变量是否可用？
3. 样本是否合适？
4. 时间是否相关？
5. 测量是否有效？
```

## 第三章：变量重构

### 3.1 变量处理策略

**直接使用**:
- 原始变量直接满足需求
- 编码方式适合分析

**重新编码**:
```
原变量 → 新变量
连续 → 分类
多分类 → 二分类
反向编码
```

**组合创建**:
```
新变量 = f(原变量1, 原变量2, ...)
- 加总指数
- 均值指数
- 因子得分
```

### 3.2 测量等价性

**跨时间等价**:
- 问题表述是否一致？
- 编码方式是否变化？
- 测量尺度是否相同？

**跨群体等价**:
- 概念是否等价？
- 测量是否等价？
- 结构是否等价？

## 第四章：分析执行

### 4.1 分析类型

| 类型 | 目的 | 方法 |
|------|------|------|
| 描述分析 | 描述分布 | 描述统计 |
| 解释分析 | 解释关系 | 回归分析 |
| 比较分析 | 比较差异 | 方差分析 |
| 趋势分析 | 分析变化 | 时间序列 |

### 4.2 限制说明

**必须说明的限制**:
1. 原始研究设计的限制
2. 测量工具的限制
3. 样本代表性的限制
4. 分析者假设的限制
