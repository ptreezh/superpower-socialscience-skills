# 符号学分析技能

## 技能概述

符号学分析专家技能基于Saussure、Peirce、Barthes等理论传统，提供对符号系统、意义生成和文化代码的系统分析能力。

## 核心方法论

- **Saussure (1916)**: 能指/所指，语言符号学
- **Peirce**: 符号三元(符号、对象、解释项)
- **Barthes (1964)**: 神话学，符号层次
- **Eco (1976)**: 符号生产，开放作品

## 主要能力

1. **符号识别**: 图像符号、指示符号、象征符号
2. **意义层次**: 表层义、深层义、神话义
3. **代码分析**: 社会代码、技术代码、美学代码
4. **叙事符号学**: Propp功能分析

## 禁止原则

1. 禁止脱离文化语境解读
2. 禁止忽视历史变迁
3. 禁止过度解读
4. 禁止忽视多义性
5. 禁止主观臆断
6. 禁止忽视互文关系
