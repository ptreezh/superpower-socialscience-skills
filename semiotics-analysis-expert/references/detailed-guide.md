## 第一章：符号学基础

### 1.1 学科渊源

符号学是研究符号和意义生成的学科，有两个主要传统：

**Saussure传统**:
- 索绪尔《普通语言学教程》(1916)
- 语言符号学，结构主义
- 能指/所指二元模型

**Peirce传统**:
- 皮尔士符号学
- 符号三元模型
- 图像/指示/象征分类

### 1.2 基本概念

**Saussure**:
```
符号 = 能指 + 所指
      (形式)   (概念)

例: "树" = [语音/文字] + [树的概念]
```

**Peirce**:
```
符号三元关系
├── 符号: 代表某物的媒介
├── 对象: 被代表的事物
└── 解释项: 符号产生的意义效果
```

## 第二章：符号类型学

### 2.1 Peirce三分类

| 类型 | 关系特征 | 示例 |
|------|---------|------|
| 图像符号 | 相似性 | 照片、肖像、地图 |
| 指示符号 | 因果/邻近 | 烟→火、温度计、脚印 |
| 象征符号 | 约定俗成 | 文字、语言、国旗 |

### 2.2 符号类型分析

**图像符号分析**:
- 识别相似特征
- 分析相似程度
- 考察文化因素

**指示符号分析**:
- 确定因果/邻近关系
- 分析指示强度
- 考察语境依赖

**象征符号分析**:
- 追溯约定历史
- 分析文化编码
- 考察变迁可能

### 3.1 表层义

- 直接、字面的意义
- 首层编码结果
- 相对稳定

### 3.2 深层义

- 隐含、联想的意义
- 二次编码结果
- 文化依赖

### 3.3 神话义(Barthes)

```
语言系统:
能指: 玫瑰的形式
所指: 花的概念

神话系统:
能指: "玫瑰"符号
所指: "爱情"意识形态
```
