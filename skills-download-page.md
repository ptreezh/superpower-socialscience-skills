## 📋 使用说明

### 1. 下载 ZIP 包

点击上面的下载链接，下载对应的技能 ZIP 包。

### 2. 解压到技能目录

将 ZIP 包解压到你的 AI CLI 工具的技能目录：
- OpenClaw: `~/.openclaw/skills/`
- WorkBuddy: `~/.workbuddy/skills/`
- Qwen: `~/.qwen/skills/`

### 3. 使用技能

```bash
# OpenClaw
openclaw --skill skill-name "任务描述"

# WorkBuddy
workbuddy --skill skill-name "任务描述"

# Qwen
qwen --skill skill-name "任务描述"
```
