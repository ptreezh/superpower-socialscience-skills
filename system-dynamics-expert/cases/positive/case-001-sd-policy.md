---

## 📋 建模背景

### 研究问题
评估公共卫生政策对疾病传播的长期影响

### 关键变量

```yaml
存量:
  - 易感人群 (S)
  - 感染人群 (I)
  - 康复人群 (R)
  - 免疫人群 (V)

流量:
  - 感染率 (S→I)
  - 康复率 (I→R)
  - 免疫率 (R→V)
  - 出生率
  - 死亡率

参数:
  - 传播系数
  - 康复系数
  - 基础再生数R0
```

---

## 🎯 建模执行

### Phase 1: 问题界定

**动态问题**: 疾病传播有周期性波动

**参考模式**:
```
感染人数
    |
    |   ╱─╲
    |  ╱  ╲
    | ╱   ╲
    |╱────╲________
    |
    └─────────────→ 时间
```

### Phase 2: 系统建模

**CLD**: SIR模型 + 政策干预回路

```yaml
主回路:
  R+: 感染 → 康复 → 免疫 → 易感减少 → 感染下降
  B-: 感染 → 康复 → 易感减少 → 感染下降

政策回路:
  B-: 政策干预 → 感染率下降 → 感染减少
  B-: 疫苗接种 → 易感直接转免疫 → 感染减少
```

**SFD**:
```
[S] --infection--> [I] --recovery--> [R] --immunity--> [V]
  ↑                                        |
  +--------------------------------------+
```

### Phase 3: 仿真分析

**情景测试**:
1. 基准情景（无干预）
2. 疫苗接种情景（50%接种率）
3. 社交距离情景（减少50%接触）

**发现**:
- 基准: 感染率峰值60%，周期性波动
- 疫苗: 峰值降至20%，波动平缓
- 社交距离: 峰值降至40%，但仍有波动
- 联合: 峰值降至10%，几乎消除波动

**敏感性**: 传播系数是关键参数，1%变化导致峰值5%变化
