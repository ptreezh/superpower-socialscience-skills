## 🎯 高频分析模式

### Pattern 1: 政策SD建模

**频率**: 高频 (20次/年)
**成功率**: 92%
**平均时长**: 3-4小时

#### 标准建模流程

```yaml
Phase 1: 问题界定（1小时）
  Task 1.1: 动态问题识别
    - 输出: 问题陈述
    - 验证: 适合SD

  Task 1.2: 关键变量选择
    - 输出: 变量清单
    - 验证: 数据可获得

  Task 1.3: 参考模式（30分钟）
    - 输出: 历史行为图
    - 验证: 有数据支持

Phase 2: 系统建模（1.5小时）
  Task 2.1: CLD绘制（30分钟）
    - 输出: 因果回路图
    - 验证: 关键回路包含

  Task 2.2: SFD绘制（30分钟）
    - 输出: 存量流量图
    - 验证: 存量流量完整

  Task 2.3: 方程编写（30分钟）
    - 输出: 方程文档
    - 验证: 数学正确

Phase 3: 仿真分析（1.5小时）
  Task 3.1: 模型校准（30分钟）
    - 输出: 校准报告
    - 验证: 拟合优度R²>0.7

  Task 3.2: 行为重现（30分钟）
    - 输出: 行为模式报告
    - 验证: 关键模式重现

  Task 3.3: 敏感性分析（30分钟）
    - 输出: 敏感性报告
    - 验证: 多参数测试
```

## 🛠️ SD工具链

### 1. Vensim (首选)

```
Population(t) = Population(t-dt) + (Births - Deaths) * dt
Births = Population * Birth_Rate
Deaths = Population * Death_Rate
```

### 2. PySD (Python)

```python
import pysd
model = pysd.read_vensim('model.mdl')
result = model.run(params={'Birth_Rate': 0.1})
result.plot()
```

## 🔍 核心概念

### 存量与流量

```yaml
存量 (Stock):
  - 累积量
  - 随时间变化
  - 例如: 人口、库存、知识

流量 (Flow):
  - 改变存量
  - 流入/流出
  - 例如: 出生、死亡、生产
```

### 反馈回路

```yaml
正反馈 (R+):
  - 增强变化
  - 指数增长
  - 例如: 人口增长

负反馈 (B-):
  - 抵消变化
  - 寻求目标
  - 例如: 温度调节
```
