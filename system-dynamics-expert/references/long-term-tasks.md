# 系统动力学长时研究项目指南

## 📋 研究周期：3-12个月

### 适用场景

- **政策分析** (公共卫生、环境、经济政策)
- **组织管理** (供应链、项目管理、运营优化)
- **城市系统** (人口、交通、资源分配)
- **生态系统** (资源管理、环境可持续性)
- **商业战略** (市场扩散、竞争动态)

---

## 🎯 SD项目分解模板

### Phase 1: 问题界定 (Month 1)

**Task 1.1**: 动态问题识别 (2周)
- **输出**: 动态问题陈述
- **验证**: 问题具有SD适用性

**Task 1.2**: 关键变量选择 (2周)
- **输出**: 变量清单、数据来源
- **验证**: 数据可获得性

**Task 1.3**: 参考模式 (2周)
- **输出**: 历史行为模式图
- **验证**: 模式有数据支持

### Phase 2: 系统建模 (Months 2-3)

**Task 2.1**: 因果回路图CLD (1个月)
- **输出**: CLD图 (PNG/PDF)
- **验证**: 关键回路包含

**Task 2.2**: 存量流量图SFD (1个月)
- **输出**: SFD图 (PNG/PDF)
- **验证**: 存量流量逻辑完整

**Task 2.3: 模型方程 (1个月)
- **输出**: 方程文档
- **验证**: 方程数学正确

### Phase 3: 仿真与分析 (Months 4-6)

**Task 3.1**: 模型校准 (2个月)
- **输出**: 校准报告
- **验证**: 历史数据拟合R²>0.7

**Task 3.2**: 行为重现 (2个月)
- **输出**: 行为模式重现报告
- **验证**: 关键模式重现

**Task 3.3**: 敏感性分析 (2个月)
- **输出**: 敏感性分析报告
- **验证**: 多参数测试

### Phase 4: 干预设计 (Months 7-9)

**Task 4.1**: 杠杆点识别 (1个月)
- **输出**: 杠杆点清单
- **验证**: 杠杆点基于模型

**Task 4.2**: 政策情景 (2个月)
- **输出**: 情景测试报告
- **验证**: 多情景对比
