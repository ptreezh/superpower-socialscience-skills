# 负面案例：主观感受式分析

## 案例信息
- **分析对象**: 艺术摄影
- **分析方法**: 主观感受式
- **问题**: 缺乏方法论
- **时间**: 2026-03-18

## 错误分析示范

### 原图描述
[一张黑白人像摄影]

### 错误分析
"这张照片让我感觉很悲伤，模特的表情很忧郁，整体氛围很压抑。我觉得摄影师想表达的是孤独和绝望。构图方面，我觉得有点偏，不太舒服。"

## 问题诊断

| 问题 | 表现 |
|------|------|
| 主观过度 | "我觉得"、"让我感觉" |
| 方法缺失 | 无理论框架支撑 |
| 证据不足 | 未引用图像特征 |
| 臆断意图 | "摄影师想表达" |
| 描述模糊 | "有点偏"、"不太舒服" |

## 正确分析示范

### 形式分析
- **构图**: 主体位于画面右侧三分之一处，符合三分法
- **光线**: 侧面光，强调面部轮廓
- **色调**: 高对比度黑白，强化戏剧性

### 互动分析
- **接触**: 模特未直视镜头，建立观察距离
- **距离**: 中景，社会距离
- **态度**: 略微俯视，观看者位置较高

### 文化解读
- **黑白摄影**: 艺术传统、严肃性
- **视角**: 建立观看者与被看者的距离
- **情绪表达**: 需结合原始语境分析

## 教训总结

1. ❌ "我觉得"不是分析
2. ❌ 必须使用方法论框架
3. ❌ 分析必须基于图像证据
4. ✅ 主观感受需转化为分析语言

---

**警示等级**: ⚠️ 典型错误
