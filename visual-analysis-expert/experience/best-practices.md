# 视觉分析最佳实践

## 核心原则

### 原则1：四维度综合
```
分析框架：
生产 → 图像 → 流通 → 受众
```
任一维度都不可遗漏。

### 原则2：方法论先行
- 使用明确的理论框架
- 分析语言规范化
- 避免主观感受式描述

### 原则3：证据支撑
- 所有分析基于图像特征
- 避免无依据的推测
- 引用可验证的特征

---

## 分析流程

### 第一步：收集材料
1. 获取高质量图像
2. 收集生产信息
3. 了解流通渠道
4. 研究受众反馈

### 第二步：形式描述
1. 构图要素
2. 色彩分析
3. 光线分析
4. 视角分析

### 第三步：互动分析
1. 接触类型
2. 社会距离
3. 态度立场
4. 情态程度

### 第四步：意义解读
1. 外延意义
2. 内涵意义
3. 文化符号
4. 意识形态

### 第五步：权力分析
1. 观看关系
2. 凝视结构
3. 权力分配
4. 抵抗可能

---

## 常见错误避免

| 错误 | 正确做法 |
|------|---------|
| 主观感受 | 使用分析语言 |
| 单维度分析 | 四维度综合 |
| 忽视语境 | 重建生产语境 |
| 忽视权力 | 分析观看政治 |
| 静态分析 | 考虑历史变迁 |
