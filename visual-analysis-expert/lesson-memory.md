# 视觉分析经验记忆

## 成功模式

### 模式1: 四维度综合
- **场景**: 广告图像分析
- **做法**: 同时分析生产、图像、流通、受众四维度
- **效果**: 全面把握图像意义

### 模式2: 观看关系优先
- **场景**: 肖像摄影分析
- **做法**: 从观看权力角度切入
- **效果**: 揭示权力结构

### 模式3: 技术形式结合
- **场景**: 数字图像分析
- **做法**: 结合技术特征和意义分析
- **效果**: 理解媒介特性

## 失败教训

### 教训1: 形式主义
- **问题**: 仅描述构图色彩
- **原因**: 忽视意义和社会维度
- **修正**: 增加文化解读环节

### 教训2: 主观过度
- **问题**: "我感觉..."
- **原因**: 缺乏方法论约束
- **修正**: 基于理论框架分析

### 教训3: 去语境化
- **问题**: 脱离原始语境
- **原因**: 忽视生产背景
- **修正**: 重建原始语境
