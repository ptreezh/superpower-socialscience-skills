# SocienceAI Website

> 让社会科学研究人人可为

## 快速开始

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
vercel --prod
```

## 目录结构

```
docs/
├── index.md              # 首页
├── methodologies/        # 方法论页面
├── guide/               # 使用指南
├── about/               # 关于
└── blog/                # 博客
```
