## 什么是方法论技能？

每个技能是一个 `SKILL.md` 文件，包含：

- **方法论定义**：基于经典学术理论
- **分析流程**：从数据准备到结果解释
- **子Agent支持**：并行处理提升效率
- **质量保证**：6大绝对禁止原则

```bash
# 在 Claude Code 中直接使用
```
