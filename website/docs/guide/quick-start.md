# 快速开始

5 分钟体验 SocienceAI 方法论技能。

## 步骤 1：选择方法论

浏览 [方法论列表](/methodologies/)，根据研究问题选择合适的方法论：

| 研究类型 | 推荐方法论 |
|----------|-----------|
| 探索性研究 | [扎根理论](/methodologies/grounded-theory)、[民族志](/methodologies/ethnography) |
| 验证性研究 | [回归分析](/methodologies/regression-analysis)、[SEM](/methodologies/sem-analysis) |
| 比较研究 | [QCA](/methodologies/qca-analysis)、[DID](/methodologies/did-analysis) |
| 战略分析 | [SWOT](/methodologies/swot-analysis)、[蓝海战略](/methodologies/blue-ocean-strategy) |

## 步骤 2：加载技能

```bash
/skill grounded-theory-expert
```

## 步骤 3：提供数据

将研究数据（访谈文本、问卷数据等）提供给 Claude Code。

## 步骤 4：获取分析结果

技能会自动执行分析流程，生成结构化报告。
